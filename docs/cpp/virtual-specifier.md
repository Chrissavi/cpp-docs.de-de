# <a name="virtual-specifier"></a>virtual-Spezifizierer

Das [Virtual](../cpp/virtual-cpp.md) -Schlüsselwort kann nur auf nicht statische Klassenmember-Funktionen angewendet werden. Es gibt an, dass die Bindung von Aufrufen der Funktion bis zur Laufzeit verzögert wird. Weitere Informationen finden Sie unter [virtuelle Funktionen](../cpp/virtual-functions.md).
