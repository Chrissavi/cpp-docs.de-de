# <a name="program-termination"></a>Programmbeendigung

C++ bietet mehrere Möglichkeiten, um ein Programm zu verlassen:

- Rufen Sie die [beenden](../cpp/exit-function.md) Funktion.

- Rufen Sie die [Abbrechen](../cpp/abort-function.md) Funktion.

- Führen Sie eine [zurückgeben](../cpp/return-statement-in-program-termination-cpp.md) -Anweisung vom `main`.
