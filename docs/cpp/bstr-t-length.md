**Microsoft-spezifisch**

Gibt die Anzahl von Zeichen in `_bstr_t` ohne das abschließende NULL-Zeichen des gekapselten `BSTR` zurück.

## <a name="syntax"></a>Syntax

```
unsigned int length ( ) const throw( );
```
