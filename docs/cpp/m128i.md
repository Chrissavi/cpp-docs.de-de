# <a name="__m128i"></a>__m128i

**Microsoft Specific**

Der **__m128i** Datentyp, für die Verwendung mit den Streaming SIMD Extensions \<2 (SSE2) Anweisungen intrinsics, ist in emmintrin.h> definiert.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Bemerkungen

Sie sollten nicht direkt auf die **felder __m128i** zugreifen. Sie können diese Typen jedoch im Debugger finden. Eine Variable vom Typ **__m128i** den XMM[0-7]-Registern zugeordnet.

Variablen des Typs **__m128i** werden automatisch an 16-Byte-Grenzen ausgerichtet.

> [!NOTE]
> Die Verwendung von Variablen vom Typ **__m128i** `movdqa` bewirkt, dass der Compiler die SSE2-Anweisung generiert. Diese Anweisung verursacht keinen Fehler auf Pentium III Prozessoren, sondern führt zu einem `movdqa` stillen Ausfall, mit möglichen Nebenwirkungen, die durch die Anweisungen verursacht werden, die auf Pentium III Prozessoren übersetzt werden.

Der **__m128i** Datentyp wird auf ARM-Prozessoren nicht unterstützt.

**END Microsoft Spezifisch**
