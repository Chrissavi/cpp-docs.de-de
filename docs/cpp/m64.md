# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Microsoft-spezifisch  
 Die **__m64** -Datentyp ist für die Verwendung mit MMX und 3DNow! systeminternen Funktionen ein, und in \<xmmintrin.h >.  
  
```cpp 
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Hinweise  
 Greifen Sie auf keine der **__m64** direkt Felder. Sie können diese Typen jedoch im Debugger finden. Eine Variable vom Typ **__m64** wird den MM [0-7]-Registern zugeordnet.  
  
 Variablen vom Typ **_m64** werden automatisch an 8-Byte-Grenzen ausgerichtet.  
  
 Die **__m64** -Datentyp wird nicht unterstützt, auf [!INCLUDE[vcprx64](../assembler/inline/includes/vcprx64_md.md)] Prozessoren. Anwendungen, die __m64 als Teil der systeminternen MMX-Funktionen verwenden, müssen zur Verwendung der systeminternen Funktionen SSE und SSE2 umgeschrieben werden.  
  
**Ende Microsoft-spezifisch**
