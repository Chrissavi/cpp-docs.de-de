# <a name="numerical-limits-c"></a>Numerische Grenzen (C++)

Die beiden standardmäßigen Includedateien, \<Limits. h > und \<float. h->, definieren die numerischen Grenzwerte oder die minimalen und maximalen Werte, die eine Variable eines bestimmten Typs enthalten kann. Diese Mindestgebühren und Höchstwerte sind garantiert für jeden C++ Compiler portabel, der dieselbe Datendarstellung wie ANSI C verwendet. Die \<Limits. h > Includedatei definiert die [numerischen Grenzwerte für](../cpp/integer-limits.md)ganzzahlige Typen, und \<float. h-> definiert die [numerischen Begrenzungen für Gleit Komma Typen](../cpp/floating-limits.md).
