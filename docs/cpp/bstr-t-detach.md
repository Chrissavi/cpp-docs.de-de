# <a name="_bstr_tdetach"></a>_bstr_t::Detach

**Microsoft-spezifisch**

Gibt `BSTR` zurück, das von `_bstr_t` umschlossen ist, und trennt `BSTR` von `_bstr_t`.

## <a name="syntax"></a>Syntax

```
BSTR Detach( ) throw;
```

## <a name="return-value"></a>Rückgabewert

Der `BSTR`-Code, der mit `_bstr_t` umschlossen wurde.

## <a name="example"></a>Beispiel

Ein Beispiel für die Verwendung von **Detach**finden Sie unter [_bstr_t:: Assign](../cpp/bstr-t-assign.md) .
