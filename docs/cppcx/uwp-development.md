# <a name="uwp-development-with-c"></a>UWP-Entwicklung mit C++

Die universelle Windows-Plattform (UWP) ist die moderne Programmierschnittstelle für Windows. Mit der UWP, wenn Sie eine Anwendung oder Komponente einmal schreiben und auf Windows 10-Geräten bereitstellen. Sie können eine Komponente in C++ schreiben und in einer beliebigen anderen UWP-kompatiblen Sprache geschriebene Anwendungen können Sie sie.

Die meisten der UWP-Dokumentation befindet sich in der Windows-Inhaltsstruktur an [Dokumentation für die universelle Windows-Plattform](/windows/uwp/). Dort sehen Sie Anfang Tutorials sowie Referenzdokumentation. 

Für neue UWP-apps und Komponenten, empfehlen wir die Verwendung von [C++ / WinRT](/windows/uwp/cpp-and-winrt-apis/), einen neuen C ++ 17-standardsprachprojektion für Windows-Runtime-APIs. C++ / WinRT finden Sie in das Windows 10 SDK Version 1803 gerechnet. C++ / WinRT ist nur in Headerdateien implementiert wird, und soll Sie mit erstklassigen Zugriff auf die moderne Windows-API bereitstellen.

Der Inhalt an dieser Stelle werden zwei älteren UWP-Technologien behandelt, für C++:

- die [C++ / CX-spracherweiterung](visual-c-language-reference-c-cx.md)
- die [Windows Runtime C++-Vorlagenbibliothek (WRL)](../windows/windows-runtime-cpp-template-library-wrl.md).

Beide dieser Technologien werden weiterhin unterstützt, aber im Allgemeinen nicht für die neue Entwicklung empfohlen werden.
