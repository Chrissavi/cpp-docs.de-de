# <a name="namespaces-reference-ccx"></a>Namespaceverweis (C++/CX)

In den Artikeln in diesem Abschnitt der Dokumentation werden Namespaces beschrieben, die den Compiler C++für/CX. unterstützen.

## <a name="compiler-supplied-namespaces"></a>Vom Compiler bereitgestellte Namespaces

Um das Programmieren von Programmen zu vereinfachen, die auf das C++Windows-Runtime abzielen, stellen der/CX-Compiler und seine unterstützenden Header Dateien Namespaces zur Verfügung, die einen breiten Bereich von Typen definieren. Die Namespaces definieren die integrierten numerischen Typen. Zeichen folgen, Arrays und Auflistungen; C++ Ausnahmen, die Windows-Runtime Fehler darstellen. und sprachspezifische Erweiterungen zu Standard-Windows-Runtime Typen.
