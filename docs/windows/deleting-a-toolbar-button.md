# <a name="deleting-a-toolbar-button"></a>Löschen einer Symbolleisten-Schaltfläche

### <a name="to-delete-a-toolbar-button"></a>So löschen Sie eine Symbolleisten-Schaltfläche

1. Wählen Sie die Symbolleisten-Schaltfläche, und ziehen Sie es aus der Symbolleiste.

Informationen zum Hinzufügen von Ressourcen zu verwalteten Projekten finden Sie unter [Ressourcen in Desktop-Apps](/dotnet/framework/resources/index) in die *(.NET Framework Developer's Guide*. Weitere Informationen zum manuellen Hinzufügen von Ressourcendateien zu verwalteten Projekten, den Zugriff auf Ressourcen, zum Anzeigen statischer Ressourcen und Zuweisen von Ressourcenzeichenfolgen zu Eigenschaften, finden Sie unter [Erstellen von Ressourcendateien für Desktop-Apps](/dotnet/framework/resources/creating-resource-files-for-desktop-apps).
