# <a name="flipping-an-image-image-editor-for-icons"></a>Kippen eines Bilds (Bildbearbeitung für Symbole)
Sie können die kippen oder Drehen eines Bilds auf:  
  
-   Erstellen Sie ein Spiegelbild des ursprünglichen.  
  
-   Aktivieren Sie das Image auf dem Kopf stehend.  
  
-   Drehen Sie das Bild auf der rechten Seite um 90 Grad, zu einem Zeitpunkt.  
  
### <a name="to-flip-the-image-horizontally-mirror-image"></a>Das Bild horizontal gekippt (Spiegelbild)  
  
1.  Von der **Image** Menü wählen **Horizontal kippen**.  
  
### <a name="to-flip-the-image-vertically-turn-upside-down"></a>Das Bild vertikal gekippt (Kopf)  
  
1.  Von der **Image** Menü wählen **vertikal kippen**.  
  
### <a name="to-rotate-the-image-90-degrees"></a>Um das Bild um 90 Grad drehen  
  
1.  Von der **Image** Menü wählen **um 90 Grad drehen**.  
  
    > [!NOTE]
    >  Sie können auch die [Zugriffstasten (Tastenkombination)](../windows/accelerator-keys-image-editor-for-icons.md) für diese Befehle oder Zugriff auf die Befehle im Kontextmenü (klicken Sie außerhalb des Abbilds, in dem Bild-Editor).
