# <a name="inserting-a-space-between-buttons-on-a-toolbar"></a>Einfügen eines Abstands zwischen den Schaltflächen einer Symbolleiste
Im Allgemeinen um ein Leerzeichen zwischen den Schaltflächen einzufügen, einfach ziehen Sie sie von anderen auf der Symbolleiste. Um Speicherplatz zu entfernen, ziehen Sie sie zueinander.  
  
### <a name="to-insert-a-space-before-a-button-that-is-not-followed-by-a-space"></a>Um ein Leerzeichen vor einer Schaltfläche einzufügen, die nicht durch ein Leerzeichen folgt  
  
1.  Ziehen Sie die Schaltfläche rechts oder nach unten Sie, bis sie die Schaltfläche "Weiter" während des laufenden Vorgangs zu überlappt.  
  
### <a name="to-insert-a-space-before-a-button-which-is-followed-by-a-space-and-to-retain-that-trailing-space"></a>Ein Leerzeichen vor eine Schaltfläche eingefügt, von einem Leerzeichen gefolgt, und diese Leerzeichen beibehalten werden sollen.  
  
1.  Ziehen Sie die Schaltfläche am rechten oder unteren Rand der Schaltfläche "Weiter" ist nur berühren, oder nur geringfügig überlappt.  
  
### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-close-up-that-following-space"></a>Fügen Sie kein Leerzeichen, bevor Sie eine Schaltfläche, die durch ein Leerzeichen folgt und schließen Sie den folgenden geladen  
  
1.  Ziehen Sie die Schaltfläche rechts oder nach unten Sie, bis sie die Schaltfläche "Weiter" während des laufenden Vorgangs zu überlappt.
