# <a name="evening-the-spacing-between-controls"></a>Ausgleichen des Abstands zwischen Steuerelementen
Dialog-Editor können Sie Speicherplatz Steuerelemente gleichmäßig zwischen den äußersten ausgewählten Steuerelemente.  
  
### <a name="to-even-the-spacing-between-controls"></a>Sogar den Abstand zwischen Steuerelementen  
  
1.  Wählen Sie die Steuerelemente, die Sie neu anordnen möchten.  
  
2.  Aus der **Format** Menü wählen **gleichmäßig**, und wählen Sie dann eine der folgenden Abstand Ausrichtungen:  
  
    -   **Über**: Leerzeichen Steuerelemente gleichmäßig zwischen den am weitesten links stehende und dem äußeren rechten Steuerelement ausgewählt.  
  
    -   **Nach unten**: verteilt die Steuerelemente gleichmäßig zwischen dem obersten und untersten Steuerelements ausgewählt.
