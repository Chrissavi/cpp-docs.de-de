# <a name="ole-db-consumer-attributes"></a>OLE DB-Consumerattribute

Die OLE DB-Consumerattribute fügen Code basierend auf den [OLE DB Consumer Templates](../../data/oledb/ole-db-consumer-templates-reference.md)ein, um einen funktionierenden OLE DB-Consumer zu erstellen, der Aufgaben wie das Öffnen von Tabellen, das Ausführen von Befehlen und den Zugriff auf Daten ausführt.

|Attribut|BESCHREIBUNG|
|---------------|-----------------|
|[db_accessor](db-accessor.md)|Bindet Spalten in einem Rowset und bindet sie an die entsprechenden Accessorzuordnungen.|
|[db_column](db-column.md)|Bindet eine angegebene Spalte an das Rowset.|
|[db_command](db-command.md)|Führt einen OLE DB-Befehl aus.|
|[db_param](db-param.md)|Ordnet die angegebene Membervariable einem Eingabe- oder Ausgabeparameter zu.|
|[db_source](db-source.md)|Erstellt und kapselt eine Verbindung über einen Anbieter zu einer Datenquelle.|
|[Db_table](db-table.md)|Öffnet eine OLE DB-Tabelle.|
