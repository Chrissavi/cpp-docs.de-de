# <a name="displaybind"></a>displaybind

Gibt eine Eigenschaft an, die dem Benutzer als Bindable angezeigt werden soll.

## <a name="syntax"></a>Syntax

```cpp
[displaybind]
```

## <a name="remarks"></a>Bemerkungen

Das **Display BIND** C++-Attribut verfügt über die gleiche Funktionalität wie das [attributebind](/windows/win32/Midl/displaybind) -Attribut "Mittel l".

## <a name="example"></a>Beispiel

Ein Beispiel für die Verwendung von **Display BIND**finden Sie im Beispiel für [bindbare](bindable.md) .

| Attribut Kontext | Wert |
|-|-|
|**Zielgruppe**|Schnittstellenmethode|
|**REPEATABLE**|Nein|
|**Erforderliche Attribute**|Keine|
|**Ungültige Attribute**|Keine|
