# <a name="unique-c"></a>unique (C++)

Gibt einen eindeutigen Zeiger an.

## <a name="syntax"></a>Syntax

```cpp
[unique]
```

## <a name="remarks"></a>Bemerkungen

Das **eindeutige** C++-Attribut verfügt über die gleiche Funktionalität wie das [eindeutige](/windows/win32/Midl/unique) Mittel-Attribut.
