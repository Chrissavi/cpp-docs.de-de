# <a name="attributes-by-group"></a>Attribute nach Gruppen

Die C++ Attribute sind in die folgenden funktionalen Gruppen gegliedert.

|attribute|BESCHREIBUNG|
|---------------|-----------------|
|[COM-Attribute](com-attributes.md)|Fügen Sie Code zur Unterstützung zahlreicher Bereiche der com-Entwicklung und .NET Framework Common Language Runtime Entwicklung ein.|
|[IDL-Attribute](idl-attributes.md)|Aktivieren Sie die Änderung der IDL-Datei aus einer Quell Code Datei ohne einen Assistenten und ohne Kenntnis der Struktur und Syntax dieser Datei.|
|[OLE DB-Consumerattribute](ole-db-consumer-attributes.md)|Einfügen von Code – basierend auf den [OLE DB Consumervorlagen](../../data/oledb/ole-db-consumer-templates-reference.md)– zum Erstellen eines funktionierenden OLE DB Consumers, der Aufgaben wie das Öffnen von Tabellen, das Ausführen von Befehlen und das Zugreifen auf Daten ausführt.|
|[Compilerattribute](compiler-attributes.md)|Wird vom Microsoft C++ -Compiler bereitgestellt.|

Informationen dazu, wie Sie benutzerdefinierte Attribute für Ihre .NET Framework-Anwendung definieren, finden Sie unter [benutzerdefinierte Attribute](../../extensions/user-defined-attributes-cpp-component-extensions.md).
