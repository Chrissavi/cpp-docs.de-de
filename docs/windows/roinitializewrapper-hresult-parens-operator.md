# <a name="roinitializewrapperhresult-operator"></a>RoInitializeWrapper::HRESULT()-Operator

## <a name="syntax"></a>Syntax  
  
```cpp  
operator HRESULT()  
```  
  
## <a name="requirements"></a>Anforderungen

 **Namespace:** Microsoft::WRL::Wrappers
