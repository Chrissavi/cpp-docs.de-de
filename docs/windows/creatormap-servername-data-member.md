# <a name="creatormapservername-data-member"></a>CreatorMap::serverName-Datenmember

## <a name="syntax"></a>Syntax

```cpp
const wchar_t* serverName;
```

## <a name="remarks"></a>Hinweise

Speichert den Servernamen für die CreatorMap an.
