# <a name="creating-an-empty-windows-desktop-application"></a>Erstellen einer leeren Windows-Desktopanwendung

### <a name="to-create-an-empty-windows-desktop-application"></a>So erstellen Sie eine leere Windows-Desktopanwendung

1. Wählen Sie im Menü **Datei** den Befehl **Neu** > **Projekt** aus.

2. Klicken Sie im linken Bereich des Dialogfelds **Neues Projekt** auf **Win32** , und wählen Sie im mittleren Bereich **Win32-Konsolenanwendung**aus.

3. Geben Sie einen Namen für das neue Projekt sowie einen Pfad zum Projektverzeichnis ein, und klicken Sie auf **OK**.

4. Klicken Sie im [Win32-Anwendungs-Assistenten](../windows/win32-application-wizard.md)auf die Seite **Anwendungseinstellungen** . Wählen Sie zunächst den **Anwendungstyp** aus, den Sie mit der Quellcodedatei erstellen möchten, und aktivieren Sie dann unter **Zusätzliche Optionen** das Kontrollkästchen **Leeres Projekt**.

5. Klicken Sie auf **OK**.

Das Projekt wird im **Projektmappen-Explorer** mit drei Verzeichnissen angezeigt, in denen Quelldateien, Headerdateien und Ressourcendateien gespeichert werden.

Im nächsten Schritt können Sie [dem leeren Visual C++-Projekt Dateien hinzufügen](../windows/adding-files-to-an-empty-win32-applications.md).
