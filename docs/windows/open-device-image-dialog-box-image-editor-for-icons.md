# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>Open &lt;Gerät&gt; Bildeigenschaften (Dialogfeld) (C++) (Bildbearbeitung für Symbole)

Verwenden Sie das Dialogfeld zu öffnen, um Gerätebildern in C++-Projekten zu öffnen. Vorhandene Images mit der Geräte in der aktuellen Ressourcendatei (Bilder, die Teil der aktuellen Ressource) aufgeführt.

### <a name="current-images"></a>Aktuellen Images

Listet die Images, die in der Ressource enthalten. Wählen Sie den Image-Typ, die, den Sie öffnen möchten.

## <a name="requirements"></a>Anforderungen

Keiner
