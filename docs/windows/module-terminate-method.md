# <a name="moduleterminate-method"></a>Module::Terminate-Methode

## <a name="syntax"></a>Syntax  
  
```  
void Terminate();  
```  
  
## <a name="remarks"></a>Hinweise  
 Gibt die Factorys im Cache frei.
