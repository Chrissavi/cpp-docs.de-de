# <a name="text-tool-dialog-box-image-editor-for-icons"></a>Text-Tool (Dialogfeld) (Bildbearbeitung für Symbole)
Verwenden der **Texttool** (Dialogfeld), um einen Cursor, Bitmap oder Symbol Ressource Text hinzuzufügen.  
  
 Um dieses Dialogfeld zuzugreifen, öffnen die [Bildbearbeitung](../windows/window-panes-image-editor-for-icons.md). Wählen Sie **Tools** aus der **Image** Menü, und wählen Sie dann die **Texttool** Befehl.  
  
 **Schaltfläche "Schriftart"**  
 Öffnet die [Schriftart für Texttool (Dialogfeld)](../windows/text-tool-font-dialog-box-image-editor-for-icons.md), in dem Sie die Schriftart, Format oder die Cursor Schriftgrad ändern können. Änderungen werden auf den in den Textbereich angezeigten Text angewendet.  
  
 **Textbereich**  
 Zeigt den Text, der als Teil der Ressource angezeigt wird. Anfangs ist dieser Bereich leer.  
  
> [!NOTE]
>  Wenn **transparenten Hintergrund** festgelegt ist, wird nur der Text in das Image platziert werden. Wenn **nicht transparenter Hintergrund** festgelegt ist, wird ein umschließendes Rechteck, das ausgefüllt, mit der [Hintergrundfarbe](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), hinter dem Text eingefügt wird. Weitere Informationen finden Sie unter [Auswählen eines transparenten oder deckenden Hintergrundes](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md).  
  
 Sie können mit der rechten Maustaste auf die **Texttool** (Dialogfeld), auf ein Standard-Kontextmenü zuzugreifen, die eine Liste der standard-Windows-Befehle enthält.
