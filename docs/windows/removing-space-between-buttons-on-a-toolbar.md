# <a name="removing-space-between-buttons-on-a-toolbar"></a>Entfernen eines Abstands zwischen den Schaltflächen einer Symbolleiste
Das folgende Verfahren erfahren Sie, wie Sie ein Leerzeichen zwischen den Schaltflächen einer Symbolleiste zu entfernen.  
  
### <a name="to-remove-a-space-between-buttons-on-a-toolbar"></a>So entfernen Sie ein Leerzeichen zwischen den Schaltflächen einer Symbolleiste  
  
1.  Ziehen Sie die Schaltfläche auf einer Seite des Speicherplatzes auf die Schaltfläche auf der anderen Seite für den Speicherplatz, bis sie die Schaltfläche "Weiter", etwa um die Hälfte überlappt.  
  
     Wenn ist kein Platz im Zweifelsfall die Schaltfläche, der Sie außerhalb der Reichweite von ziehen und Sie die Schaltfläche mit den mehr als zur Hälfte hinter die benachbarte Schaltfläche ziehen, fügt der Symbolleisten-Editor auch ein Leerzeichen auf der entgegengesetzten Seite der Schaltfläche, die Sie ziehen.
