# <a name="synclocktunlock-method"></a>SyncLockT::Unlock-Method
Unterstützt die WRL-Infrastruktur und ist nicht direkt aus Ihrem Code verwendet werden soll.  
  
## <a name="syntax"></a>Syntax  
  
```  
void Unlock();  
```
