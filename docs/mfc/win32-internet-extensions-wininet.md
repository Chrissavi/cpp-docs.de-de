# <a name="win32-internet-extensions-wininet"></a>Win32-Interneterweiterungen (WinInet)
Eine Internetclientanwendung ist ein Programm, das Informationen aus einer Netzwerk-Datenquelle (Server) über Internet-Protokolle wie Gopher, FTP oder HTTP-zugreift. Eine Internetclientanwendung kann es sich um einen Server zum Abrufen von Daten, z. B. Weather Maps, Aktienkurse oder "Zeitung" Schlagzeilen, z. B. zugreifen. Der Internet-Client kann den Server über ein externes Netzwerk (Internet) oder eines internen Netzwerks (Intranet bezeichnet) zugreifen.  
  
 Win32-Interneterweiterungen oder WinInet, werden von MFC zum Erstellen einer Internetclientanwendung enthält. MFC kapselt diese interneterweiterungen in einem Satz von standard, einfach zu verwendende Klassen. Sie können eine WinInet-Clientanwendung schreiben, durch den Aufruf der Win32-Funktionen direkt oder durch Verwenden der MFC-WinInet-Klassen.  
  
 Die Funktionen der Microsoft Win32-Internet (WinInet), die Sie treffen im Internet einen wesentlicher Bestandteil jeder Anwendung unterstützen. Die neuen Funktionen in WININET enthalten sind. DLL, den Zugriff auf das Internet mithilfe von HTTP (Hypertext Transfer Protocol), FTP (File Transfer Protocol) und Gopher vereinfachen.
