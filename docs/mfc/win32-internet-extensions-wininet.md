# <a name="win32-internet-extensions-wininet"></a>Win32-Interneterweiterungen (WinInet)

Eine Internet-Clientanwendung ist ein Programm, das auf die Informationen aus einer Netzwerk-Datenquelle (Server) über Internet-Protokolle wie Gopher, HTTP oder FTP zugreift. Eine Internetclientanwendung kann es sich um einen Server zum Abrufen von Daten wie z. B. Wetterkarten, Aktienkurse oder Pressemeldungen, z. B. zugreifen. Der Internetclient kann der Server über ein externes Netzwerk (Internet) oder ein internes Netzwerk (auch als Intranet bezeichnet) zugreifen.

Win32-Interneterweiterungen oder WinInet, werden von MFC zum Erstellen einer Internetclientanwendung enthält. MFC kapselt diese Internet-Erweiterungen in einer Reihe von standardmäßigen, leicht zu bedienende Klassen. Sie können eine WinInet-Clientanwendung schreiben, durch Aufruf der Win32-Funktionen direkt oder durch Verwenden der MFC-WinInet-Klassen.

Die Funktionen der Microsoft Win32 Internet (WinInet), die Sie bei der Erstellung im Internets eines wesentlicher Bestandteil jeder Anwendung unterstützen. Die neuen Funktionen in WININET enthalten sind. DLL, den Zugriff auf das Internet per, Gopher, HTTP (Hypertext Transfer Protocol) und FTP (File Transfer Protocol) vereinfachen.
