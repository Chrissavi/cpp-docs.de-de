# <a name="using-ccomboboxex"></a>Verwenden von CComboBoxEx

Die erweiterten Kombinationsfeld-Steuerelement ist eine Erweiterung der standard-Kombinationsfeld-Steuerelements, die systemeigenen Unterstützung für Elementbilder bereitstellt. Diese Images können verwendet werden, um den Status der einzelnen Elemente im Kombinationsfeld, z. B. die derzeit ausgewählte und nicht ausgewählte Elemente anzugeben. Um Elementbilder leicht zugänglich zu machen, bietet das Steuerelement die Liste unterstützt.

Verwenden Sie dieses Steuerelement, um die Funktionalität eines Kombinationsfelds bereitzustellen, ohne manuell zeichnen.
