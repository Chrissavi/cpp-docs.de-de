# <a name="using-ccomboboxex"></a>Verwenden von CComboBoxEx

Das erweiterte Kombinations Feld-Steuerelement ist eine Erweiterung des Standard-Kombinations Feld-Steuer Elements, das systemeigene Unterstützung für Element Bilder bereitstellt. Diese Bilder können verwendet werden, um den Status einzelner Elemente im Kombinations Feld anzugeben, z. b. die aktuell ausgewählten und nicht ausgewählten Elemente. Damit Element Bilder problemlos zugänglich sind, bietet das-Steuerelement Unterstützung für Bildlisten.

Verwenden Sie dieses Steuerelement, um die Funktionalität eines Kombinations Felds bereitzustellen, ohne die Element Grafiken manuell zeichnen zu müssen.
