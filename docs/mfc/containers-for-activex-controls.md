# <a name="containers-for-activex-controls"></a>Container für ActiveX-Steuerelemente

Sie können ActiveX-Steuerelemente verwenden, die in Visual C++ in anderen Anwendungen entwickelt wurden, sofern Sie die Containment-Steuerelement Kapselung unterstützen Eine Reihe von Microsoft-Anwendungen, beginnend mit den aufgelisteten Versionen, unterstützen die Unterstützung des ActiveX-Steuer Elements.

>[!IMPORTANT]
> ActiveX ist eine ältere Technologie, die nicht für die neue Entwicklung verwendet werden sollte. Weitere Informationen zu modernen Technologien, die ActiveX ersetzen, finden Sie unter ActiveX-Steuer [Elemente](activex-controls.md).

> [!NOTE]
> Die folgende Liste ist keine komplette Liste von Anwendungen, die ActiveX-Steuerelemente unterstützen, aber die am häufigsten beim Testen verwendete Gruppe darstellt:

Folgende Anwendungen unterstützen die Unterstützung der ActiveX-Steuerelement

- Internet Explorer 3. x und höher

- Visual Basic 4. x und höher

- Visual C++ 4. x und höher

- Zugriff 95 und höher

- Excel 97 und höher

- Word 97 und höher

- Zugriff 97 und höher

- FrontPage 97 und höher

- PowerPoint 97 und höher

- Visual InterDev 97 und höher

Folgendes sind nicht-Microsoft-Anwendungen, die die Unterstützung der ActiveX-Steuerung unterstützen:

- PowerBuilder

- Del

- C++-Generator

- NCompass-Plug-in für Netscape-Navigator
