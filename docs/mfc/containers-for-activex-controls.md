# <a name="containers-for-activex-controls"></a>Container für ActiveX-Steuerelemente

Sie können ActiveX-Steuerelemente verwenden, die in Visual C++ in anderen Anwendungen entwickelt wurden, sofern sie ActiveX-Steuerelementbeschränkung unterstützen. Eine Reihe von Microsoft-Anwendungen, beginnend mit den aufgeführten Versionen, unterstützen die ActiveX-Steuerelementbeschränkung.

>[!IMPORTANT]
> ActiveX ist eine legacy Technologie, die nicht für Neuentwicklungen verwendet werden sollte. Weitere Informationen zu modernen Technologien, die ActiveX ablösen, finden Sie unter [ActiveX Controls](activex-controls.md).

> [!NOTE]
> Die folgende Liste enthält keine vollständige Liste von Anwendungen, die ActiveX-Steuerelemente unterstützen, stellt jedoch den Satz dar, der am häufigsten beim Testen verwendet wird:

Zu den Anwendungen, die Die ActiveX-Steuerelementbeschränkung unterstützen, gehören:

- Internet Explorer 3.x und höher

- Visual Basic 4.x und höher

- Visual C++ 4.x und höher

- Zugang 95 und höher

- Excel 97 und höher

- Word 97 und höher

- Zugang 97 und höher

- FrontPage 97 und höher

- PowerPoint 97 und höher

- Visual InterDev 97 und höher

Im Folgenden finden Sie Nicht-Microsoft-Anwendungen, die ActiveX-Steuerelementbeschränkung unterstützen:

- PowerBuilder

- Delphi

- C++-Generator

- NCompass Plug-in für Netscape Navigator
