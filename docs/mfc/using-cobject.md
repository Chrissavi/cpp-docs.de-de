# <a name="using-cobject"></a>Verwenden von CObject

[CObject](../mfc/reference/cobject-class.md) ist die Basisklasse für die meisten Microsoft Foundation Class-Bibliothek (MFC). Die `CObject`-Klasse enthält viele nützliche Funktionen, die Sie möglicherweise in Ihre eigenen Programmobjekte integrieren möchten, einschließlich der Serialisierungsunterstützung, Lauf Zeit Klassen Informationen und der Objekt Diagnoseausgabe. Wenn Sie Ihre Klasse von `CObject`ableiten, kann die Klasse diese `CObject` Features ausnutzen.
