# <a name="active-technology-on-the-internet"></a>Active Technology für das Internet

Die aktive Technologie ist eine offene Plattform, mit der Entwickler spannende, dynamische Inhalte und Anwendungen für das globale Internet oder für das interne Netzwerk eines Unternehmens, das als Intranet bezeichnet wird, erstellen können. Die wichtigsten Technologien von Microsoft für die Internet Programmierung werden unten beschrieben.

>[!IMPORTANT]
> ActiveX ist eine ältere Technologie, die nicht für die neue Entwicklung verwendet werden sollte. Weitere Informationen zu modernen Technologien, die ActiveX ersetzen, finden Sie unter ActiveX-Steuer [Elemente](activex-controls.md).

## <a name="activex-controls"></a>ActiveX-Steuerelemente

ActiveX-Steuerelemente (früher OLE-Steuerelemente) sind Objekte, die in Webseiten oder eine beliebige andere Anwendung, die ein ActiveX-Steuerelement Container ist, eingefügt werden können. Beispiele hierfür sind Schaltflächen, Börsen Ticker und Diagramm Steuerelemente. Weitere Informationen finden Sie unter [ActiveX-Steuerelemente im Internet](activex-controls-on-the-internet.md).

## <a name="internet-data-download-services"></a>Internet Data Download-Dienste

Daten können über das Internet mithilfe allgemeiner Protokolle heruntergeladen werden: http, FTP und Gopher. Die MFC-WinInet-Klassen vereinfachen das Übertragen von Daten mithilfe von http-, FTP-und Gopher-Protokollen durch abstrahieren der TCP/IP-und Winsock-Protokolle. Die asynchronen MFC-Monikerklassen bieten eine Möglichkeit zum Herunterladen von Dateien ohne Blockierung und zum asynchronen Rendering von großen Objekten. Weitere Informationen finden Sie unter [Win32-Internet Erweiterungen (WinInet)](win32-internet-extensions-wininet.md).

## <a name="active-scripts"></a>Aktive Skripts

VBScript und andere Skriptsprachen verbinden Steuerelemente und fügen den Webseiten interaktive Funktionalität hinzu. Die Skripterstellung verschiebt die Verarbeitung vom Server auf den Client. Formulareinträge können z. b. auf dem Client überprüft und dann an den Server gesendet werden.

## <a name="html-extensions"></a>HTML-Erweiterungen

HTML-Erweiterungen, wie das Objekttag, wurden hinzugefügt, um Steuerelemente und Skripts zu unterstützen.
