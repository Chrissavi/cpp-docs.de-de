# <a name="root-class-cobject"></a>Stammklasse: CObject

Die meisten Klassen in der MFC-Bibliothek (Microsoft Foundation Class) werden von einer einzelnen Basisklasse im Stammverzeichnis der Klassenhierarchie abgeleitet. `CObject` bietet eine Reihe nützlicher Funktionen für alle von ihm abgeleiteten Klassen mit sehr geringem mehr Aufwand. Weitere Informationen zu `CObject` und den zugehörigen Funktionen finden [Sie unter Verwenden von CObject](../mfc/using-cobject.md).

[CObject](../mfc/reference/cobject-class.md)<br/>
Die ultimative Basisklasse der meisten MFC-Klassen. Unterstützt das Serialisieren von Daten und das Abrufen von Laufzeitinformationen zu einer Klasse.

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
-Struktur, die verwendet wird, um die exakte Klasse eines Objekts zur Laufzeit zu bestimmen.
