# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Tracker: Implementieren von Trackern in einer OLE-Anwendung

Tracker bieten eine grafische Benutzeroberfläche, damit Benutzer mit OLE-Client-Elementen interagieren können. Mit Tracker für die verschiedenen Stilen OLE-Client-Elemente mit schraffierten Rahmen, angezeigt werden können Größe ändern, Handles oder einer Vielzahl von anderen visuellen Effekten. Dieser Artikel beschreibt:

- [Gewusst wie: Implementieren der nachverfolgung im Code](../mfc/how-to-implement-tracking-in-your-code.md).

- [Gummibandtechnik und Tracker](../mfc/rubber-banding-and-trackers.md).

Der Artikel befasst sich auch die Verwendung von Stilen mit Tracker. Darüber hinaus macht es mehrere Verweise auf das MFC-OLE-Beispiel [OCLIENT](../visual-cpp-samples.md).
