# <a name="using-your-old-toolbars"></a>Verwenden der bisherigen Symbolleisten

Wenn Sie frühere Versionen von Visual C++ verwendet haben, erstellen Sie benutzerdefinierte Symbolleisten, die neue Implementierung der Klasse [CToolBar](../mfc/reference/ctoolbar-class.md) Sie zu Problemen führen kann. Damit Sie nicht der bisherigen Symbolleisten freizugeben, verwenden Sie die neue Funktionen haben, wird die alte Implementierung noch unterstützt.

Die alten Symbolleisten, nur die neuen Formats Symbolleisten verwendet DOCKTOOL Beispiel nicht.

Sie können nicht herkömmliche Symbolleisten mit der Symbolleisten-Ressourcen-Editor bearbeiten.
