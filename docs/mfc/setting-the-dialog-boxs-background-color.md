# <a name="setting-the-dialog-boxs-background-color"></a>Festlegen der Hintergrundfarbe für das Dialogfeld
Sie können die Farbe des Hintergrunds in den Dialogfeldern durch Verarbeiten von WM_CTLCOLOR-Nachrichten für das Dialogfeld festlegen. Die Farbe, die Sie festlegen, wird nur die im angegebenen Dialogfeld verwendet.
