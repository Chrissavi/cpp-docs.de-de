# <a name="setting-the-dialog-boxs-background-color"></a>Festlegen der Hintergrundfarbe für das Dialogfeld

Sie können die Hintergrundfarbe der Dialogfelder festlegen, indem Sie WM_CTLCOLOR-Meldungen für das Dialogfeld Fenster verarbeiten. Die Farbe, die Sie festlegen, wird nur für das angegebene Dialogfeld verwendet.

Ein Beispiel finden Sie im [codexpert-Blog](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) .
