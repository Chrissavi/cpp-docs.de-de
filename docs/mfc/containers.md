# <a name="containers"></a>Container

Eine Containeranwendung ist eine Anwendung, die eingebettete oder verknüpfte Elemente in Ihre eigenen Dokumente integrieren kann. Die von einer Containeranwendung verwalteten Dokumente müssen OLE-Verbund Dokument Komponenten und von der Anwendung selbst erstellte Daten speichern und anzeigen können. Eine Containeranwendung muss außerdem Benutzern das Einfügen neuer Elemente oder das Bearbeiten vorhandener Elemente gestatten.
