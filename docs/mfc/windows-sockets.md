# <a name="windows-sockets"></a>Windows-Sockets

Diese Artikelreihe behandelt die MFC-Implementierung von Windows Sockets. MFC bietet zwei Klassen zur Unterstützung der Programmierung von netzwerkanwendungen mit der Windows Sockets-API. Klasse [CAsyncSocket](../mfc/reference/casyncsocket-class.md) kapselt der Windows Sockets-API eine für ein Modell, sodass erweiterte Netzwerk-Programmierer, die Leistungsfähigkeit und Flexibilität. Klasse [CSocket](../mfc/reference/csocket-class.md) bietet eine vereinfachte Benutzeroberfläche für das Serialisieren von Daten in und aus einem [CArchive](../mfc/reference/carchive-class.md) Objekt.
