# <a name="property-sheets-mfc"></a>Eigenschaftenblätter (MFC)

In dieser Artikelreihe wird erläutert, wie Unterstützung für Eigenschaftenseiten in MFC-Anwendungen implementieren. Ein Eigenschaftenblatt, auch bekannt als ein Dialogfeld im Registerformat, bietet eine Möglichkeit, eine große Anzahl von Steuerelementen in einem Dialogfeld verwalten. Das Eigenschaftenblatt enthält Eigenschaftenseiten, jeweils basierend auf einer separaten Dialogfeldvorlagen-Ressource. Sie können Ihrem Dialogfeld-Steuerelemente in logische Gruppen unterteilen und platzieren jede Gruppe auf ihre eigene Eigenschaftenseite.
