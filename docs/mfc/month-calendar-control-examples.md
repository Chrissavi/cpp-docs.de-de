# <a name="month-calendar-control-examples"></a>Beispiele für Monatskalender-Steuerelemente

Die [CMNCTRL1](../overview/visual-cpp-samples.md) -Beispielanwendung veranschaulicht die verschiedenen Attribute der- `CMonthCalCtrl` Klasse. Das-Steuerelement, das sich auf einer separaten Registerkarte des Beispiels befindet, veranschaulicht die grundlegende Funktionalität und ermöglicht es dem Benutzer, bestimmte Attribute dynamisch zu ändern.
