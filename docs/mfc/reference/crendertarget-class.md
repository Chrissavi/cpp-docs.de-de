##  <a name="createcompatiblerendertarget"></a>Crendertarget:: buildkompatiblerendertarget

Erstellt ein neues Bitmap-Renderziel zur Verwendung während der zwischengeschalteten Offscreen-Zeichnung, das mit dem aktuellen Renderziel kompatibel ist.

```
BOOL CreateCompatibleRenderTarget(
    CBitmapRenderTarget& bitmapTarget,
    CD2DSizeF sizeDesired = CD2DSizeF(0., 0.),
    CD2DSizeU sizePixelDesired = CD2DSizeU(0, 0),
    D2D1_PIXEL_FORMAT* desiredFormat = NULL,
    D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS options = D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS_NONE);
```

### <a name="parameters"></a>Parameters

*bitmapTarget*<br/>
Wenn diese Methode zurückgegeben wird, enthält Sie die Adresse eines Zeigers auf ein neues Bitmap-Renderziel. Dieser Parameter wird nicht initialisiert übergeben.

*sizeDesired*<br/>
Die gewünschte Größe des neuen Renderziels in geräteunabhängigen Pixeln, wenn Sie sich vom ursprünglichen Renderziel unterscheiden soll, oder NULL. Weitere Informationen finden Sie im Abschnitt "Hinweise".

*sizePixelDesired*<br/>
Die gewünschte Größe des neuen Renderziels in Pixel, wenn diese vom ursprünglichen Renderziel abweichen soll, oder NULL. Weitere Informationen finden Sie im Abschnitt "Hinweise".

*desiredFormat*<br/>
Das gewünschte Pixel Format und der Alpha Modus des neuen Renderziels oder NULL. Wenn das Pixel Format auf DXGI_FORMAT_UNKNOWN festgelegt ist oder dieser Parameter NULL ist, verwendet das neue Renderziel das gleiche Pixel Format wie das ursprüngliche Renderziel. Wenn der Alpha Modus D2D1_ALPHA_MODE_UNKNOWN ist oder dieser Parameter NULL ist, wird der Alpha-Modus des neuen Renderziels standardmäßig D2D1_ALPHA_MODE_PREMULTIPLIED. Weitere Informationen zu unterstützten Pixel Formaten finden Sie unter Unterstützte Pixel Formate und Alpha-Modi.

*options*<br/>
Ein-Wert, der angibt, ob das neue Renderziel mit GDI kompatibel sein muss.

### <a name="return-value"></a>Rückgabewert

Wenn die Methode erfolgreich ausgeführt wird, wird true zurückgegeben. Andernfalls wird false zurückgegeben.

##  <a name="drawbitmap"></a>Crendertarget::D rawbitmap

Zeichnet den formatierten Text, der vom angegebenen idschreitetextlayout-Objekt beschrieben wird.

```
void DrawBitmap(
    CD2DBitmap* pBitmap,
    const CD2DRectF& rectDest,
    float fOpacity = 1.0,
    D2D1_BITMAP_INTERPOLATION_MODE interpolationMode = D2D1_BITMAP_INTERPOLATION_MODE_LINEAR,
    const CD2DRectF* pRectSrc = NULL);
```

### <a name="parameters"></a>Parameters

*pBitmap*<br/>
Die zu Rendering-Bitmap.

*rectDest*<br/>
Die Größe und Position in geräteunabhängigen Pixeln im Koordinaten Bereich des Renderziels für den Bereich, in den die Bitmap gezeichnet wird. Wenn das Rechteck nicht ordnungsgemäß angeordnet ist, wird nichts gezeichnet, aber das Renderziel wird nicht in einen Fehlerzustand versetzt.

*fOpacity*<br/>
Ein Wert zwischen 0,0 f und 1.0 f, der einen Deckkraft Wert angibt, der auf die Bitmap angewendet werden soll. Dieser Wert wird mit den Alpha Werten des Inhalts der Bitmap multipliziert.

*interpolationMode*<br/>
Der Interpolations Modus, der verwendet werden soll, wenn die Bitmap durch den Zeichnungs Vorgang skaliert oder gedreht wird.

*pRectSrc*<br/>
Die Größe und Position des Bereichs innerhalb der zu zeichnenden Bitmap in geräteunabhängigen Pixeln im Koordinaten Bereich der Bitmap.
