# <a name="mapping-messages-to-functions"></a>Zuordnen von Meldungen zu Funktionen

Der [Klassen-Assistent](mfc-class-wizard.md) ermöglicht Ihnen das Binden von Meldungs Handlern (Member-Funktionen von MFC-Benutzerschnittstellen Klassen) an die Nachrichten, die von den Ressourcen der Anwendung generiert werden. [MFC-Nachrichten](../../mfc/messages-and-commands-in-the-framework.md) Zuordnungen werden verwendet, um die Bindung zu erstellen.

Wenn Sie Klassenansicht verwenden, um eine neue Klasse zu erstellen, die von einer der Frameworkklassen abgeleitet ist, wird automatisch eine komplette und funktionale Klasse in den Header Dateien (. h) und in Implementierungs Dateien (. cpp) eingefügt, die Sie angeben.

> [!NOTE]
>  Um eine neue Klasse hinzuzufügen, die keine Nachrichten verarbeitet, erstellen Sie die-Klasse direkt im Text-Editor.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>So definieren oder entfernen Sie einen Nachrichten Handler mithilfe des Klassen-Assistenten

1. Klicken Sie in **Klassenansicht**mit der rechten Maustaste auf die-Klasse.

1. Wählen Sie im Kontextmenü [Klassen-Assistent](mfc-class-wizard.md)aus.
