# <a name="mapping-messages-to-functions"></a>Zuordnen von Meldungen zu Funktionen

Mit dem [Klassen-Assistenten](mfc-class-wizard.md) können Sie Nachrichtenhandler (Memberfunktionen von MFC-Benutzeroberflächenklassen) an die Nachrichten binden, die von den Ressourcen Ihrer Anwendung generiert werden. Sie verwenden [MFC-Nachrichtenzuordnungen,](../../mfc/messages-and-commands-in-the-framework.md) um die Bindung zu erstellen.

Wenn Sie die Klassenansicht verwenden, um eine neue Klasse zu erstellen, die von einer der Frameworkklassen abgeleitet ist, platziert sie automatisch eine vollständige und funktionale Klasse in den von Ihnen angegebenen Headerdateien (.h) und Implementierungsdateien (.cpp).

> [!NOTE]
> Um eine neue Klasse hinzuzufügen, die keine Nachrichten verarbeitet, erstellen Sie die Klasse direkt im Texteditor.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>So definieren oder entfernen Sie einen Nachrichtenhandler mithilfe des Klassen-Assistenten

1. Klicken Sie in der **Klassenansicht**mit der rechten Maustaste auf die Klasse.

1. Wählen Sie im Kontextmenü [Klassenassistent](mfc-class-wizard.md)aus.
