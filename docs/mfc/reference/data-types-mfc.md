# <a name="data-types-mfc"></a>Datentypen (MFC)

In diesem Thema werden die Datentypen aufgelistet, die am häufigsten in der Microsoft Foundation Class-Bibliothek verwendet werden. Die meisten Datentypen sind identisch mit denen im Platform Software Development Kit (SDK), während andere für MFC eindeutig sind.

Informationen zu den Datentypen, die sowohl im Windows SDK als auch in MFC verwendet werden, finden Sie unter [Windows-Datentypen](/windows/win32/WinProg/windows-data-types).

Folgende Datentypen sind für die Microsoft Foundation Class-Bibliothek eindeutig:

- Positionieren Sie einen Wert, mit dem die Position eines Elements in einer Auflistung bezeichnet wird. wird von MFC-Auflistungs Klassen verwendet.

- Lpcrect einen 32-Bit-Zeiger auf eine Konstante (nicht änderbare `RECT` ) Struktur.
