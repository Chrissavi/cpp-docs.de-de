# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Vom MFC-DLL-Assistenten erstellte Klassen und Funktionen

Der Code, den der MFC-DLL-Assistent generiert, hängt von der Art der DLL ab, die Sie erstellen, und von den ausgewählten Optionen. Der MFC-DLL-Assistent generiert denselben Code für beide Formen regulärer MFC-DLLs.

|Art der DLL|Option|Klassen|Functions|
|-----------------|------------|-------------|---------------|
|[Erweiterung](../../build/extension-dlls-overview.md)|Keine|Keine|`DllMain`|
|[Regulär](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Keine|Anwendungsklasse abgeleitet von`CWinApp`|Keine|
|[Regulär](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Automation|Anwendungsklasse abgeleitet von`CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Erweiterung](../../build/extension-dlls-overview.md)|Fenster-Sockets|Keine|`DllMain`|
|[Regulär](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Fenster-Sockets|Anwendungsklasse abgeleitet von`CWinApp`|`InitInstance`enthält den Aufruf an`AfxSocketInit`|
