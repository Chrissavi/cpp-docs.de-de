# <a name="adding-an-mfc-class"></a>Hinzufügen einer MFC-Klasse
Verwenden Sie zum Hinzufügen von Klassen, die Bibliotheksklassen Microsoft Foundation Class (MFC) abgeleitet, dem Projekt die **Klasse hinzufügen** verfügbaren Befehle [Klassenansicht](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Geben Sie den Namen der neuen Klasse, wählen Sie die Basisklasse aus, und wählen Sie die ID des Dialogfelds, mit denen sie zugeordnet (sofern vorhanden) ist. Der Code-Assistenten eine Headerdatei und eine Implementierungsdatei erstellt und dem Projekt hinzugefügt.  
  
> [!NOTE]
>  Sie können eine ATL-COM-Anwendung MFC-Klassen hinzufügen, wenn Sie anfänglich [erstellt die Anwendung mit MFC-Unterstützung](../../atl/reference/mfc-support-in-atl-projects.md). Sie können auch MFC-Klassen hinzufügen, die Win32-Projekte, die MFC-Unterstützung verfügen.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>So fügen Sie eine MFC-Klasse zu Ihrem Projekt hinzu  
  
1.  In der Klassenansicht mit der rechten Maustaste des Namens des Projekts. Klicken Sie auf **hinzufügen** , und klicken Sie dann auf **Klasse hinzufügen** So öffnen die [Klasse hinzufügen](../../ide/add-class-dialog-box.md) (Dialogfeld).  
  
2.  Wählen Sie im Vorlagenbereich **MFC-Klasse** , und drücken Sie die **hinzufügen** Schaltfläche.  
  
3.  Definieren Sie die Einstellungen für die neue Klasse in der [MFC-Klassen-Assistent](../../mfc/reference/mfc-add-class-wizard.md) (Dialogfeld).  
  
4.  Klicken Sie auf **Fertig stellen** um den Assistenten schließen und die neue Klasse in der Klassenansicht anzuzeigen. Sie können auch anzeigen, von dem Assistenten erstellten Dateien **Projektmappen-Explorer**.
