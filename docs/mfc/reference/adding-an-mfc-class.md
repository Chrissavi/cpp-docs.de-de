# <a name="adding-an-mfc-class"></a>Hinzufügen einer MFC-Klasse

Um Klassen hinzuzufügen, die von MFC-Bibliotheksklassen (Microsoft Foundation Class) zu Ihrem Projekt abgeleitet wurden, verwenden Sie die Schaltfläche **Klasse hinzufügen** im [Klassen-Assistenten](mfc-class-wizard.md). Geben Sie den Namen der neuen Klasse an, wählen Sie die Basisklasse aus, und wählen Sie die ID des Dialogfelds aus, dem sie zugeordnet ist (falls vorhanden). Der Assistent erstellt eine Headerdatei und eine Implementierungsdatei und fügt sie Ihrem Projekt hinzu.

> [!NOTE]
> Sie können einer ATL COM-Anwendung MFC-Klassen hinzufügen, wenn Sie die Anwendung ursprünglich [mit MFC-Unterstützung erstellt](../../atl/reference/mfc-support-in-atl-projects.md)haben. Sie können auch MFC-Klassen zu Win32-Projekten hinzufügen, die MFC-Unterstützung haben.

### <a name="to-add-an-mfc-class-to-your-project"></a>So fügen Sie Ihrem Projekt eine MFC-Klasse hinzu

Klicken Sie in der Klassenansicht mit der rechten Maustaste auf den Projektnamen. Klicken Sie auf **Klassen-Assistent,** und klicken Sie dann auf **Klasse hinzufügen,** um das Dialogfeld [MFC-Klasse hinzufügen](mfc-add-class-wizard.md) zu öffnen. Verwenden Sie den **Klassen-Assistenten,** um Nachrichtenhandler oder neue Methoden hinzuzufügen.
