# <a name="adding-an-mfc-class"></a>Hinzufügen einer MFC-Klasse

Um dem Projekt von Microsoft Foundation Class (MFC)-Bibliotheks Klassen abgeleitete Klassen hinzuzufügen, verwenden Sie die Schaltfläche **Klasse hinzufügen** im [Klassen-Assistenten](mfc-class-wizard.md). Geben Sie den Namen der neuen Klasse an, wählen Sie die Basisklasse aus, und wählen Sie die ID des Dialog Felds aus, mit dem es verknüpft ist (sofern vorhanden). Der Assistent erstellt eine Header Datei und eine Implementierungs Datei und fügt Sie dem Projekt hinzu.

> [!NOTE]
>  Sie können MFC-Klassen zu einer ATL-COM-Anwendung hinzufügen, wenn Sie [die Anwendung anfänglich mit MFC-Unterstützung erstellt](../../atl/reference/mfc-support-in-atl-projects.md)haben. Sie können auch MFC-Klassen zu Win32-Projekten hinzufügen, die über MFC-Unterstützung verfügen.

### <a name="to-add-an-mfc-class-to-your-project"></a>So fügen Sie dem Projekt eine MFC-Klasse hinzu

Klicken Sie in Klassenansicht mit der rechten Maustaste auf den Projektnamen. Klicken Sie auf **Klassen-Assistent** und dann auf **Klasse hinzufügen** , um das Dialogfeld [MFC-Klasse hinzufügen](mfc-add-class-wizard.md) zu öffnen Verwenden Sie den **Klassen-Assistenten** , um Meldungs Handler oder neue Methoden hinzuzufügen.
