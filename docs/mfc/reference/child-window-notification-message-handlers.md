|Kategorie|Beschreibung|
|--------------|-----------------|
|[Generischer Steuerelementhandler](../../mfc/reference/generic-control-handler.md)|Handler für den generischen Steuerungscodes für Benachrichtigung.|
|[Handler für Benutzerschaltflächen](../../mfc/reference/user-button-handlers.md)|Handler für den Benutzer auf die Schaltfläche Notification-Codes.|

|[Handler für Bearbeitungssteuerelemente](../../mfc/reference/edit-control-handlers.md)|Der Handler für das Bearbeiten von Steuerungscodes für Benachrichtigung.|
|[Listenfeldhandler](../../mfc/reference/list-box-handlers.md)|Der Handler für Listencodes Box-Benachrichtigung.|
