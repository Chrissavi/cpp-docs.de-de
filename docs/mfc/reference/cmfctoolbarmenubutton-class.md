## <a name="remarks"></a>Hinweise

Ein `CMFCToolBarMenuButton` kann als Menü, ein Menü Element mit einem Untermenü, eine Schaltfläche, die entweder einen Befehl ausführt oder ein Menü anzeigt, oder eine Schaltfläche, die nur ein Menü anzeigt, angezeigt werden. Sie bestimmen das Verhalten und die Darstellung der Menü Schaltfläche durch Angeben von Parametern, wie z. b. dem Bild, dem Text, dem Menü Handle und der Befehls- `CMFCToolbarMenuButton::CMFCToolbarMenuButton`ID, die der Schaltfläche im Konstruktor zugeordnet ist.

Eine von der `CMFCToolbarMenuButton` -Klasse abgeleitete benutzerdefinierte Klasse muss das [DECLARE_SERIAL](run-time-object-model-services.md#declare_serial) -Makro verwenden. Das [DECLARE_DYNCREATE](run-time-object-model-services.md#declare_dyncreate) -Makro generiert einen Fehler, wenn die Anwendung geschlossen wird.

##  <a name="cmfctoolbarmenubutton"></a>Cmfctoolbarmenubutton:: cmfctoolbarmenubutton

Erstellt ein `CMFCToolBarMenuButton`-Objekt.

```
CMFCToolBarMenuButton();
CMFCToolBarMenuButton(const CMFCToolBarMenuButton& src);

CMFCToolBarMenuButton(
    UINT uiID,
    HMENU hMenu,
    int iImage,
    LPCTSTR lpszText=NULL,
    BOOL bUserButton=FALSE);
```

### <a name="parameters"></a>Parameter

*src*<br/>
in Ein vorhandenes `CMFCToolBarMenuButton` -Objekt, das in `CMFCToolBarMenuButton` dieses-Objekt kopiert werden soll.

*uiID*<br/>
in Die ID des Befehls, der ausgeführt werden soll, wenn ein Benutzer auf die Schaltfläche klickt. oder (uint)-1 für eine Menü Schaltfläche, die nicht direkt einen Befehl ausführt.

*hMenu*<br/>
in Ein Handle für ein Menü. oder NULL, wenn die Schaltfläche über kein Menü verfügt.

*iImage*<br/>
in Der Index des Bilds für die Schaltfläche. oder-1, wenn diese Schaltfläche kein Symbol enthält oder das Symbol für den Befehl verwendet, der von *uiid*angegeben wird. Der Index ist für jedes `CMFCToolBarImages` Objekt in der Anwendung identisch.

*lpszText*<br/>
in Der Text der Symbolleisten-Menü Schaltfläche.

*bUserButton*<br/>
in TRUE, wenn auf der Schaltfläche ein benutzerdefiniertes Bild angezeigt wird. FALSE, wenn die Schaltfläche ein vordefiniertes Bild anzeigt, das mit dem durch *uiid*angegebenen Befehl verknüpft ist.

### <a name="remarks"></a>Hinweise

Wenn *uiid* eine gültige Befehls-ID ist, führt die Schaltfläche diesen Befehl aus, wenn der Benutzer darauf klickt. Wenn *HMENU* ein gültiges Menü Handle ist, wird auf der Schaltfläche ein Dropdown Menü angezeigt, wenn es in einer Symbolleiste oder einem Untermenü angezeigt wird, wenn es in einem Menü angezeigt wird. Wenn sowohl *uiid* als auch *HMENU* gültig sind, ist die Schaltfläche eine Trenn Schaltfläche mit einem Teil, der den Befehl ausführt, wenn der Benutzer darauf klickt, und einen Teil mit einem Pfeil nach unten, mit dem ein Menü angezeigt wird, wenn der Benutzer darauf klickt. Wenn *HMENU* jedoch gültig ist, kann ein Benutzer nicht auf die Schaltfläche klicken, um einen Befehl auszuführen, wenn die Schaltfläche in ein Menü eingefügt wird.
