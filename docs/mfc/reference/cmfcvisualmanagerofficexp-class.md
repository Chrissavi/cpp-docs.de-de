# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP-Klasse

Die `CMFCVisualManagerOfficeXP` ermöglicht es einer Anwendung die Darstellung eines Microsoft Office XP.

## <a name="syntax"></a>Syntax

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>Member

Alle Member dieser Klasse sind virtuelle Funktionen, die von den Vorgänger dieser Klasse abgeleitet sind [CMFCVisualManager-Klasse](../../mfc/reference/cmfcvisualmanager-class.md).

## <a name="remarks"></a>Hinweise

Sie verwenden die `CMFCVisualManagerOfficeXP` Klasse so ändern Sie die visuelle Darstellung Ihrer Anwendung, die von Microsoft Office XP entsprechen.
