# <a name="editing-a-message-handler"></a>Bearbeiten eines Meldungshandlers
Nachdem Sie einen Meldungshandler definiert haben, gelangen Sie auf die Member-Funktion Definition zum Hinzufügen oder Ändern von Code.  
  
 Springen zu einer Funktionsdefinition Member aus der [Dialog-Editor](../../windows/dialog-editor.md), doppelklicken Sie auf ein Steuerelement für die bereits ein Ereignishandler definiert ist. Dies wechselt Sie in die Datei, in der das ausgewählte Steuerelement Message-Handler definiert ist.  
  
 Wechseln Sie zur Definition einer Memberfunktion aus [Klassenansicht](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), doppelklicken Sie auf den Funktionsnamen in der Klassenansicht.
