# <a name="debugging-support-classes"></a>Debugunterstützungsklassen

MFC stellt die folgenden Klassen bereit, mit denen Sie Probleme bei der dynamischen Speicher Belegung Debuggen können.

[CDumpContext](reference/cdumpcontext-class.md)<br/>
Stellt ein Ziel für diagnosedumps bereit.

[CMemoryState](reference/cmemorystate-structure.md)<br/>
-Struktur, die Momentaufnahmen der Speichernutzung bereitstellt. Wird auch verwendet, um frühere und spätere Speicher Momentaufnahmen zu vergleichen.
