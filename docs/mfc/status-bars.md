# <a name="status-bars"></a>Statusleisten

Statusleisten versehen Sie Ihre Anwendung einen Ort zum Nachrichten und nützliche Informationen für den Benutzer anzuzeigen, ohne Unterbrechung der Arbeit. In der Regel am unteren Rand eines Fensters angezeigt wird, haben Statusleisten "Bereiche", die z. b. "Indikatoren" und "Meldungszeile." Die Indikatoren geben den Status von Elementen, wie Rollen ist, Sie, ob die makroaufzeichnung aktiviert ist, und so weiter. Die Message-Zeile in der Statusleiste kann Informationen zum Programmstatus oder einer Symbolleiste oder ein Menüelement, auf die der Benutzer mit der Maus zeigt anzeigen.

Erstellen Sie eine Statusleiste in Ihrem Programm durch Auswählen der **anfängliche Statusleiste** Option im MFC-Anwendungs-Assistenten.
