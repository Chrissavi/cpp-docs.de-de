# <a name="commonly-added-member-functions"></a>Häufig hinzugefügte Memberfunktionen

Wenn das Dialogfeld die Schaltflächen "OK" oder "Abbrechen" enthält, müssen Sie in der Dialogfeld Klasse nachrichtenhandlermember-Funktionen schreiben, um auf die von Ihnen generierten Steuerelement Benachrichtigungs Meldungen zu reagieren. Ein Beispiel finden Sie im Beispielprogramm [Scribble](../overview/visual-cpp-samples.md) . Sie können auch Steuerelement Benachrichtigungs Meldungen von anderen Steuerelementen in Ihrem Dialogfeld verarbeiten.
