# <a name="application-framework"></a>Anwendungsframework

Das Kernstück der Microsoft Foundation Class-Bibliothek (MFC) ist eine Kapselung eines großen Teils der Windows-API in C++ Form. Bibliotheks Klassen stellen Fenster, Dialogfelder, Geräte Kontexte, gängige GDI-Objekte wie Pinsel und Stifte, Steuerelemente und andere standardmäßige Windows-Elemente dar. Diese Klassen stellen eine bequeme C++ Member-Funktions Schnittstelle für die Strukturen in Windows bereit, die Sie kapseln. Weitere Informationen zur Verwendung dieser Klassen finden Sie unter [Fenster Objekt Themen](window-objects.md).

Die MFC-Bibliothek bietet jedoch auch eine Ebene zusätzlicher Anwendungsfunktionen, die auf der C++-Kapselung der Windows-API basiert. Diese Ebene ist ein funktionierendes Anwendungs Framework für Windows, das den größten Teil der allgemeinen Benutzeroberfläche von Programmen für Windows bereitstellt, einschließlich Symbolleisten, Status leisten, drucken, Druckvorschau, Datenbankunterstützung und ActiveX-Unterstützung. Wenn [Sie die Klassen zum Schreiben von Anwendungen für Windows verwenden,](using-the-classes-to-write-applications-for-windows.md) wird das Framework ausführlich erläutert.
