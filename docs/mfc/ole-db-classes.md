# <a name="ole-db-classes"></a>OLE DB-Klassen

Die OLE DB-Unterstützung in MFC besteht derzeit aus der [COleDBRecordView](../mfc/reference/coledbrecordview-class.md)-Klasse. `COleDBRecordView` zeigt Datenbankdaten Sätze in Steuerelementen über eine Formularansicht an, die direkt mit einem [CRowset](../data/oledb/crowset-class.md) -Objekt verbunden ist. Weitere Informationen zu den OLE DB Consumervorlagen finden Sie unter [Liste der OLE DB Consumer-Vorlagen](../data/oledb/ole-db-consumer-templates-reference.md).
