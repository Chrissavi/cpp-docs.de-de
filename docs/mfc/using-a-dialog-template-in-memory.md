# <a name="using-a-dialog-template-in-memory"></a>Verwenden einer Dialogvorlage im Speicher

Anstatt die Methoden zu verwenden, die in der Tabelle für die [Dialog Erstellung](../mfc/creating-a-dialog-class-with-code-wizards.md)angegeben sind, können Sie beide Arten von Dialogfeldern indirekt aus einer Dialogfeld Vorlage im Arbeitsspeicher erstellen. Weitere Informationen finden Sie unter Class [CDialog](../mfc/reference/cdialog-class.md) in der *MFC-Referenz*.
