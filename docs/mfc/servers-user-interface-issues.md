# <a name="servers-user-interface-issues"></a>Server: Probleme mit der Benutzeroberfläche

Eine Server-Anwendung verfügt über eine Reihe von Funktionen, die die Benutzeroberfläche, um OLE-Elemente, um containeranwendungen bereitzustellen hinzugefügt werden muss. Weitere Informationen in den Menüs und die zusätzlichen Ressourcen, die an eine Serveranwendung hinzugefügt werden müssen, finden Sie unter [Menüs und Ressourcen: Servererweiterungen](../mfc/menus-and-resources-server-additions.md).
