# <a name="activex-controls"></a>ActiveX-Steuerelemente

In Visual C++ Sie mithilfe von MFC oder ATL ActiveX-Steuerelemente erstellen.

>[!IMPORTANT]
> ActiveX ist eine ältere Technologie, die nicht für die neue Entwicklung verwendet werden sollte. Viele Funktionen von ActiveX-Steuerelementen können auf einfachere und sicherere Weise mit modernen Technologien wie HTML5 und JavaScript, modernen Browser Erweiterungen oder webassemblymodulen ausgeführt werden. Weitere Informationen finden [Sie in einer Unterbrechung der Vergangenheit, Teil 2: sagen von "Goodbye to ActiveX, VBScript, AttachEvent...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) und [native Messaging](/microsoft-edge/extensions/guides/native-messaging) -und [Microsoft Edge-Erweiterungen](/microsoft-edge/extensions) und [Webassembly](https://webassembly.org/).
