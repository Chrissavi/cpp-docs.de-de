# <a name="activex-controls"></a>ActiveX-Steuerelemente

In Visual C++ können Sie ActiveX-Steuerelementen mit MFC oder ATL erstellen.

>[!IMPORTANT]
> ActiveX ist eine veraltete Technologie, die nicht für Neuentwicklungen verwendet werden soll. Viele Funktionen von ActiveX-Steuerelemente können auf eine Weise einfacher und sicherer mit modernen Technologien wie HTML5 und JavaScript, modernen Browsererweiterungen oder WebAssembly Modulen ausgeführt werden. Weitere Informationen finden Sie unter [eine Unterbrechung aus der Vergangenheit, Teil 2: Saying Goodbye, ActiveX, VBScript, "AttachEvent"...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) und [einheitlichen Messaging](/microsoft-edge/extensions/guides/native-messaging) und [Microsoft Edge-Erweiterungen](/microsoft-edge/extensions) und [WebAssembly](https://webassembly.org/).
