# <a name="mfc-technical-notes"></a>Technische Hinweise zu MFC

Ein technischer Hinweis ist ein Dokument, das von Programmierern für Programmierer geschrieben wurde.

Jeder technische Hinweis beschreibt ein Problem oder eine Funktion, die den Rahmen der restlichen MFC-Dokumentation sprengen. Die bereitgestellten technischen Hinweise entsprechen Anforderungen für Informationen von Benutzern sowie spezialisierte Informationen, die die MFC-Entwickler für erweiterte Benutzer erwarten.

Es gibt zwei Möglichkeiten, die technischen Hinweise zu durchsuchen:

- [Technische Hinweise nach Zahl](technical-notes-by-number.md)

- [Technische Hinweise nach Kategorie](technical-notes-by-category.md)
