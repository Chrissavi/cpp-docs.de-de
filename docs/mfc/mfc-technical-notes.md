# <a name="mfc-technical-notes"></a>Technische Hinweise zu MFC

Ein Dokument, das für Programmierer, die von Programmierern geschrieben ist, ein technischer Hinweis.

Jede technischer Hinweis beschreibt ein Problem oder eine Funktion, die den Rest der Dokumentation zu MFC nicht eingegangen ist. Die technischen Anmerkungen zu dieser Version bereitgestellten wider, Anforderungen für Informationen zu Benutzern als auch spezielle Informationen, die die MFC-Entwickler erwarten fortgeschrittene Benutzer möchten.
