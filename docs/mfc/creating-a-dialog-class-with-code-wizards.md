# <a name="creating-a-dialog-class-with-code-wizards"></a>Erstellen einer Dialogfeldklasse mit Code-Assistenten

In der folgenden Tabelle werden die Aufgaben im Zusammenhang mit dem Dialogfeld aufgelistet, die von den Code-Assistenten

### <a name="dialog-related-tasks"></a>Dialog bezogene Aufgaben

|Aufgabe|Gilt für. . .|
|----------|--------------------|
|[Erstellen Sie eine neue CDialog-abgeleitete Klasse](creating-your-dialog-class.md) zum Verwalten des Dialog Felds.|Alle Dialogfelder.|
|Ordnen Sie der Dialogfeld Klasse [Windows-Meldungen](handling-windows-messages-in-your-dialog-box.md) zu.|Jede Nachricht, die Sie behandeln möchten.|
|Deklarieren Sie Klassenmember-Variablen, um die Steuerelemente im Dialogfeld darzustellen.|Jedes Steuerelement, das einen Text oder numerischen Wert ergibt, auf den Sie über das Programm zugreifen möchten.|
|Geben [Sie an, wie Daten](dialog-data-exchange-and-validation.md) zwischen den Steuerelementen und den Element Variablen ausgetauscht werden sollen.|Jedes Steuerelement, auf das Sie über das Programm zugreifen möchten.|
|Geben Sie [Validierungsregeln](dialog-data-exchange-and-validation.md) für die Element Variablen an.|Jedes Steuerelement, das bei Bedarf einen Text-oder einen numerischen Wert ergibt.|
