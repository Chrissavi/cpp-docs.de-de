# <a name="concurrencyprecise_math-namespace"></a>Concurrency::precise_math-Namespace

Funktionen im `precise_math`-Namespace sind C99-kompatibel. Es sind sowohl Versionen mit einfacher Genauigkeit als auch mit doppelter Genauigkeit der einzelnen Funktionen enthalten. Beispielsweise ist `acos` die Version mit doppelter Genauigkeit und `acosf` die Version mit einfacher Genauigkeit. Diese Funktionen, einschließlich der Funktionen mit einfacher Genauigkeit, erfordern eine erweiterte Unterstützung doppelter Genauigkeit auf der Zugriffstaste. Sie können den [Accelerator:: supports_double_precision](accelerator-class.md#supports_double_precision) verwenden, um zu bestimmen, ob Sie diese Funktionen auf einer bestimmten Zugriffstaste ausführen können.

## <a name="syntax"></a>Syntax

```cpp
namespace precise_math;
```
