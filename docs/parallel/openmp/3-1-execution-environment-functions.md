# <a name="31-execution-environment-functions"></a>3.1 Ausführungsumgebungsfunktionen

In diesem Abschnitt beschriebenen Funktionen auswirken, und Überwachen von Threads, Prozessoren und der parallelen Umgebung:

- die **Omp_set_num_threads** Funktion.

- die **Omp_get_num_threads** Funktion.

- die **Omp_get_max_threads** Funktion.

- die **Omp_get_thread_num** Funktion.

- die **Omp_get_num_procs** Funktion.

- die **Omp_in_parallel** Funktion.

- die **Omp_set_dynamic** Funktion.

- die **Omp_get_dynamic** Funktion.

- die **Omp_set_nested** Funktion.

- die **Omp_get_nested** Funktion.
