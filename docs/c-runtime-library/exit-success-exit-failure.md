Hierbei handelt es sich um Argumente für die [exit](reference/exit-exit-exit.md)- und [_exit](reference/exit-exit-exit.md)-Funktion und die Rückgabewerte für die [atexit](reference/atexit.md)- und [_onexit](reference/onexit-onexit-m.md)-Funktion.

|Konstante|Defined Values|
|-|-|
|EXIT_SUCCESS|0|
|EXIT_FAILURE|1|
