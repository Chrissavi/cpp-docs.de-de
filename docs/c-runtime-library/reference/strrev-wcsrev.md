# <a name="strrev-wcsrev"></a>strrev, wcsrev

Die Microsoft-spezifischen Funktionsnamen `strrev` und `wcsrev` sind als veraltet markierte Aliase für die Funktionen [_strrev und _wcsrev](strrev-wcsrev-mbsrev-mbsrev-l.md) . Standardmäßig generieren Sie eine [Compilerwarnung (Stufe 3) C4996](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md). Die Namen sind veraltet, da Sie nicht den Standard mäßigen C-Regeln für Implementierungs spezifische Namen folgen. Die Funktionen werden jedoch weiterhin unterstützt.

Es wird empfohlen, stattdessen [_strrev und _wcsrev zu](strrev-wcsrev-mbsrev-mbsrev-l.md) verwenden. Oder Sie können diese Funktionsnamen weiterhin verwenden und die Warnung deaktivieren. Weitere Informationen finden Sie unter [Deaktivieren der Namen der Warnungs](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#turn-off-the-warning) -und [POSIX-Funktionen](../../error-messages/compiler-warnings/compiler-warning-level-3-c4996.md#posix-function-names).
