# <a name="fabs-fabsf-fabsl"></a>fabs, fabsf, fabsl

Berechnet den absoluten Wert des Gleitkommaarguments.

## <a name="syntax"></a>Syntax

```C
double fabs(
   double x
);
float fabs(
   float x
); // C++ only
long double fabs(
   long double x
); // C++ only
float fabsf(
   float x
);
long double fabsl(
   long double x
);
```

### <a name="parameters"></a>Parameter

*x*<br/>
Gleitkommawert.

## <a name="return-value"></a>Rückgabewert

Die **Fabs** -Funktionen geben den absoluten Wert des Arguments *x*zurück. Es gibt keine Fehlerrückgabe.

|Eingabe|SEH-Ausnahme|Matherr-Ausnahme|
|-----------|-------------------|-----------------------|
|± QNAN,IND|Keine|_DOMAIN|

## <a name="remarks"></a>Bemerkungen

C++ ermöglicht überladen, sodass Sie über Ladungen von **Fabs** aufzurufen können, wenn Sie den- \<cmath> Header einschließen. In einem C-Programm übernimmt **Fabs** immer und gibt einen zurück **`double`** .

Standardmäßig ist der globale Status dieser Funktion auf die Anwendung beschränkt. Informationen hierzu finden Sie unter [globaler Status in der CRT](../global-state.md).
