# <a name="xor"></a>xor

Eine Alternative zum ^-Operator.

## <a name="syntax"></a>Syntax

```C

#define xor ^
```

## <a name="remarks"></a>Bemerkungen

Das Makro gibt den Operator ^ aus.

## <a name="example"></a>Beispiel

```cpp
// iso646_xor.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a ^ b;
   cout << result << endl;

   result= a xor_eq b;
   cout << result << endl;
}
```

```Output
1
1
```
