# <a name="freeentry-usedentry"></a>_FREEENTRY, _USEDENTRY

## <a name="syntax"></a>Syntax

```
#include <malloc.h>
```

## <a name="remarks"></a>Hinweise

Diese Konstanten stellen Werte dar, die von den `_heapwalk`-Routinen dem **_useflag**-Element der **_HEAPINFO**-Struktur zugewiesen werden. Sie geben den Status des Heapeintrags an.
