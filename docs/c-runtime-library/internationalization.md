# <a name="internationalization"></a>Internationalisierung

Die Microsoft-Lauf Zeit Bibliothek stellt viele Routinen bereit, mit denen Sie Ihre APP für internationale Märkte wie die folgenden anpassen können:
- [Gebiets Schema bezogene Routinen](../c-runtime-library/locale.md)
- breit Zeichenroutinen
- multibytezeichenroutinen
- generische Text Routinen

Die meisten Gebiets Schema bezogenen Routinen werden aus Gründen der Unterstützung auch kategorisiert.

Multibyte-Routinen und breit Zeichenroutinen werden zusammen mit ihren Einzel Byte Zeichen-Entsprechungen beschrieben, wenn Sie vorhanden sind.

Iso646 Operator Alternativen sind ebenfalls enthalten.
