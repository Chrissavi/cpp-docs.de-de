# <a name="intel39s-mmx-instruction-set"></a>Intel&#39;s-MMX-Anweisungs Satz

**Microsoft-spezifisch**

Mit dem C++ Microsoft-Compiler können Sie den MMX-Anweisungs Satz (Multimedia Extension) von Intel im Inline Assembler verwenden. Die MMX-Anweisungen werden auch von der debuggerdisassembly unterstützt. Der Compiler generiert eine Warnmeldung, wenn die Funktion MMX-Anweisungen enthält, aber keine EMMS-Anweisung enthält, um den multimediarestate zu leeren. Weitere Informationen finden Sie auf der Intel-Website.

**Ende Microsoft-spezifisch**
