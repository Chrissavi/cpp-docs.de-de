# <a name="perioderrdif-perioderrdifi"></a>&period;ERRDIF, &period;ERRDIFI

Generiert einen Fehler aus, wenn die Textelemente unterscheiden.

## <a name="syntax"></a>Syntax

> **.ERRDIF**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Hinweise

Wenn  **&period;ERRDIFI** wird angegeben, beim Vergleich wird die Groß-/Kleinschreibung.
