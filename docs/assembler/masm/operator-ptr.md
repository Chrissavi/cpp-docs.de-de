# <a name="operator-ptr"></a>Operator PTR

Der erste Operator erzwingt, dass die *Ausdruck* behandelt werden soll, mit dem angegebenen *Typ*. Der zweite Operator gibt einen Zeiger auf *Typ*.

## <a name="syntax"></a>Syntax

> *Typ* PTR *Ausdruck*

> [[*Abstand*]] PTR *Typ*
