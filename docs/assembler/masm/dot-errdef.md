# <a name="errdef"></a>.ERRDEF

Wird ein Fehler generiert, wenn *Namen* ist eine zuvor definierte Bezeichnung, Variable oder ein Symbol.

## <a name="syntax"></a>Syntax

> . ERRDEF Name [[, Nachricht]]
