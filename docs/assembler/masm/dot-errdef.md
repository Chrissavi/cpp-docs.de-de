# <a name="errdef"></a>.ERRDEF

Generiert einen Fehler, wenn *Name* eine zuvor definierte Bezeichnung, Variable oder ein Symbol ist.

## <a name="syntax"></a>Syntax

> **. Errdef** - *Name* ⟦ __,__ *Message*⟧
