# <a name="ml-nonfatal-error-a2047"></a>Nicht schwerwiegender ML-Fehler A2047

**leere Zeichenfolge (null)**

Eine Zeichenfolge besteht aus einem Trennzeichen Paar von Anführungszeichen und ohne Zeichen in.

Damit eine Zeichenfolge gültig ist, muss Sie 1-255 Zeichen enthalten.
