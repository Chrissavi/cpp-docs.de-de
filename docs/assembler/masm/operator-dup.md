# <a name="operator-dup"></a>Operator DUP

Gibt *die Anzahl von* Deklarationen von *InitialValue*an.

## <a name="syntax"></a>Syntax

> *count* **DUP** __(__ *InitialValue* ⟦ __,__ *InitialValue* ... ⟧ __)__
