# <a name="exit-32-bit-masm"></a>. Exit (32-Bit-MASM)

Generiert einen Beendigungs Code. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> . Exit [[Ausdruck]]

## <a name="remarks"></a>Hinweise

Gibt den optionalen *Ausdruck* an die Shell zurück.
