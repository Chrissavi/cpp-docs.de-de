# <a name="operator-ge"></a>Operator GE

Gibt "true" (-1) zurück, wenn *expression1* ist größer als oder gleich *expression2*, oder False (0) zurück, wenn er nicht ist.

## <a name="syntax"></a>Syntax

> expression1 GE expression2
