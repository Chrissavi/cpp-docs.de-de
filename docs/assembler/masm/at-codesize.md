# <a name="codesize"></a>@CodeSize

0 für **WINZIG**, **kleine**, **COMPACT**, und **Flatfile** Modelle und 1 für **Mittel**,  **GROßE**, und **sehr groß** Modelle (numerisch gleichgesetzt).

## <a name="syntax"></a>Syntax

> @CodeSize
