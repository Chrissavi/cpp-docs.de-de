# <a name="ends"></a>ENDS

Markiert das Ende der Segment-, Struktur-oder Union- *namens* , das zuvor mit [Segment](segment.md)- [, Struktur-,](struct-masm.md) [Union](union.md)-oder vereinfachten Segment Direktiven begonnen wurde.

## <a name="syntax"></a>Syntax

> *Name* **endet**
