# <a name="operator-sectionrel"></a>Operator SECTIONREL

Der Abschnitt relativen Offset des Ausdrucks relativ zum Abschnitt mit den das Ziel in die endgültige ausführbare zurückgegeben.

## <a name="syntax"></a>Syntax

> SECTIONREL-Ausdruck

## <a name="remarks"></a>Hinweise

SECTIONREL ist nur bei der Ausgabe von COFF-Objekt verfügbar.
