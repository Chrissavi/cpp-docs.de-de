# <a name="ml-warning-a4004"></a>ML-Warnung A4004

**kann nicht von CS ausgehen**

Es wurde versucht, einen Wert für das CS-Register zu übernehmen. CS wird immer auf das aktuelle Segment oder die aktuelle Gruppe festgelegt.
