# <a name="data"></a>@data

Der Name der standarddatengruppe. Ergibt für alle Modelle, mit Ausnahme von DGROUP **Flatfile**. Ergibt **Flatfile** unter der **Flatfile** Speichermodell (Textmakro).

## <a name="syntax"></a>Syntax

> @data
