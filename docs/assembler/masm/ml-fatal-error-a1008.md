# <a name="ml-fatal-error-a1008"></a>Schwerwiegender ML-Fehler A1008

**unvollständige makroschachtelung**

Entweder wurde ein Makro nicht vor dem Ende der Datei beendet, oder der abschließende [direktivenendm](../../assembler/masm/endm.md) wurde außerhalb eines Makroblocks gefunden.

Eine Ursache für diesen Fehler ist das Weglassen des Punktes vor [. Wiederholen](../../assembler/masm/dot-repeat.md) Sie oder [. Während](../../assembler/masm/dot-while.md).
