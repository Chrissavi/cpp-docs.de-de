# <a name="ml-fatal-error-a1008"></a>Schwerwiegender ML-Fehler A1008

**unvollständige makroschachtelung**

Entweder wurde ein Makro nicht vor dem Ende der Datei beendet, oder der abschließende [direktivenendm](endm.md) wurde außerhalb eines Makroblocks gefunden.

Eine Ursache für diesen Fehler ist das Weglassen des Punktes vor [. Wiederholen](dot-repeat.md) Sie oder [. Während](dot-while.md).
