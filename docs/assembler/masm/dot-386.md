# <a name="386"></a>.386

Können die Assembly nicht privilegierten Anweisungen für den Prozessor 80386; deaktiviert die Assembly von Anweisungen, die mit höheren Prozessoren eingeführt wurde.

## <a name="syntax"></a>Syntax

> .386

## <a name="remarks"></a>Hinweise

Außerdem können 80387 Anweisungen.
