# <a name="386-32-bit-masm"></a>.386 (32-Bit-MASM)

Aktiviert die Assembly nicht privilegierter Anweisungen für den 80386-Prozessor. deaktiviert die Assembly der Anweisungen, die mit späteren Prozessoren eingeführt wurden. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> **.386**

## <a name="remarks"></a>Hinweise

Außerdem werden 80387-Anweisungen aktiviert.
