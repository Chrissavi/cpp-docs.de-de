# <a name="ml-warning-a4014"></a>ML-Warnung A4014

Anweisungen und initialisierte Daten in Segmenten von BSS nicht unterstützt.

Es wurde versucht, initialisierte Daten in einem BSS-Abschnitt definiert.  Ein BSS-Abschnitt ist als eine Klasse definiert, deren Name BSS ist.  Dazu gehören die vereinfachte Segment `.data?`.
