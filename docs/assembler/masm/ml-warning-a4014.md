# <a name="ml-warning-a4014"></a>ML-Warnung A4014

Anweisungen und initialisierte Daten werden in BSS-Segmenten nicht unterstützt.

Es wurde versucht, initialisierte Daten innerhalb eines BSS-Abschnitts zu definieren.  Ein BSS-Abschnitt ist als eine Klasse definiert, deren Name BSS ist.  Dies schließt das vereinfachte Segment `.data?`ein.
