# <a name="486p-32-bit-masm"></a>.486p (32-Bit-MASM)

Aktiviert die Assembly aller Anweisungen (einschließlich privilegierter Berechtigungen) für den 80486-Prozessor. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> .486P
