# <a name="datasize"></a>@DataSize

0 für **WINZIG**, **kleine**, **Mittel**, und **Flatfile** modelliert, 1 für **COMPACT** und  **GROßE** Modelle und 2 für **sehr groß** Modell (numerisch gleichgesetzt).

## <a name="syntax"></a>Syntax

> @DataSize
