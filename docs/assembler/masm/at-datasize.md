# <a name="datasize"></a>\@DataSize

0 für **kleine**, **kleine**, **mittlere**und **flache** Modelle, 1 für **kompakte** und **große** Modelle und 2 für ein **riesiges** Modell (numeric equate).

## <a name="syntax"></a>Syntax

> **\@DataSize**
