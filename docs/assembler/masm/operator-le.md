# <a name="operator-le"></a>Operator LE

Gibt "true" (-1) zurück, wenn *expression1* ist kleiner als oder gleich *expression2*, oder False (0) zurück, wenn er nicht ist.

## <a name="syntax"></a>Syntax

> expression1 LE expression2
