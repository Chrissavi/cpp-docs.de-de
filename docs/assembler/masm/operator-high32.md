# <a name="operator-high32"></a>Operator HIGH32

Gibt die hohen 32 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **HIGH32** - *Ausdruck*
