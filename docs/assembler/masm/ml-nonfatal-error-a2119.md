# <a name="ml-nonfatal-error-a2119"></a>Nicht schwerwiegender ML-Fehler A2119

**der Sprachtyp muss angegeben werden.**

Einer Prozedur Definition oder einem Prototyp wurde kein Sprachtyp zugewiesen.

Wenn kein Standard Sprachentyp angegeben ist, muss in den einzelnen Prozeduren oder im Prototyp der Prozedur ein Sprachtyp deklariert werden. Ein Standard Sprachtyp wird mithilfe von festgelegt [. Model](../../assembler/masm/dot-model.md) -Direktive, **Option lang**oder die ML-Befehlszeilenoptionen **/GC** oder **/GD**.
