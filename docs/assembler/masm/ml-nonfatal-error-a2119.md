# <a name="ml-nonfatal-error-a2119"></a>Nicht schwerwiegender ML-Fehler A2119

**Language-Typ muss angegeben werden**

Eine Definition der Prozedur oder einen Prototyp wurde nicht mit einen Language-Typ angegeben.

Ein Language muss in jeder Definition der Prozedur oder einen Prototyp deklariert werden, wenn ein Standardtyp für die Sprache nicht angegeben ist. Ein Standardtyp für die Sprache festgelegt ist, verwenden entweder die [. Modell](../../assembler/masm/dot-model.md) -Direktive **OPTION LANG**, oder die ML-Befehlszeilenoptionen **/GC** oder **/GD**.
