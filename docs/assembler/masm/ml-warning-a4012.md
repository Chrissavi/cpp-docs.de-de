# <a name="ml-warning-a4012"></a>ML-Warnung A4012

**Zeilennummern Informationen für das Segment ohne die Klasse "Code"**

Es gab Anweisungen in einem Segment, das keinen Klassennamen enthielt, der auf "Code" endet. Der Assembler hat für diese Anweisungen keine CodeView-Informationen generiert.

CodeView kann keine Module mit Code in Segmenten mit Klassennamen verarbeiten, die nicht mit "Code" enden.
