# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

Gewährt-Assembly, wenn die Textelemente unterscheiden.

## <a name="syntax"></a>Syntax

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Hinweise

Wenn **IFDIFI** wird angegeben, beim Vergleich wird die Groß-/Kleinschreibung. Finden Sie unter [IF](../../assembler/masm/if-masm.md) für die vollständige Syntax.
