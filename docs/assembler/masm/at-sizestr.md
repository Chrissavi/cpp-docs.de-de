# <a name="sizestr"></a>@SizeStr

Ein Makrofunktion, die die Länge der angegebenen Zeichenfolge zurückgibt. Gibt eine ganze Zahl zurück.

## <a name="syntax"></a>Syntax

> @SizeStr(String)
