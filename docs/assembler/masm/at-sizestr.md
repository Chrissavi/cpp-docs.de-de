# <a name="sizestr"></a>\@sizestr

Eine Makrofunktion, die die Länge der angegebenen Zeichenfolge zurückgibt. Gibt eine ganze Zahl zurück.

## <a name="syntax"></a>Syntax

> **\@sizestr (** *Zeichenfolge* **)**
