# <a name="while-32-bit-masm"></a>. WHILE (32-Bit-MASM)

Generiert Code, der den Anweisungsblock ausführt *, während die* *Bedingung* "true" ist. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> **. WHILE** *-\*
> *Anweisungen*\
> **.ENDW**
