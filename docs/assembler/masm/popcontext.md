# <a name="popcontext"></a>POPCONTEXT

Stellt einen Teil oder den gesamten aktuellen *Kontext* wieder her (gespeichert durch die [PUSHCONTEXT](../../assembler/masm/pushcontext.md) -Direktive). Der *Kontext* kann angenommen **werden (** nur 32-Bit-massm), **Radix**, **Auflistung**, **CPU** (nur 32-Bit-massm) oder **alle**.

## <a name="syntax"></a>Syntax

> **POPCONTEXT** - *Kontext*
