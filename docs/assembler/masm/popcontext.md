# <a name="popcontext"></a>POPCONTEXT

Stellt Teils oder aller aktuellen `context` (gespeichert werden, indem die [PUSHCONTEXT](../../assembler/masm/pushcontext.md) Richtlinie). Die `context` kann **setzt**, `RADIX`, **auflisten**, **CPU**, oder **alle**.

## <a name="syntax"></a>Syntax

> POPCONTEXT-Kontext
