# <a name="elseif"></a>ELSEIF

Kombiniert [else](else-masm.md) und [if](if-masm.md) in einer-Anweisung.

## <a name="syntax"></a>Syntax

> **ElseIf** *ConstantExpression*\
> *Anweisungen*\
> ⟦**Else** -\
> *else-Anweisungen*⟧ \
> **EndIf**
