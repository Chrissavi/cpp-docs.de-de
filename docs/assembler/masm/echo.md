# <a name="echo"></a>ECHO
Zeigt *Nachricht* in das Standardausgabegerät (wird standardmäßig auf dem Bildschirm).  
  
## <a name="syntax"></a>Syntax  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Identisch mit [% OUT](../../assembler/masm/percent-out.md).
