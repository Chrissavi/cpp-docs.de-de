# <a name="echo"></a>ECHO

Zeigt eine *Meldung* an das Standardausgabe Gerät an (standardmäßig der Bildschirm).

## <a name="syntax"></a>Syntax

> **Echo** *Meldung*

## <a name="remarks"></a>Hinweise

Identisch mit [% out](../../assembler/masm/percent-out.md).
