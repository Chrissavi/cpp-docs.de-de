# <a name="instr"></a>INSTR

Sucht das erste Vorkommen des *textitem2* in *textitem1*.

## <a name="syntax"></a>Syntax

> *Namen* INSTR [[*Position*,]] *textitem1*, *textitem2*

## <a name="remarks"></a>Hinweise

Das Starten *Position* ist optional. Jedes Element mit Text kann ein Zeichenfolgenliteral, eine Konstante vorangestellt werden eine `%`, oder die Zeichenfolge, die von einem Makrofunktion zurückgegeben wird.
