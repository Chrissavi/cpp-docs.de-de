# <a name="real8"></a>REAL8

Zugewiesen und initialisiert Sie optional eine (8 Byte)-Gleitkommazahl mit doppelter Genauigkeit für die einzelnen *Initialisierer*.

## <a name="syntax"></a>Syntax

> REAL8-Initialisierer Name [[, Initialisierer]]...
