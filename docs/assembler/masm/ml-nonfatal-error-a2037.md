# <a name="ml-nonfatal-error-a2037"></a>Nicht schwerwiegender ML-Fehler A2037

**Anweisung ist in der Strukturdefinition nicht zulässig.**

Eine Strukturdefinition enthalten eine ungültige Anweisung.

Eine Struktur kann nicht enthalten, Anweisungen, Bezeichnungen, Prozeduren, ablaufsteuerung Anweisungen [. Beim Start](../../assembler/masm/dot-startup.md), oder [. EXIT](../../assembler/masm/dot-exit.md).
