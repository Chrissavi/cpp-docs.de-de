# <a name="ml-nonfatal-error-a2037"></a>Nicht schwerwiegender ML-Fehler A2037

**die Anweisung ist in der Struktur Definition nicht zulässig.**

Eine Struktur Definition enthielt eine ungültige Anweisung.

Eine Struktur kann keine Anweisungen, Bezeichnungen, Prozeduren, Ablauf Steuerungs Anweisungen, enthalten [. Starten](dot-startup.md)oder [. Beenden](dot-exit.md)Sie.
