# <a name="repeat"></a>WIEDERHOLUNG

Markiert einen Block, bei dem es sich um wiederholte *Ausdrucks* Zeiten handelt. Identisch mit [rept](rept.md).

## <a name="syntax"></a>Syntax

> *Ausdruck* **wiederholen**\
> *Anweisungen*\
> **ENDM**
