# <a name="sdword"></a>SDWORD

Zugewiesen und initialisiert Sie optional ein Vorzeichen Doppelwort (4 Bytes) des Speichers für die einzelnen `initializer`. Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig.

## <a name="syntax"></a>Syntax

> SDWORD-Initialisierer Name [[, Initialisierer]]...
