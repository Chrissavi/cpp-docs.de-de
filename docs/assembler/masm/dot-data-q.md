# <a name="data"></a>.DATA?

(nur 32-Bit-MASM.) Bei Verwendung mit [. Model](dot-model.md), startet ein near-Daten Segment für nicht initialisierte Daten (Segment Name _BSS).

## <a name="syntax"></a>Syntax

> **.DATA?** \
> ⟦ *segmentitem* ⟧...
