# <a name="public-masm"></a>PUBLIC (MASM)

Macht jede Variable, die Bezeichnung oder die absolutes Symbol als *Namen* für alle anderen Module in der Anwendung verfügbar.

## <a name="syntax"></a>Syntax

> Name der öffentlichen [[Langtype]] [[, [[Langtype]]-Name]]...
