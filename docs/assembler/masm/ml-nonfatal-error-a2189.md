# <a name="ml-nonfatal-error-a2189"></a>Nicht schwerwiegender ML-Fehler A2189

**Ungültige Kombination mit Segment Ausrichtung**

Die Ausrichtung, die von der **align** -oder `EVEN`-Direktive angegeben wird, war größer als die aktuelle Segment Ausrichtung gemäß der **Segment** -Direktive.
