# <a name="ml-nonfatal-error-a2189"></a>Nicht schwerwiegender ML-Fehler A2189

**Ungültige Kombination mit der Segment-Ausrichtung**

Die Ausrichtung anhand der **AUSRICHTEN** oder `EVEN` Direktive war größer als das aktuelle Segment Ausrichtung gemäß der **SEGMENT** Richtlinie.
