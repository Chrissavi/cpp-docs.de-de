# <a name="ml-nonfatal-error-a2060"></a>Nicht schwerwiegender ML-Fehler A2060

**erzwungener Fehler: Zeichen folgen sind nicht gleich.**

Die bedingte error-Direktive [. ERRDIF](dot-errdif.md) oder **. ERRDIFI** wurde verwendet, um diesen Fehler zu generieren.
