# <a name="ml-nonfatal-error-a2060"></a>Nicht schwerwiegender ML-Fehler A2060

**Fehler beim erzwungenen: Zeichenfolgen nicht gleich**

Das Conditional-Error-Anweisung [. ERRDIF](../../assembler/masm/dot-errdif.md) oder **. ERRDIFI** wurde verwendet, um diesen Fehler zu generieren.
