# <a name="ml-nonfatal-error-a2206"></a>Nicht schwerwiegender ML-Fehler A2206

**fehlender Operator in Ausdruck**

Ein Ausdruck kann nicht ausgewertet werden, da ein Operator fehlt. Diese Fehlermeldung kann auch ein Nebeneffekt eines vorangehenden Programmfehlers sein.

Dieser Fehler wird in der folgenden Zeile generiert:

```asm
value1 = ( 1 + 2 ) 3
```
