# <a name="operator-low"></a>Operator LOW

Gibt die unteren 8 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **Niedriger** *Ausdruck*
