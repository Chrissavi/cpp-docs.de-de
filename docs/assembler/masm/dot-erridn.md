# <a name="perioderridn-perioderridni"></a>&period;ERRIDN, &period;ERRIDNI

Generiert einen Fehler aus, wenn die Textelemente identisch sind.

## <a name="syntax"></a>Syntax

> **.ERRIDN**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Hinweise

Wenn  **&period;ERRIDNI** wird angegeben, beim Vergleich wird die Groß-/Kleinschreibung.
