# <a name="ml-nonfatal-error-a2069"></a>Nicht schwerwiegender ML-Fehler A2069

**für diese Anweisung sind keine Operanden zulässig.**

Mindestens ein Operanden wurde mit einer Anweisung angegeben, die keine Operanden annimmt.
