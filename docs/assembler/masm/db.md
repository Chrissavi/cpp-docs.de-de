# <a name="db"></a>DB

Ordnet ein Byte des Speichers für jeden *Initialisierer*zu und initialisiert es optional. **DB** ist ein Synonym von [Byte](byte-masm.md).

## <a name="syntax"></a>Syntax

> ⟦*Name*⟧ **DB** *Initialisierer* ⟦ __,__ *Initialisierer* ... ⟧

## <a name="remarks"></a>Hinweise

Kann auch als Typspezifizierer überall dort verwendet werden, wo ein Typ zulässig ist.
