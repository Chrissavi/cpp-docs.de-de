# <a name="dw"></a>DW

Zugewiesen und initialisiert Sie optional ein Wort (2 Byte) des Speichers für die einzelnen `initializer`. Kann auch als Typspezifizierer verwendet werden ein Typ ist überall zulässig. `DW` ist ein Synonym für [WORD](../../assembler/masm/word.md).

## <a name="syntax"></a>Syntax

> [[Name]] Data Warehouse-Initialisierer [[, Initialisierer]]...
