# <a name="dw"></a>DW

Ordnet ein Wort (2 Bytes) des Speichers für jeden *Initialisierer*zu und initialisiert es optional. Kann auch als Typspezifizierer überall dort verwendet werden, wo ein Typ zulässig ist. **DW** ist ein Synonym von [Word](word.md).

## <a name="syntax"></a>Syntax

> ⟦*Name*⟧ **DW** - *Initialisierer* ⟦ __,__ *Initialisierer* ... ⟧
