# <a name="operator-lowword"></a>Operator LOWWORD

Gibt die unteren 16 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **Lowword** - *Ausdruck*
