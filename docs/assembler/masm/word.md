# <a name="word"></a>WORD

Zugewiesen und initialisiert Sie optional ein Wort (2 Byte) des Speichers für die einzelnen `initializer`. Kann auch als Typspezifizierer verwendet werden ein Typ ist überall zulässig.

## <a name="syntax"></a>Syntax

> [[*Namen*]] WORD *Initialisierer* [[, *Initialisierer*]]...
