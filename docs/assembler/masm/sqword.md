# <a name="sqword"></a>SQWORD

Zugewiesen und initialisiert Sie optional 8 Bytes mit Vorzeichen des Speichers für die einzelnen `initializer`. Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig.

## <a name="syntax"></a>Syntax

> SQWORD-Initialisierer Name [[, Initialisierer]]...
