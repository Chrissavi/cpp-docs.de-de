# <a name="struct"></a>STRUCT

Deklariert einen Strukturtyp mit den angegebenen *Feld Deklarationen*. Jedes Feld muss eine gültige Datendefinition sein. Identisch mit [Struc](struc.md).

## <a name="syntax"></a>Syntax

> *namens* **Struktur** ⟦*Ausrichtung*⟧ ⟦ __,__ **nicht eindeutige**⟧- *Feld Deklarationen*\
> *Name* **endet**

## <a name="remarks"></a>Hinweise

Das *Name* -Argument muss in der öffnenden und abschließenden Anweisung identisch sein.
