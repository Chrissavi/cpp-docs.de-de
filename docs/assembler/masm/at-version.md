# <a name="version"></a>\@Version

Die Haupt-und neben Version von MASM, die in der Befehlszeile als einzelne Zahl (Text Makro) gemeldet wird. Die MASM-Version 14.23.28107.0 erzeugt z. b. "1423".

## <a name="syntax"></a>Syntax

> **\@Version**
