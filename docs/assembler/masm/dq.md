# <a name="dq"></a>DQ

Zugewiesen und initialisiert Sie optional 8 Byte Speicherplatz für die einzelnen `initializer`. Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig. `DQ` ist ein Synonym für [QWORD](../../assembler/masm/qword.md).

## <a name="syntax"></a>Syntax

> [[Name]] DQ-Initialisierer [[, Initialisierer]]...
