# <a name="operator-highword"></a>Operator HIGHWORD
Gibt die oberen 16 Bits der die unteren 32 Bits des *Ausdruck*. MASM-Ausdrücke sind 64-Bit-Werte.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
HIGHWORD expression  
```
