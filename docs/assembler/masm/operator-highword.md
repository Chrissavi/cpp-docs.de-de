# <a name="operator-highword"></a>Operator HIGHWORD

Gibt die hohen 16 Bits der unteren 32 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **HighWord** - *Ausdruck*
