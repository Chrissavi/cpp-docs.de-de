# <a name="listall"></a>.LISTALL

Startet die Auflistung aller-Anweisungen.

## <a name="syntax"></a>Syntax

> **.LISTALL**

## <a name="remarks"></a>Hinweise

Entspricht der Kombination von [. Liste](dot-list.md) [. Listif](dot-listif.md), und [. Listmacroall](dot-listmacroall.md).
