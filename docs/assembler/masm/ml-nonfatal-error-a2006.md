# <a name="ml-nonfatal-error-a2006"></a>Nicht schwerwiegender ML-Fehler A2006

**nicht definiertes Symbol: Bezeichner**

Es wurde versucht, ein Symbol zu verwenden, das nicht definiert wurde.

Möglicherweise ist einer der folgenden Fehler aufgetreten:

- Es wurde kein Symbol definiert.

- Ein Feld war kein Member der angegebenen-Struktur.

- Ein Symbol wurde in einer Includedatei definiert, die nicht eingeschlossen wurde.

- Ein externes Symbol wurde ohne eine [extern](../../assembler/masm/extern-masm.md) -oder [EXTERNDEF](../../assembler/masm/externdef.md) -Direktive verwendet.

- Ein Symbol Name wurde falsch geschrieben.

- Auf eine lokale Code Bezeichnung wurde außerhalb des Gültigkeits Bereichs verwiesen.
