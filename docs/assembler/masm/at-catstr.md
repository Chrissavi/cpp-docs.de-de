# <a name="catstr"></a>\@.

Makrofunktion, die eine oder mehrere Zeichen folgen verkettet. Gibt eine Zeichenfolge zurück.

## <a name="syntax"></a>Syntax

> **\@"(** *Zeichenfolge1* ) (⟦ __,__ *Zeichenfolge2*... ⟧ **)**
