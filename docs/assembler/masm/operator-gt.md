# <a name="operator-gt"></a>Operator GT

Gibt "true" (-1) zurück, wenn *expression1* ist größer als *expression2*, oder False (0) zurück, wenn er nicht ist.

## <a name="syntax"></a>Syntax

> expression1 GT expression2
