# <a name="operator-gt"></a>Operator GT

Gibt true (-1) zurück, wenn *expression1* größer als *expression2*ist, oder gibt false (0) zurück, wenn dies nicht der Fall ist.

## <a name="syntax"></a>Syntax

> *expression1* **gt** *expression2*
