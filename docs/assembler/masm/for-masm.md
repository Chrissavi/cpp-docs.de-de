# <a name="for-masm"></a>FOR (MASM)

Markiert einen Block, der für jedes Mal wiederholt wird *Argument*, mit dem aktuellen *Argument* ersetzen *Parameter* bei jeder Wiederholung.

## <a name="syntax"></a>Syntax

> FÜR *Parameter* [[: REQ |: =*Standard*]], \< *Argument* [[, *Argument*]]... ><br/>
> *Anweisungen*<br/>
> ENDM

## <a name="remarks"></a>Hinweise

Identisch mit [IRP](../../assembler/masm/irp.md).
