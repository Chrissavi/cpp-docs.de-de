# <a name="operator-low32"></a>Operator LOW32

Gibt die unteren 32 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **LOW32** - *Ausdruck*
