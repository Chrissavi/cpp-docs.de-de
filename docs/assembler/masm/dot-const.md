# <a name="const-32-bit-masm"></a>. Konstant (32-Bit-MASM)

Bei Verwendung mit [. Model](dot-model.md), startet ein konstantes Daten Segment (mit Segment **Name Konstanten**).

## <a name="syntax"></a>Syntax

> **. Konstanten** ⟦ *segmentitem* ⟧...

## <a name="remarks"></a>Hinweise

Dieses Segment verfügt über das schreibgeschützte Attribut.
