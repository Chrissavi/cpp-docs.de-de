# <a name="ml-nonfatal-error-a2074"></a>Nicht schwerwiegender ML-Fehler A2074

**auf Bezeichnung kann nicht über Segment Register zugegriffen werden.**

Es wurde versucht, auf eine Bezeichnung über ein Segment Register zuzugreifen, das nicht für das Segment oder die Gruppe angenommen wurde.
