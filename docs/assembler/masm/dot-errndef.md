# <a name="errndef"></a>.ERRNDEF

Generiert einen Fehler, wenn der *Name* nicht definiert wurde.

## <a name="syntax"></a>Syntax

> **. ERRNDEF** - *Name* ⟦ __,__ *Message*⟧
