# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Assembly gewährt, wenn die Textelemente identisch sind.

## <a name="syntax"></a>Syntax

> **IFIDN**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Hinweise

Wenn **IFIDNI** wird angegeben, beim Vergleich wird die Groß-/Kleinschreibung. Finden Sie unter [IF](../../assembler/masm/if-masm.md) für die vollständige Syntax.
