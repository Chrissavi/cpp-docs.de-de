# <a name="operator-high"></a>Operator HIGH

Gibt die hohen 8 Bits der unteren 16 Bits des *Ausdrucks*zurück. MASM-Ausdrücke sind 64-Bit-Werte.

## <a name="syntax"></a>Syntax

> **Hoher** *Ausdruck*
