# <a name="fardata"></a>.FARDATA?

Bei Verwendung mit [. Modell](../../assembler/masm/dot-model.md), startet ein Datensegment ganz für nicht initialisierte Daten (Segmentnamen FAR_BSS oder *Namen*).

## <a name="syntax"></a>Syntax

> .FARDATA? [[Name]]
