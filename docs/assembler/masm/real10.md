# <a name="real10"></a>REAL10

Zugewiesen und initialisiert Sie optional eine 10-Byte-Gleitkommazahl für jede *Initialisierer*.

## <a name="syntax"></a>Syntax

> REAL10-Initialisierer Name [[, Initialisierer]]...
