# <a name="586p-32-bit-masm"></a>.586p (32-Bit-MASM)

Aktiviert die Assembly aller Anweisungen (einschließlich privilegierter Berechtigungen) für den Pentium-Prozessor. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> **.586P**
