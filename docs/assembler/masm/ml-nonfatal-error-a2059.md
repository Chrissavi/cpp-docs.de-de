# <a name="ml-nonfatal-error-a2059"></a>Nicht schwerwiegender ML-Fehler A2059

**erzwungener Fehler: Zeichen folgen gleich**

Die bedingte error-Direktive [. ERRIDN](dot-erridn.md) oder **. ERRIDNI** wurde verwendet, um diesen Fehler zu generieren.
