# <a name="end"></a>ENDE

Markiert das Ende eines Moduls und legt optional den Programm Einstiegspunkt auf *PROCID*fest.

## <a name="syntax"></a>Syntax

> **End** ⟦*PROCID*⟧

## <a name="remarks"></a>Hinweise

Das *PROCID* -Argument ist nur in 32-Bit-MASM gültig.
