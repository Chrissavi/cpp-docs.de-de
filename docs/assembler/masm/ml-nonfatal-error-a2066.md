# <a name="ml-nonfatal-error-a2066"></a>Nicht schwerwiegender ML-Fehler A2066

**nicht kompatibler CPU-Modus und Segmentgröße**

Es wurde versucht, ein Segment mit einem **USE16**-, **USE32**-oder **Flat** -Attribut zu öffnen, das nicht mit der angegebenen CPU kompatibel war, oder um eine 16-Bit-CPU in einem 32-Bit-Segment zu ändern.

Den **USE32** -und **Flat** -Attributen muss die Prozessor Anweisung. 386 oder höher vorangestellt sein.
