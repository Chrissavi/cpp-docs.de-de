# <a name="byte-masm"></a>BYTE (MASM)
Zum Belegen und optional ein Byte des Speichers für die einzelnen initialisiert `initializer`.  
  
## <a name="syntax"></a>Syntax  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Hinweise  
 Kann auch verwendet werden, als Typspezifizierer an einer beliebigen Stelle ist ein Typ zulässig.
