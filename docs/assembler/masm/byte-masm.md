# <a name="byte-masm"></a>BYTE (MASM)

Zugewiesen und initialisiert Sie optional ein Byte des Speichers für die einzelnen `initializer`.

## <a name="syntax"></a>Syntax

> [[Name]] BYTEinitializer [[, Initialisierer]]...

## <a name="remarks"></a>Hinweise

Kann auch als Typspezifizierer verwendet werden ein Typ ist überall zulässig.
