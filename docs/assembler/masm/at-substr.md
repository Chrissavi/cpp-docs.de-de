# <a name="substr"></a>\@substr

Eine Makrofunktion, die eine Teil Zeichenfolge zurückgibt, beginnend an der *Position*.

## <a name="syntax"></a>Syntax

> **\@substr (** *String* __,__ *Position* ⟦ __,__ *length*⟧ **)**
