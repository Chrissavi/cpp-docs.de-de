# <a name="ml-nonfatal-error-a2022"></a>Nicht schwerwiegender ML-Fehler A2022

**Anweisungs Operanden müssen dieselbe Größe aufweisen.**

Die Operanden einer Anweisung haben nicht die gleiche Größe.
