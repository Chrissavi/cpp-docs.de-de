# <a name="endprolog"></a>.ENDPROLOG

Signalisiert das Ende der Prolog-Deklarationen.

## <a name="syntax"></a>Syntax

> **.ENDPROLOG**

## <a name="remarks"></a>Hinweise

Es ist ein Fehler, eine der Prolog-Deklarationen außerhalb des Bereichs zwischen dem [proc](proc.md) - **Frame** und zu verwenden **. ENDPROLOG**.

Weitere Informationen finden Sie unter [MASM für x64 (ml64. exe)](masm-for-x64-ml64-exe.md).
