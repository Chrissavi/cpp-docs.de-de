# <a name="operator-lt"></a>Operator LT

Gibt "true" (-1) zurück, wenn *expression1* ist kleiner als *expression2*, oder False (0) zurück, wenn er nicht ist.

## <a name="syntax"></a>Syntax

> expression1 LT expression2
