# <a name="operator-lt"></a>Operator LT

Gibt true (-1) zurück, wenn *expression1* kleiner als *expression2*ist, oder gibt false (0) zurück, wenn dies nicht der Fall ist.

## <a name="syntax"></a>Syntax

> *expression1* **lt** *expression2*
