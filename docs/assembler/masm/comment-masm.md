# <a name="comment-masm"></a>COMMENT (MASM)
Behandelt alle *Text* zwischen oder auf derselben Zeile wie die Trennzeichen als Kommentar.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
