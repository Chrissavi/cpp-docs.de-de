# <a name="listif"></a>.LISTIF
Startet eine Liste der Anweisungen in "false" bedingte Blöcke.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
.LISTIF  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Identisch mit [. LFCOND](../../assembler/masm/dot-lfcond.md).
