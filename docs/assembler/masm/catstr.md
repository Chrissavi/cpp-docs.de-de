# <a name="catstr"></a>CATSTR

Verkettet die Textelemente.

## <a name="syntax"></a>Syntax

> Benennen Sie CATSTR [[textitem1 [[, textitem2]]...]]

## <a name="remarks"></a>Hinweise

Jedes Element mit Text kann ein Zeichenfolgenliteral, eine Konstante vorangestellt werden eine `%`, oder die Zeichenfolge, die von einem Makrofunktion zurückgegeben wird.
