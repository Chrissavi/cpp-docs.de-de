# <a name="ml-nonfatal-error-a2010"></a>Nicht schwerwiegender ML-Fehler A2010

**Ungültiger Typausdruck.**

Der Operand für [dieses](../../assembler/masm/operator-this.md) oder [ptr](../../assembler/masm/operator-ptr.md) war kein gültiger Typausdruck.
