# <a name="stack-32-bit-masm"></a>. Stack (32-Bit-MASM)

Bei Verwendung mit [. Modell](dot-model.md)definiert ein Stapel Segment (mit Segment Namen **Stapel**). Die optionale *Größe* gibt die Anzahl der Bytes für den Stapel an (standardmäßig 1.024). Die **. Die Stack-Direktive** schließt die Stapel Anweisung automatisch. (nur 32-Bit-MASM.)

## <a name="syntax"></a>Syntax

> **. Stapel** ⟦*Größe*⟧
