# <a name="operator-ne"></a>Operator NE
Gibt "true" (-1) zurück, wenn *expression1* stimmt nicht mit *expression2*, oder gibt "false" (0) zurück, wenn dies der Fall ist.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
expression1 NE expression2  
```
