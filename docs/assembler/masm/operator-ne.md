# <a name="operator-ne"></a>Operator NE

Gibt true (-1) zurück, wenn *expression1* nicht gleich *expression2*ist, oder gibt false (0) zurück, wenn dies der Fall ist.

## <a name="syntax"></a>Syntax

> *expression1* **ne** *expression2*
