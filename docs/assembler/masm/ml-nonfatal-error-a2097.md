# <a name="ml-nonfatal-error-a2097"></a>Nicht schwerwiegender ML-Fehler A2097

**Segment erwartet: Bezeichner**

Der [Group](group.md) -Direktive wurde ein Bezeichner zugewiesen, der kein definiertes Segment war.
