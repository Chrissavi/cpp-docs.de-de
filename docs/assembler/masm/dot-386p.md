# <a name="386p"></a>.386P

Können die Assembly des alle-Anweisungen (einschließlich der Berechtigungen) für den Prozessor 80386; deaktiviert die Assembly von Anweisungen, die mit höheren Prozessoren eingeführt wurde.

## <a name="syntax"></a>Syntax

> .386P

## <a name="remarks"></a>Hinweise

Außerdem können 80387 Anweisungen.
