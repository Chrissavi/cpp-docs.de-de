# <a name="group"></a>GRUPPE

(nur 32-Bit-MASM.) Fügen Sie die angegebenen *Segmente* der Gruppe namens *Name*hinzu.

## <a name="syntax"></a>Syntax

> *namens* **Gruppen** *Segment* ⟦ __,__ *Segment* ... ⟧
