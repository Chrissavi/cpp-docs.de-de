# <a name="oword"></a>OWORD

Wird als Typspezifizierer verwendet, wenn ein 16-Byte-Datentyp erforderlich ist.

## <a name="syntax"></a>Syntax

> ⟦*Name*⟧ **oWord**-*Initialisierer* ⟦ __,__ *Initialisierer* ... ⟧
