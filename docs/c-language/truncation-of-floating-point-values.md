# <a name="truncation-of-floating-point-values"></a>Abschneiden von Gleitkommawerten

**ANSI 3.2.1.4** Die Richtung des Verkürzens oder Rundens, wenn eine Gleitkommazahl in eine geringere Gleitkommazahl konvertiert wird

Wenn ein Unterlauf auftritt, wird der Wert einer Gleitkommavariablen auf Null gerundet. Ein Überlauf verursacht möglicherweise einen Laufzeitfehler, oder er führt möglicherweise zu einem unvorhersehbaren Wert, je nach angegebenen Optimierungen.
