# <a name="blank-lines"></a>Leerzeilen

**ANSI 4.9.2** Ob Leerzeichen, die unmittelbar vor einem Zeilenumbruchzeichen in einen Textstream geschrieben werden, beim Lesen angezeigt werden

Leerzeichen werden beibehalten.
