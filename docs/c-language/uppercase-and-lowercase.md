# <a name="uppercase-and-lowercase"></a>Groß- und Kleinschreibung
**ANSI 3.1.2** Ob Unterschiede bei der Groß-/Kleinschreibung wichtig sind  
  
 Microsoft C beachtet die Groß-/Kleinschreibung bei Bezeichnern innerhalb einer Kompilierungseinheit.  
  
 Der Microsoft-Linker beachtet die Groß-/Kleinschreibung. Sie müssen alle Bezeichner entsprechend der Groß-/Kleinschreibung einheitlich festlegen.
