**ANSI 4.9.2** Die Anzahl von NULL-Zeichen, die an Daten angehängt werden können, die in einen Binärstream geschrieben werden

Eine beliebige Anzahl von NULL-Zeichen kann an einen Binärstream angehängt werden.
