# <a name="type-char"></a>Typ "char"

Der **`char`** -Typ wird verwendet, um den Integerwert eines Members des darstellbaren Zeichensatzes zu speichern. Dieser ganzzahlige Wert ist der ASCII-Code, der dem angegebenen Zeichen entspricht.

**Microsoft-spezifisch**

Zeichenwerte des Typs **`unsigned char`** haben einen Bereich von 0 bis zu den hexadezimalen Zahlen 0xFF. Ein **`signed char`** hat einen Bereich von 0x80 bis 0x7F.
