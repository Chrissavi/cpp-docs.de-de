# <a name="atexit-function-c"></a>atexit-Funktion (C)

**ANSI 4.10.4.3** Der von der `atexit`-Funktion zurückgegebene Status, wenn der Wert des Arguments nicht Null, „EXIT_SUCCESS“ oder „EXIT_FAILURE“ lautet

Die `atexit`-Funktion gibt Null zurück, wenn der Vorgang erfolgreich war, sonst einen Wert ungleich Null.
