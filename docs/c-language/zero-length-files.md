# <a name="zero-length-files"></a>Dateien der Länge 0 (null)

**ANSI 4.9.3** Gibt an, ob eine Datei mit einer Länge von Null tatsächlich vorhanden ist

Dateien mit einer Länge von null (0) sind zulässig.
