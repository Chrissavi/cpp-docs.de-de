# <a name="file-position-in-append-mode"></a>Dateiposition im Append-Modus

**ANSI 4.9.3** Ob sich der Dateipositionsindikator eines Append-Modusstreams zuerst am Anfang oder am Ende der Datei befindet

Wenn eine Datei im Append-Modus geöffnet wird, zeigt der Dateipositionszeiger anfänglich auf das Ende der Datei.
