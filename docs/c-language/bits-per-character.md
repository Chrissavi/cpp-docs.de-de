# <a name="bits-per-character"></a>Bits pro Zeichen

**ANSI 2.2.4.2.1** Anzahl von Bits in einem Zeichen

Die Anzahl von Bits in einem Zeichen wird durch die Manifestkonstante **CHAR_BIT** dargestellt. Die LIMITS.H-Datei definiert **CHAR_BIT** als 8.
