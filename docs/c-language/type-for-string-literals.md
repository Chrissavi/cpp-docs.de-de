# <a name="type-for-string-literals"></a>Typ für Zeichenfolgenliterale

Der Typarray von Zeichenfolgenliteralen lautet **`char`** (d. h. **char[ ]** ). (Der Typarray von Breitzeichenzeichenfolgen lautet **`wchar_t`** (d. h. **wchar_t[ ]** ).) Dies bedeutet, dass eine Zeichenfolge ein Array mit Elementen vom Typ **`char`** ist. Die Anzahl der Elemente im Array entspricht der Anzahl von Zeichen in der Zeichenfolge plus einem beendenden Null-Zeichen.
