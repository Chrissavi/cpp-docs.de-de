# <a name="sign-of-bit-fields"></a>Vorzeichen von Bitfeldern

**ANSI 3.5.2.1** Ob ein „einfaches“-Feld als `int`signed int **-Bitfeld oder als** unsigned int-Bitfeld behandelt wird

Es gibt Bitfelder mit oder ohne Vorzeichen. Einfache Bitfelder werden als signiert behandelt.
