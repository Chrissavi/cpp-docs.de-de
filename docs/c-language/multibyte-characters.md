# <a name="multibyte-characters"></a>Mehrbytezeichen

**ANSI 2.2.1.2** Umschaltzustände für Multibytezeichen

Mehrbytezeichen werden von einigen Implementierungen, einschließlich Microsoft C, verwendet, um die Fremdsprachenzeichen darzustellen, die nicht im Basiszeichensatz enthalten sind. Allerdings unterstützt Microsoft C keine zustandsabhängigen Codierungen. Daher gibt es keine Umschaltzustände. Weitere Informationen erhalten Sie unter [Multibyte- und Breitzeichen](../c-language/multibyte-and-wide-characters.md).
