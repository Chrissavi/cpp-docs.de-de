# <a name="clock-function-c"></a>clock-Funktion (C)
**ANSI 4.12.2.1** Der Zeitraum der `clock`-Funktion  
  
 Der Zeitraum der `clock`-Funktion startet (mit einem Wert von 0), wenn das C-Programm ausgeführt wird. Es werden Zeiten zurückgegeben, die in 1/**CLOCKS_PER_SEC**(was 1/1000 für Microsoft C entspricht) gemessen werden.
