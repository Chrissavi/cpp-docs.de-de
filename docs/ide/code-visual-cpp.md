  
#### <a name="parameters"></a>Parameter  
*content*<br/>
Der Text, der als Code gekennzeichnet werden soll.

 Verwenden Sie [\<c>](../ide/c-visual-cpp.md), um anzugeben, dass ein Teil des Texts als Code gekennzeichnet werden soll.
