# <a name="creating-desktop-projects-by-using-application-wizards"></a>Erstellen von Desktopprojekten mit Anwendungs-Assistenten

Alle Visual C++-Projekttypen verfügen über einen Anwendungs-Assistenten, mit dem Sie neue Projekte schnell und einfach in Anlehnung an die Projektvorlage erstellen können.  Verwenden Sie zum Öffnen eines Anwendungs-Assistenten das Dialogfeld **Neues Projekt**, um Projekteigenschaften anzugeben, z.B. den Namen des Projekts oder das Verzeichnis und die Projektmappe, in der Ihr Projekt gespeichert wird.

#### <a name="to-open-a-visual-c-application-wizard"></a>So öffnen Sie einen Visual C++-Anwendungs-Assistenten

1. Klicken Sie im Menü **Datei** auf **Neu**und dann auf **Projekt**.

   Das Dialogfeld **Neues Projekt** wird angezeigt.

1. Wählen Sie im Bereich „Projekttypen“ den Ordner **Visual C++-Projekte** aus. Im Bereich "Vorlagen" werden Symbole der einzelnen C++-Projekttypen angezeigt.

1. Klicken Sie im Bereich "Vorlagen" auf ein Symbol, um einen Projekttyp auszuwählen. Unterhalb beider Bereiche wird eine Meldung angezeigt, die den zu erstellenden Projekttyp angibt.

1. Legen Sie die Projekteigenschaften fest, oder überspringen Sie diesen Schritt, um die Standardprojekteigenschaften von Visual Studio zu verwenden. Ausführliche Informationen finden Sie unter [Working with Project Properties (Arbeiten mit Projekteigenschaften)](../ide/working-with-project-properties.md).

1. Klicken Sie auf **OK**, um den Assistenten für Ihren Projekttyp zu öffnen. Das Hilfethema zum Assistenten finden Sie, indem Sie**F1** drücken.

Beim Erstellen eines Projekts können Sie entweder eine neue Projektmappe erstellen oder das Projekt in eine vorhandene Projektmappe aufnehmen. Weitere Informationen finden Sie unter [Vorgehensweise: Erstellen eines C++-Projekts aus vorhandenem Code](../ide/how-to-create-a-cpp-project-from-existing-code.md).
