# <a name="trytolockt-structure"></a>try_to_lock_t-Struktur

Stellt eine **Struktur** dar, die ein [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) -Objekt definiert. Wird verwendet, um einen der überladenen Konstruktoren von [unique_lock](../standard-library/unique-lock-class.md) auszuwählen.

## <a name="syntax"></a>Syntax

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Anforderungen

**Header:** \<mutex>

**Namespace:** std
