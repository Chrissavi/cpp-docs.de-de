# <a name="isvoid-class"></a>is_void-Klasse

Testet, ob der Typ ein void-Typ ist.

## <a name="syntax"></a>Syntax

```cpp
template <class T>
struct is_void;
```

### <a name="parameters"></a>Parameter

*T* der abzufragende Typ.

## <a name="remarks"></a>Hinweise

Eine Instanz des typpr√§dikats ist true, wenn der Typ *T* ist **"void"** oder eine cv-qualifizierte Form von **"void"**, andernfalls ist Sie false.

## <a name="example"></a>Beispiel

```cpp
// std__type_traits__is_void.cpp
// compile with: /EHsc
#include <type_traits>
#include <iostream>

struct trivial
    {
    int val;
    };

int main()
    {
    std::cout << "is_void<trivial> == " << std::boolalpha
        << std::is_void<trivial>::value << std::endl;
    std::cout << "is_void<void()> == " << std::boolalpha
        << std::is_void<void()>::value << std::endl;
    std::cout << "is_void<void> == " << std::boolalpha
        << std::is_void<void>::value << std::endl;

    return (0);
    }

```

```Output
is_void<trivial> == false
is_void<void()> == false
is_void<void> == true
```
