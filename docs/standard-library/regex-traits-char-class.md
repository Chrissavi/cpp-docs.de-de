# <a name="regex_traitsltchargt-class"></a>regex_traits&lt;Char&gt;-Klasse

Spezialisierung von `regex_traits` für **`char`** .

## <a name="syntax"></a>Syntax

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Bemerkungen

Die-Klasse ist eine explizite Spezialisierung der Klassen Vorlagen [regex_traits](../standard-library/regex-traits-class.md) für Elemente des Typs **`char`** (sodass Sie Bibliotheksfunktionen nutzen können, die Objekte dieses Typs bearbeiten).

## <a name="requirements"></a>Requirements (Anforderungen)

**Header:**\<regex>

**Namespace:** std
