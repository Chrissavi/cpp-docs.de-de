# <a name="regextraitsltwchartgt-class"></a>regex_traits&lt;wchar_t&gt;-Klasse

Spezialisierung von `regex_traits` für **wchar_t**.

## <a name="syntax"></a>Syntax

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Hinweise

Die-Klasse ist eine explizite Spezialisierung der Vorlagen Klasse [regex_traits](../standard-library/regex-traits-class.md) für Elemente des Typs **wchar_t** (sodass Sie Bibliotheksfunktionen nutzen kann, die Objekte dieses Typs bearbeiten).

## <a name="requirements"></a>Anforderungen

**Header:** \<regex >

**Namespace:** std
