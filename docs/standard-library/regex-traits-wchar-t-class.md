# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits&lt;wchar_t&gt;-Klasse

Spezialisierung der `regex_traits` für **wchar_t**.

## <a name="syntax"></a>Syntax

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Hinweise

Die-Klasse ist eine explizite Spezialisierung der Klassen Vorlage [regex_traits](../standard-library/regex-traits-class.md) für Elemente vom Typ **wchar_t** (sodass Sie Bibliotheksfunktionen nutzen können, die Objekte dieses Typs bearbeiten).

## <a name="requirements"></a>Anforderungen

**Header:** \<regex >

**Namespace:** std
