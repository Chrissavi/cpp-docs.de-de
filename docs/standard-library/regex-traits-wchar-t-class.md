# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits&lt;wchar_t&gt;-Klasse

Spezialisierung von `regex_traits` für **`wchar_t`** .

## <a name="syntax"></a>Syntax

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Bemerkungen

Die-Klasse ist eine explizite Spezialisierung der Klassen Vorlagen [regex_traits](../standard-library/regex-traits-class.md) für Elemente des Typs **`wchar_t`** (sodass Sie Bibliotheksfunktionen nutzen können, die Objekte dieses Typs bearbeiten).

## <a name="requirements"></a>Requirements (Anforderungen)

**Header:**\<regex>

**Namespace:** std
