# <a name="ltscoped_allocatorgt"></a>&lt;scoped_allocator&gt;

Definiert die Container-Klassen Vorlage scoped_allocator.

## <a name="syntax"></a>Syntax

```cpp
#include <scoped_allocator>
```

### <a name="operators"></a>Operatoren

|Operator|Beschreibung|
|-|-|
|[Operator!=](../standard-library/scoped-allocator-operators.md#op_neq)|Testet, ob das scoped_allocator-Objekt auf der linken Seite des Operators ungleich dem Listenobjekt auf der rechten Seite ist.|
|[operator==](../standard-library/scoped-allocator-operators.md#op_eq_eq)|Testet, ob das scoped_allocator-Objekt auf der linken Seite des Operators gleich dem Listenobjekt auf der rechten Seite ist.|

### <a name="classes"></a>Klassen

|Klasse|Beschreibung|
|-|-|
|[scoped_allocator_adaptor-Klasse](../standard-library/scoped-allocator-adaptor-class.md)|Eine Klassen Vorlage, die ein Schachteln von einem oder mehreren Zuweisungen kapselt.|
