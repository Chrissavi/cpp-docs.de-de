# <a name="container-classdifferencetype"></a>Container-Klasse::difference_type

> [!NOTE]
> Dieses Thema ist in der Microsoft C++ -Dokumentation als nicht funktionales Beispiel für Container, die C++ in der Standard Bibliothek verwendet werden. Weitere Informationen finden Sie unter [C++-Standardbibliothekcontainer](../standard-library/stl-containers.md).

Beschreibt ein Objekt, das die Differenz zwischen den Adressen von zwei beliebigen Elementen in der gesteuerten Sequenz darstellen kann

## <a name="syntax"></a>Syntax

```

typedef T1 difference_type;
```

## <a name="remarks"></a>Hinweise

Sie wird hier als Synonym für den nicht angegebenen Typ `T1` (in der Regel `Alloc::difference_type`) beschrieben.
