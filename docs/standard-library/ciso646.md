# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Schließt den Standard-C-Bibliotheksheader \<iso646.h> ein und fügt die verknüpften Namen zum `std`-Namespace hinzu.

> [!NOTE]
> Verwendung dieses Headers ist nicht erforderlich, im C++.

## <a name="requirements"></a>Anforderungen

**Header:** \<ciso646 >

**Namespace:** std

## <a name="remarks"></a>Hinweise

Durch Einschließen dieses Headers wird sichergestellt, dass die mit externer Bindung im Standard-C-Bibliotheksheader deklarierten Namen im `std`-Namespace deklariert werden.
