# <a name="ltcwctypegt"></a>&lt;cwctype&gt;

Schließt den Standard-C-Bibliotheksheader \<wctype.h> ein und fügt die verknüpften Namen zum `std`-Namespace hinzu.

## <a name="syntax"></a>Syntax

```cpp
#include <cwctype>
```

## <a name="remarks"></a>Hinweise

Durch Einschließen dieses Headers wird sichergestellt, dass die mit externer Bindung im Standard-C-Bibliotheksheader deklarierten Namen im `std`-Namespace deklariert werden.

## <a name="constants"></a>Konstanten

```cpp
namespace std {
    using wint_t = see below ;
    using wctrans_t = see below ;
    using wctype_t = see below ;
}

#define WEOF see below
```
