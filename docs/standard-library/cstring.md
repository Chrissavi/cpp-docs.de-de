# <a name="ltcstringgt"></a>&lt;cstring&gt;

Schließt den Standard-C-Bibliotheksheader \<string.h> ein und fügt die verknüpften Namen zum `std`-Namespace hinzu.

## <a name="syntax"></a>Syntax

```cpp
#include <cstring>
```

## <a name="remarks"></a>Hinweise

Durch Einschließen dieses Headers wird sichergestellt, dass die mit externer Bindung im Standard-C-Bibliotheksheader deklarierten Namen im `std`-Namespace deklariert werden.

## <a name="constants"></a>Konstanten

```cpp
namespace std {
    using size_t = see below;
}

#define NULL
```
