# <a name="ltcwchargt"></a>&lt;cwchar&gt;

Schließt den Standard-C-bibliotheksheader \<uchar.h > und fügt die verknüpften Namen zum die `std` Namespace.

## <a name="requirements"></a>Anforderungen

**Header:** \<Cuchar >

**Namespace:** std

## <a name="member-functions"></a>Memberfunktionen

|||
|-|-|
|[c16rtomb](../standard-library/cuchar-functions.md#c16rtomb)|Keine Beschreibung.|
|[c32rtomb](../standard-library/cuchar-functions.md#c32rtomb)|Keine Beschreibung.|
|[mbrtoc16](../standard-library/cuchar-functions.md#mbrtoc16)|Keine Beschreibung.|
|[mbrtoc32](../standard-library/cuchar-functions.md#mbrtoc32)|Keine Beschreibung.|

## <a name="remarks"></a>Hinweise

Durch Einschließen dieses Headers wird sichergestellt, dass die mit externer Bindung im Standard-C-Bibliotheksheader deklarierten Namen im `std`-Namespace deklariert werden.
