# <a name="safeint-library"></a>SafeInt-Bibliothek

**SafeInt** ist eine portable Bibliothek, die mit MSVC, GCC oder Clang verwendet werden kann, um Ganzzahlüberläufe zu verhindern, die möglicherweise aus von der Anwendung durchgeführten mathematischen Operationen resultieren könnten. Die neueste Version dieser Bibliothek finden Sie unter [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt) .
