# <a name="expression-evaluator-error-cxx0043"></a>Ausdrucksauswertungsfehler CXX0043

Dieser Zeiger wird außerhalb der Member-Funktion verwendet.

Der- **`this`** Zeiger kann nur für nicht statische Member-Funktionen verwendet werden.

Dieser Fehler ist mit CAN0043 identisch.
