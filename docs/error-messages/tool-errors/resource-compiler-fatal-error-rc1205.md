# <a name="resource-compiler-fatal-error-rc1205"></a>Ressourcencompiler: Schwerwiegender Fehler RC1205
Ungültige Codepage  
  
 Angeben der Codepage (/ C) folgte Option eine ungültige Codepage. Finden Sie unter [Codepages](../../c-runtime-library/code-pages.md) in der *Run-Time Library Reference* für Weitere Informationen.
