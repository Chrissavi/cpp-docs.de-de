# <a name="bscmake-error-bk1509"></a>BSCMAKE-Fehler BK1509

nicht genügend Heap Speicher.

Für BSCMAKE ist nicht genügend Arbeitsspeicher verfügbar, einschließlich des virtuellen Arbeitsspeichers.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)

1. Freigeben von Speicherplatz.

1. Vergrößern Sie die Auslagerungs Datei.

1. Vergrößern Sie die Größe der Windows-Auslagerungs Datei.

1. Reduzieren Sie den Arbeitsspeicher, den BSCMAKE erfordert, indem Sie/Ei oder/es verwenden, um einige Eingabedateien auszuschließen, oder/em, um Makro Texte auszuschließen.
