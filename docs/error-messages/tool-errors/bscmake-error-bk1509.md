# <a name="bscmake-error-bk1509"></a>BSCMAKE-Fehler BK1509

kein weiterer Heapspeicher verfügbar

BSCMAKE war nicht genügend Arbeitsspeicher, einschließlich des virtuellen Speichers.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)

1. Geben Sie Festplattenspeicher frei.

1. Erhöhen Sie die Größe der Auslagerungsdatei.

1. Vergrößern Sie die Windows-Auslagerungsdatei.

1. Reduzieren Sie den Speicher, die, den mithilfe von Ei BSCMAKE erfordert, dass, oder/es einsetzen, um einige auszuschließen Eingabedateien oder/em um Makrotext zu beseitigen.
