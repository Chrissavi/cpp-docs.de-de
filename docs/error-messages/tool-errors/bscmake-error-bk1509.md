# <a name="bscmake-error-bk1509"></a>BSCMAKE-Fehler BK1509
Heap erschöpft  
  
 BSCMAKE war nicht genügend Arbeitsspeicher, einschließlich des virtuellen Speichers.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)  
  
1.  Geben Sie Speicherplatz frei.  
  
2.  Vergrößern der Auslagerungsdatei.  
  
3.  Vergrößern Sie die Windows-Auslagerungsdatei.  
  
4.  Reduzieren Sie die BSCMAKE erfordert, dass Ei mit Speicher oder/es einsetzen, um einige auszuschließen Eingabedateien oder/em, um Makrotext zu beseitigen.
