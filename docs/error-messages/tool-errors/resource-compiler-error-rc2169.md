# <a name="resource-compiler-error-rc2169"></a>Ressourcencompiler: Fehler RC2169

Ressourcen-Dateiname entspricht nicht 2.03-Format

Ein Format wird von der angegebenen Ressource älter als Version 2.03 verwendet. Die Ressourcendatei konvertiert werden muss oder neu erstellt, mit dem Format für Version 3.00 oder höher.
