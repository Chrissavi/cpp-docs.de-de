# <a name="resource-compiler-error-rc2169"></a>Ressourcencompiler: Fehler RC2169

der Ressourcen Dateiname ist nicht im 2,03-Format.

Die angegebene Ressource hat ein Format vor Version 2,03 verwendet. Die Ressourcen Datei muss mit dem Format für Version 3,00 oder höher konvertiert oder neu erstellt werden.
