# <a name="c-runtime-errors-rxxxx"></a>C-Laufzeitfehler (Rxxxx)

Die C-Laufzeitbibliothek (CRT) eventuell einen Laufzeitfehler, wenn Ihre app geladen oder ausgeführt wird. Auch wenn jede Nachricht auf der Microsoft Visual verweist C++ -Laufzeitbibliothek, es bedeutet nicht, dass ein Fehler in der Bibliothek vorhanden ist. Diese Fehler geben entweder einen Fehler im Code Ihrer app oder eine Bedingung, die die Laufzeitbibliothek, z.B. zu wenig Arbeitsspeicher nicht verarbeiten kann. Endbenutzer Ihrer App möglicherweise dieser Fehler angezeigt, es sei denn, Ihre Schreiben Ihrer app zu verhindern, dass sie oder den Fehler zu erfassen und präsentieren einer benutzerfreundlichen Fehlermeldung Nachricht für Ihre Benutzer stattdessen.
