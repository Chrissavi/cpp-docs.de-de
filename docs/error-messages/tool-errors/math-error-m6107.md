# <a name="math-error-m6107"></a>Mathematischer Fehler M6107

nicht emuliert

Es wurde versucht, eine Coprozessoranweisung ausführen, die ist ungültig oder wird vom Emulator nicht unterstützt.

Das Programm mit Exitcode 135 wird beendet.
