# <a name="linker-tools-warning-lnk4216"></a>Linkertoolwarnung LNK4216
Exportierte Eintrag-Eintrag  
  
 Ein Einstiegspunkt wurde aus einer DLL exportiert. Der Einstiegspunkt einer DLL muss nicht exportiert werden.
