# <a name="linker-tools-error-lnk1196"></a>Linkertoolfehler LNK1196

Ungültiges oder beschädigtes Importobjekt: Unbekannte Version

Die Importbibliothek ist beschädigt. Erstellen Sie die Bibliothek neu.
