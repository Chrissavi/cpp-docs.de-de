# <a name="linker-tools-error-lnk1136"></a>Linkertoolfehler LNK1136

Ungültige oder beschädigte Datei.

Die Eingabedatei weist entweder einen beschädigten Header auf oder hat eine Größe von 0 (null) oder nicht normal.

Linkertoolfehler LNK1136 kann auch ausgelöst werden, wenn die Datei fälschlicherweise als zusätzliche Abhängigkeit angegeben wurde.
