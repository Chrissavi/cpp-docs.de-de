# <a name="command-line-warning-d9028"></a>Befehlszeilenwarnung D9028

> minimaler Neuerstellungs Fehler, Wiederherstellen des normalen Builds

Die Datei "Project. IDB" ist beschädigt. Löschen Sie die Datei, und erstellen Sie neu.

Weitere Informationen zu minimalen neubuilds und der IDB-Datei finden Sie in der veralteten [/GM-Compileroption](../../build/reference/gm-enable-minimal-rebuild.md).
