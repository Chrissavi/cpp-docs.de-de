# <a name="expression-evaluator-error-cxx0025"></a>Ausdrucksauswertungsfehler CXX0025

Operator benötigt Struktur/Union

Ein Operator, der einen Ausdruck von `struct`-oder **Union** -Typ annimmt, wurde auf einen Ausdruck angewendet, der keine `struct` oder **Union**ist.

Komponenten der Klassen-, Struktur-oder Union-Variablen müssen einen voll qualifizierten Namen aufweisen. Komponenten können nicht ohne vollständige Spezifikation eingegeben werden.

Dieser Fehler ist mit CAN0025 identisch.
