# <a name="command-line-error-d8036"></a>Befehlszeilenfehler D8036

"/Option" ist für mehrere Quelldateien unzulässig.

Diese Compileroptionen können nicht mit mehreren Quelldateien verwendet werden:

- Assemblydateiauflistung (/FA)

- Objektdatei umbenennen (/FO)

- Erstellen von Quell Browserinformationen ohne lokale Variablen (/FR)

- Erstellen von Quell Browserinformationen, die lokale Variablen enthalten (/FR)
