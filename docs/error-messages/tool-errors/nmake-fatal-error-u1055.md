# <a name="nmake-fatal-error-u1055"></a>NMAKE: Schwerwiegender Fehler U1055

nicht genügend Umgebungs Speicher

Das Betriebssystem hat nicht genügend Platz für Umgebungsvariablen.

Vergrößern Sie den Umgebungs Bereich, oder legen Sie weniger Umgebungsvariablen fest.
