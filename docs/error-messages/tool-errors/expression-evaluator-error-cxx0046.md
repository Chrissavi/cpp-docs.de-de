# <a name="expression-evaluator-error-cxx0046"></a>Ausdrucksauswertungsfehler CXX0046

Argumentliste für Member-Funktion erforderlich.

Ein Ausdruck, der als Member-Funktion bezeichnet wird, aber keine tatsächlichen Parameter angegeben hat.

Dieser Fehler ist mit CAN0046 identisch.
