# <a name="expression-evaluator-error-cxx0072"></a>Ausdrucksauswertungsfehler CXX0072
Fehler: Typ fehlen oder "Unbekannt"  
  
 Die PCH-Datei nicht verknüpft, oder der Code enthält einen Verweis auf einen Typ, der in einem Modul nicht mit/ZI kompiliert wird.  
  
 Dieser Fehler ist mit CAN0067 identisch.
