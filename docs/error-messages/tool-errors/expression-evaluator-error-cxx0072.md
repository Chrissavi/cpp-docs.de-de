# <a name="expression-evaluator-error-cxx0072"></a>Ausdrucksauswertungsfehler CXX0072

Fehler: Typinformationen fehlen oder sind unbekannt.

Die PCH-Datei wurde nicht verknüpft, oder der Code enthält einen Verweis auf einen Typ, der sich in einem Modul befindet, das nicht mit/Zi. kompiliert wurde.

Dieser Fehler ist mit CAN0067 identisch.
