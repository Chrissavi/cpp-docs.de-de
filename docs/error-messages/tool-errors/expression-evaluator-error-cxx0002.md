# <a name="expression-evaluator-error-cxx0002"></a>Ausdrucksauswertungsfehler CXX0002

Fehler beim Zugriff auf den Benutzerspeicher

Der Ausdruck versucht, auf Arbeitsspeicher zu verweisen, der nicht dem Programm zugeordnet ist, das gerade deentschlgt wird.

Dieser Fehler ist mit CAN0002 identisch.
