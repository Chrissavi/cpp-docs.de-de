# <a name="bscmake-error-bk1504"></a>BSCMAKE-Fehler BK1504

Position in Datei ' Dateiname ' nicht möglich [: Reason]

BSCMAKE konnte nicht an einen Speicherort in der Datei verschoben werden.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:

1. Ein vollständiger Datenträger.

1. Ein Hardwarefehler.

1. Datei Verkürzung aufgrund von unzureichendem Speicherplatz oder einer Unterbrechung des Compilers beim Erstellen der SBR-Datei.
