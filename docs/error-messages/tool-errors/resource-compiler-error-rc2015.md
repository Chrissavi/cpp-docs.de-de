# <a name="resource-compiler-error-rc2015"></a>Ressourcencompiler: Fehler RC2015

zu viele Zeichen in der Konstante

Eine Zeichenkonstante enthält mehr als zwei Zeichen.

Zeichenkonstanten sind ein Zeichen (standard-Zeichenkonstanten) oder zwei Zeichen (lange Zeichenkonstanten) auf.

Beachten Sie, dass eine Escapesequenz (z. B. \t für den Tabulator) in ein einzelnes Zeichen konvertiert wird.
