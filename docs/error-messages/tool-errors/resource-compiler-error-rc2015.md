# <a name="resource-compiler-error-rc2015"></a>Ressourcencompiler: Fehler RC2015

zu viele Zeichen in konstanter Größe

Eine Zeichen Konstante enthielt mehr als zwei Zeichen.

Zeichen Konstanten sind auf ein Zeichen (Standard Zeichen Konstanten) oder zwei Zeichen (lange Zeichen Konstanten) beschränkt.

Beachten Sie, dass eine Escapesequenz (z. b. \t für Tab) in ein einzelnes Zeichen konvertiert wird.
