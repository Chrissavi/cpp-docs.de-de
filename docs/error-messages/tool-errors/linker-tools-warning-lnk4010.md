# <a name="linker-tools-warning-lnk4010"></a>Linkertoolwarnung LNK4010

Ungültige subsystemversionsnummer. standardmäßige SubSystemVersion wird angenommen

Sie können eine Version für das Subsystem des Bilds angeben ([/Subsystem](../../build/reference/subsystem-specify-subsystem.md)). Jedes Subsystem hat eine minimale Versions Anforderung. Wenn die angegebene Version kleiner als das Minimum ist, tritt diese Warnung auf, und der Linker verwendet nur das Standard Subsystem.
