# <a name="linker-tools-warning-lnk4002"></a>Linkertoolwarnung LNK4002
im Objekt definiertes Symbol  
  
 Das Symbol, angezeigt in seiner ergänzten Form, wurde angegeben, in seiner nicht ergänzten Form im `object`, jedoch eine eindeutige Übereinstimmung mit einem ergänzten Symbol konnte nicht gefunden werden. Diese Warnung wird immer vorangestellt Warnung [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) und schwerwiegende Fehler [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
