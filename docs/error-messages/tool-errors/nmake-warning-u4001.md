# <a name="nmake-warning-u4001"></a>NMAKE: Warnung U4001

> die Befehlsdatei kann nur über die Befehlszeile aufgerufen werden.

Eine Befehlsdatei, die vom at Sign ( **\@** )-Spezifizierer aufgerufen wird, darf keine Spezifikation für eine andere Befehlsdatei enthalten. Eine solche Schachtelung ist nicht zulässig. Die Spezifikation wurde ignoriert.
