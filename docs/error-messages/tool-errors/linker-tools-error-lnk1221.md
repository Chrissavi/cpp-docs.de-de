# <a name="linker-tools-error-lnk1221"></a>Linkertoolfehler LNK1221
ein Subsystem kann nicht abgeleitet werden und muss definiert sein  
  
 Der Linker verfügt nicht über genügend Informationen, welches Subsystem abzuleiten, die verwendet werden soll.  
  
 Verwenden Sie zum Beheben dieses Fehlers [/Subsystem](../../build/reference/subsystem-specify-subsystem.md).
