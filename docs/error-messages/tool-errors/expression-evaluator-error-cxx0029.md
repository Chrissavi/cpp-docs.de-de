# <a name="expression-evaluator-error-cxx0029"></a>Ausdrucksauswertungsfehler CXX0029
kein Struct-Zeiger  
  
 Der Memberauswahloperator (**->**) angewendet wurde, um einen Ausdruck, der kein Zeiger auf eine Struktur ist.  
  
 Überprüfen Sie, dass der gesamte Ausdruck in Klammern ordnungsgemäß ist oder den Adressausdruck in die entsprechende Struktur Zeigertyp umgewandelt.  
  
 Dieser Fehler ist mit CAN0029 identisch.
