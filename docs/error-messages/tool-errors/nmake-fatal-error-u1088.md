# <a name="nmake-fatal-error-u1088"></a>NMAKE: Schwerwiegender Fehler U1088

Ungültiges Trennzeichen ':: ' bei Rückschluss Regel

Auf eine Rückschluss Regel muss ein einzelner Doppelpunkt ( **:** ) folgen.
