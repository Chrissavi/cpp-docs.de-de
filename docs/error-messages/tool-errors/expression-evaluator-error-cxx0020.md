# <a name="expression-evaluator-error-cxx0020"></a>Ausdrucksauswertungsfehler CXX0020

Operanden Typen sind für diesen Vorgang nicht schlecht.

Ein Operator wurde auf einen Ausdruck angewendet, der einen ungültigen Typ für diesen Operator hat.

Beispielsweise ist es nicht zulässig, die Adresse eines Registers zu übernehmen oder ein Array mit einem Gleit Komma Ausdruck zu abonnieren.

Dieser Fehler ist mit CAN0020 identisch.
