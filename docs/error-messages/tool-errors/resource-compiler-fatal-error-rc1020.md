# <a name="resource-compiler-fatal-error-rc1020"></a>Ressourcencompiler: Schwerwiegender Fehler RC1020

Unerwartetes "#endif".

Eine `#endif`-Direktive wurde ohne übereinstimmende `#if`, **#ifdef**oder **#ifndef** Direktive angezeigt.

Stellen Sie sicher, dass für jede `#if`-, **#ifdef**-und **#ifndef** -Anweisung eine passende `#endif` vorhanden ist.
