# <a name="bscmake-error-bk1508"></a>BSCMAKE-Fehler BK1508

die temporäre Datei "Dateiname" kann nicht gelöscht werden [: Reason]

BSCMAKE kann eine seiner temporären Dateien nicht löschen.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:

1. Die Datei ist in einem anderen Prozess geöffnet.

1. Ein Hardwarefehler.
