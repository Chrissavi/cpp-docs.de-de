# <a name="expression-evaluator-error-cxx0004"></a>Ausdrucksauswertungsfehler CXX0004

Syntaxfehler

Die Syntax des Ausdrucks ist falsch.

Geben Sie den Ausdruck mit der richtigen Syntax ein.

Dieser Fehler ist mit CAN0004 identisch.
