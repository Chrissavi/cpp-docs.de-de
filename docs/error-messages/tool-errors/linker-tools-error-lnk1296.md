# <a name="linker-tools-error-lnk1296"></a>Linkertoolfehler LNK1296
Dateiname kann nicht geladen  
  
 Die angegebene DLL war nicht verfügbar.  
  
 Die möglichen Speicherorte für die DLL sind das aktuelle Verzeichnis, das Verzeichnis "System", das Windows-Verzeichnis und in der PATH-Umgebungsvariablen angegebenen Verzeichnisse.
