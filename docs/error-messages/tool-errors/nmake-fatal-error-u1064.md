# <a name="nmake-fatal-error-u1064"></a>NMAKE: Schwerwiegender Fehler U1064

Makefile nicht gefunden und kein Ziel angegeben

In der NMAKE-Befehlszeile wurde kein Makefile oder ein Ziel angegeben, und das aktuelle Verzeichnis enthielt keine Datei mit dem Namen "Makefile".

NMAKE erfordert entweder ein Makefile-oder ein Befehlszeilen Ziel (oder beides). Um ein Makefile für NMAKE verfügbar zu machen, geben Sie entweder die Option/F an, oder platzieren Sie im aktuellen Verzeichnis eine Datei mit dem Namen Makefile. NMAKE kann ein Befehlszeilen Ziel mithilfe einer Inferenz-Regel erstellen, wenn kein Makefile-Element angegeben wird.
