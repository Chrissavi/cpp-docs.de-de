# <a name="nmake-fatal-error-u1064"></a>NMAKE: Schwerwiegender Fehler U1064
MAKEFILE nicht gefunden und kein Ziel angegeben  
  
 NMAKE-Befehlszeile wurde ein Makefile oder ein Ziel angegeben, und das aktuelle Verzeichnis enthielt eine Datei namens MAKEFILE nicht.  
  
 NMAKE erfordert entweder einem Makefile oder ein Befehlszeilen-Ziel (oder beides). Um einem Makefile NMAKE zur Verfügung stellen zu können, geben Sie die Option/f oder fügen Sie eine Datei mit dem Namen MAKEFILE im aktuellen Verzeichnis. NMAKE Erstellen eines Befehlszeilentools-Ziels mithilfe einer Rückschlussregel Makefile nicht angegeben ist.
