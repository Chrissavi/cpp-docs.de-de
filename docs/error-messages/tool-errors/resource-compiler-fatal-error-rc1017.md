# <a name="resource-compiler-fatal-error-rc1017"></a>Ressourcencompiler: Schwerwiegender Fehler RC1017

Ungültiger Ausdruck für Ganzzahlkonstante

Der Ausdruck in einem `#if` Richtlinie ist nicht vorhanden oder wird nicht auf eine Konstante ausgewertet.
