# <a name="linker-tools-warning-lnk4065"></a>Linkertoolwarnung LNK4065

'Funktion' kann nicht sortiert werden; ignoriert

Die angegebene Funktion wurde nicht als eine Paketfunktion kompiliert. Kompilieren Sie mit Hilfe [/Gy](../../build/reference/gy-enable-function-level-linking.md).
