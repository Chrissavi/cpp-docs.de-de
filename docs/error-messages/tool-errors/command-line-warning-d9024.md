Unbekannte Quelle file "Typ"Dateiname"", Objektdatei wird angenommen

Es werden die folgenden Erweiterungen erkannt:

- .cpp (C++-Quelldatei)

- obj (Objekt-Datei)

- LIB (Library-Datei)
