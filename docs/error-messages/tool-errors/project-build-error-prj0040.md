# <a name="project-build-error-prj0040"></a>Projektbuildfehler PRJ0040
Interner Fehler beim Buildvorgang. Kann nicht fortgesetzt werden. Laden Sie Projekt erneut, und wiederholen Sie den Vorgang.  
  
 Das Projektsystem konnte das Projekt verarbeiten.  
  
 Laden Sie das Projekt erneut.
