# <a name="project-build-error-prj0040"></a>Projektbuildfehler PRJ0040

Interner Fehler beim erstellen. Der Vorgang kann nicht fortgesetzt werden. Laden Sie das Projekt erneut, und versuchen Sie es erneut.

Das Projekt System konnte das Projekt nicht verarbeiten.

Versuchen Sie, das Projekt erneut zu laden.
