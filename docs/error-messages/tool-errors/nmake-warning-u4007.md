# <a name="nmake-warning-u4007"></a>NMAKE: Warnung U4007

der Dateiname 'Dateiname' zu lang sein. bis 8.3 Abschneiden

Der Basisname der angegebenen Datei verfügt über mehr als acht Zeichen oder die Erweiterung hat mehr als drei Zeichen. NMAKE: abgeschnitten, den Namen in eine 8-Zeichen-Basis und eine drei Zeichen.

Wenn von Ihrem Dateisystem lange Dateinamen unterstützt werden, schließen Sie den Namen in doppelte Anführungszeichen (**"**).
