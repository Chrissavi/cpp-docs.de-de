# <a name="nmake-warning-u4007"></a>NMAKE: Warnung U4007

Dateiname ' Dateiname ' ist zu lang. Abschneiden auf 8,3

Der Basisname der angegebenen Datei enthält mehr als acht Zeichen, oder die Erweiterung hat mehr als drei Zeichen. NMAKE hat den Namen auf eine 8-Zeichen-Basis und eine Erweiterung mit drei Zeichen abgeschnitten.

Wenn lange Dateinamen von Ihrem Dateisystem unterstützt werden, müssen Sie den Namen in doppelte Anführungszeichen ( **"** ) einschließen.
