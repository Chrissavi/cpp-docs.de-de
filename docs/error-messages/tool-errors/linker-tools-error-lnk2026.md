# <a name="linker-tools-error-lnk2026"></a>Linkertoolfehler LNK2026

Unsicheres Modul für SAFESEH-Image

[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) wurde angegeben, aber ein Modul war nicht mit der Funktion für die sichere Ausnahmebehandlung kompatibel. Wenn Sie dieses Modul mit **/SAFESEH**verwenden möchten, müssen Sie das Modul neu kompilieren.
