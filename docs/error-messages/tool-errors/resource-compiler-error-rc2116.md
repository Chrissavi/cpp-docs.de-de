Zahl erwartet als ID  
  
 Erwarten eine Zahl für die `id` -Feld der Control-Anweisung in der **Dialogfeld** Anweisung. Stellen Sie sicher, dass eine Zahl oder `#define` -Anweisung für die Steuerelement-ID.
