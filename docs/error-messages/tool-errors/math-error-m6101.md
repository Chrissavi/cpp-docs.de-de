# <a name="math-error-m6101"></a>Mathematischer Fehler M6101

ungültig

Ungültiger Vorgang.

Dieser Fehler kann verursacht werden, wenn ein Operand NAN (not a Number) oder Infinity ist.

Das Programm endet mit dem Exitcode 129.
