# <a name="nmake-fatal-error-u1077"></a>NMAKE: Schwerwiegender Fehler U1077
'Programm': Rückgabecode "Value"  
  
 Der angegebene Befehl oder das Programm Namen NMAKE ist fehlgeschlagen, und den angegebene Exitcode zurückgegeben.  
  
 Dieser Fehler unterdrücken und NMAKE-Sitzung zu fortfahren, verwenden Sie die Option/i der **. IGNORIEREN** Punkt-Direktive und den Bindestrich (**-**) Befehlsparameter. Zum Fortsetzen der NMAKE-Sitzungs für nicht verbundene Teile der Abhängigkeitsstruktur verwenden Sie die Option/k.
