# <a name="expression-evaluator-error-cxx0066"></a>Ausdrucksauswertungsfehler CXX0066

statischer Member ist nicht vorhanden.

Ein statischer Member einer Klasse konnte nicht gefunden werden oder wurde nicht definiert. Dieser Fehler kann auf einen statischen Klassenmember zurückzuführen sein, der deklariert, aber nicht definiert ist, oder er wird nur definiert und in Modulen verwiesen, die keine Debuginformationen enthalten.

Dieser Fehler ist mit CAN0066 identisch.
