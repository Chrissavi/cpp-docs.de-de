# <a name="linker-tools-warning-lnk4220"></a>Linkertoolwarnung LNK4220

Ungültige ' Linkeroption ' Wert ' Wert '; Standardwert angenommen

Mit der [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) -Option wurde ein Wert außerhalb des gültigen Bereichs angegeben. Der Standardwert für **/TLBID** ist 1.
