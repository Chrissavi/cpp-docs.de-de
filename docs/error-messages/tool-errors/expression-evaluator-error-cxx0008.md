# <a name="expression-evaluator-error-cxx0008"></a>Ausdrucksauswertungsfehler CXX0008

**"am Ende der Zeichenfolge fehlt."**

Das doppelte Anführungszeichen, das am Ende des Zeichenfolgenliterals erwartet wurde, fehlte. Geben Sie den Ausdruck erneut ein, und schließen Sie das Zeichen folgen Literalzeichen in doppelte Anführungszeichen ein.

Dieser Fehler ist mit CAN0008 identisch.
