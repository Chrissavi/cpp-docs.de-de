# <a name="profile-guided-optimization-warning-pg1039"></a>Warnung bei profilgesteuerter Optimierung: PG1039

Fehler beim Warten auf das Löschen der stillen Zeit.

Die Anwendung hat das Ausführen von [PGOSWEEP](../../build/pgosweep.md) nicht zugelassen.  Wenn möglich, führen Sie pgosweep aus, wenn sich die Anwendung im Leerlauf befindet.
