# <a name="expression-evaluator-error-cxx0009"></a>Ausdrucksauswertungsfehler CXX0009

Fehlendes ' nach Zeichenkonstante

Das einfache Anführungszeichen am Ende die Zeichenkonstante erwartet war nicht vorhanden. Geben Sie den Ausdruck an, die Zeichenkonstante in einfache Anführungszeichen einschließen.

Dieser Fehler ist mit CAN0009 identisch.
