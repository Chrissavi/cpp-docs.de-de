# <a name="expression-evaluator-error-cxx0009"></a>Ausdrucksauswertungsfehler CXX0009
Fehlendes ' nach Zeichenkonstante  
  
 Das einfache Anführungszeichen am Ende der Zeichenkonstante erwartet fehlte. Geben Sie den Ausdruck, die Zeichenkonstante in einfache Anführungszeichen einschließen.  
  
 Dieser Fehler ist mit CAN0009 identisch.
