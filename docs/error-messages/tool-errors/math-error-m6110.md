# <a name="math-error-m6110"></a>Mathematischer Fehler M6110

Stapelüberlauf

Ein Gleit Komma Ausdruck hat einen Gleit Komma Stapelüberlauf verursacht.

Stapelüberlauf-Gleit Komma Ausnahmen werden zusätzlich zu den acht Ebenen, die normalerweise vom 8087/287/387-Coprozessor unterstützt werden, bis zu einem Grenzwert von sieben Ebenen erfasst.

Das Programm endet mit dem Exitcode 138.
