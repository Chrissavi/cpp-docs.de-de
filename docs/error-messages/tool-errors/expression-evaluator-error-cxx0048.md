# <a name="expression-evaluator-error-cxx0048"></a>Ausdrucksauswertungsfehler CXX0048

Aufruf Sequenz wird nicht unterstützt.

Eine Funktion, die im Ausdruck angegeben ist, verwendet eine Aufruf Sequenz, die von der C-Ausdrucks Auswertung nicht unterstützt wird. Diese Funktion kann nicht in einem Überwachungsfenster Ausdruck aufgerufen werden.

Dieser Fehler ist mit CAN0048 identisch.
