# <a name="bscmake-error-bk1517"></a>BSCMAKE-Fehler BK1517

Quelldatei für Sbrfile, kompiliert mit/Yc und/Yu

Die SBR-Datei bezieht sich auf sich selbst. Sie wurde wahrscheinlich nach der Kompilierung mit/Yc. mit/Yu neu kompiliert. Setzen Sie die Compileroption für die Quelldatei auf/Yc, und wählen Sie neu **Erstellen** aus, um neue SBR-Dateien zu generieren. Kompilieren Sie die Quelldatei nicht mit/Yu. erneut.
