# <a name="expression-evaluator-error-cxx0058"></a>Ausdrucksauswertungsfehler CXX0058
Der überladene Operator wurde nicht gefunden.  
  
 Ein Klassentyp wurde angegeben, wie des linken Operand in einem Ausdruck, jedoch ein überladener Operator für die Klasse nicht definiert wurde.  
  
 Dieser Fehler ist mit CAN0058 identisch.
