# <a name="expression-evaluator-error-cxx0058"></a>Ausdrucksauswertungsfehler CXX0058

überladener Operator nicht gefunden

Ein Klassentyp wurde als Linker Operand in einem Ausdruck angegeben, aber für die Klasse wurde kein überladener Operator definiert.

Dieser Fehler ist mit CAN0058 identisch.
