# <a name="resource-compiler-fatal-error-rc1208"></a>Ressourcencompiler: Schwerwiegender Fehler RC1208

Eingabedatei hat. RES-Erweiterung

Die. RES-Erweiterung wird für die Compilerausgabe Ressource verwendet. Die. RC-Erweiterung sollte für den Ressourcencompiler-Skripts verwendet werden.
