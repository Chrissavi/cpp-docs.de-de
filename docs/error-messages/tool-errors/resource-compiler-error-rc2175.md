# <a name="resource-compiler-error-rc2175"></a>Ressourcencompiler: Fehler RC2175

der Ressourcen Dateiname ist nicht im 3,00-Format.

Die angegebene Ressource hat ein Format vor Version 3,00 verwendet. Die Ressourcen Datei muss mit dem Format für Version 3,00 oder höher konvertiert oder neu erstellt werden.
