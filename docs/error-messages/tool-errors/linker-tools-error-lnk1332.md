# <a name="linker-tools-error-lnk1332"></a>Linkertoolfehler LNK1332

Es wurde festgestellt,\<Anzahl > Windows-Runtime Typen in einem Modul importiert und in einem anderen Modul definiert wurde.

Beim Erzeugen des aktuellen Ziels hat der Linker <`count`> Windows-Runtime Typen erkannt, die jeweils in einem Modul importiert und auch in einem anderen Modul definiert sind.

### <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

- Korrigieren Sie jeden der Linkertoolfehler lnk2039-Fehler im Build entsprechend dem Vorschlag in der Fehlermeldung.
