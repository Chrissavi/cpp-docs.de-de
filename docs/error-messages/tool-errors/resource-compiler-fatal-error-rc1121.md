# <a name="resource-compiler-fatal-error-rc1121"></a>Ressourcencompiler: Schwerwiegender Fehler RC1121

E/A-Fehler beim Lesen der Datei

Der Ressourcencompiler konnte eine Datei nicht lesen. Überprüfen Sie, ob das Laufwerk, das die Datei enthält, verfügbar und die Datei gültig ist.
