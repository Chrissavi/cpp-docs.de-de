# <a name="expression-evaluator-error-cxx0011"></a>Ausdrucksauswertungsfehler CXX0011

Fehlende schließende Klammer

Der Ausdruck enthält unausgeglichene eckigen Klammern. Geben Sie den Ausdruck mit ausgeglichene eckige Klammern ein.

Dieser Fehler ist identisch mit CAN0011.
