# <a name="linker-tools-error-lnk1113"></a>Linkertoolfehler LNK1113
Ungültiger Typ Computertyp  
  
 Die im Objektheader angegebene Computertyp ist ungültig.  
  
 Dieser Fehler kann verursacht werden, wenn die Datei beschädigt ist. Erstellen Sie das Objekt neu.
