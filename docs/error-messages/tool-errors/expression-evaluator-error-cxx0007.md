# <a name="expression-evaluator-error-cxx0007"></a>Ausdrucksauswertungsfehler CXX0007

fehlende Rechte Klammer

Im Ausdruck wurden unausgeglichene Klammern gefunden. Geben Sie den Ausdruck mit ausgeglichenen Klammern erneut ein.

Dieser Fehler ist mit CAN0007 identisch.
