# <a name="expression-evaluator-error-cxx0018"></a>Ausdrucksauswertungsfehler CXX0018

Ungültiger Registrierungs Name

Ein angegebenes Register ist nicht vorhanden oder kann nicht angezeigt werden.

Der Überwachungsfenster kann die folgenden Register anzeigen:

AX SP DS IP

BX BP ist FL

CX Si SS-GS

DX di CS SS

EAX ESP DS EIP

ebx EBP es EFL

ECX-ESI FS

EDX    EDI    CS

Dieser Fehler ist mit CAN0018 identisch.
