# <a name="linker-tools-warning-lnk4205"></a>Linkertoolwarnung LNK4205

für "filename" fehlen aktuelle Debuginformationen für das Verweis Modul. Objekt wird verknüpft, als ob keine Debuginformationen

Die PDB-Datei enthält veraltete Informationen. Der Linker verknüpft weiterhin das Objekt ohne Debuginformationen. Möglicherweise möchten Sie die Objektdatei mit der [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) -Option neu kompilieren.
