# <a name="resource-compiler-fatal-error-rw1023"></a>Ressourcencompiler: Schwerwiegender Fehler RW1023

E/a-Fehler beim Schreiben der Datei, vollständiges Laufwerk

Der freie Speicherplatz muss mindestens doppelt so groß wie die ausführbare Datei sein, die Sie erstellen.
