# <a name="linker-tools-error-lnk1240"></a>Linkertoolfehler LNK1240

IDL-Inhalt konnte nicht kompiliert werden.

Der Linker hat eine mittlere Zeit für die Kompilierung der eingebetteten IDL erzeugt, aber es ist ein Problem aufgetreten. Überprüfen Sie die von "Mittel l" angegebenen Fehler.
