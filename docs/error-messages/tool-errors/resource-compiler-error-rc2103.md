# <a name="resource-compiler-error-rc2103"></a>Ressourcencompiler: Fehler RC2103

Unerwartetes Dateiende in Zeichenfolgenliteral

Eine Ende der Datei wurde vor dem Ende einer Zeichenfolge gefunden. Die Zeichenfolge fehlt möglicherweise ein schließendes doppeltes Anführungszeichen (**"**).
