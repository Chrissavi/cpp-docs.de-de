# <a name="nmake-fatal-error-u1087"></a>NMAKE: Schwerwiegender Fehler U1087

: und:: abhängigen Elemente für dasselbe Ziel nicht möglich

Ein Ziel kann nicht sowohl mit einem Doppelpunkt ( **:** ) als auch mit einer Doppelpunkt Abhängigkeit (`::`) angegeben werden.

Um ein Ziel in mehreren Beschreibungs Blöcken anzugeben, verwenden Sie `::` in den einzelnen Abhängigkeits Zeilen.
