# <a name="project-build-error-prj0027"></a>Projektbuildfehler PRJ0027

Die Unicode-Protokollmeldung "Content" enthält Inhalt, der nicht in die ANSI-Codepage des Benutzers übersetzt werden konnte.

Diese Warnung wird in der Regel nur in Verbindung mit Fehlern beim Erstellen von Batch-und/oder Antwort Dateien angezeigt.

Die Lösung für diesen Fehler besteht darin, den Inhalt des Buildprotokolls so zu aktualisieren, dass ANSI verwendet wird, oder die Codepage auf dem Computer zu installieren und als Standardsystem festzulegen.
