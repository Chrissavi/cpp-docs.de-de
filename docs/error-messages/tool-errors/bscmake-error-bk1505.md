# <a name="bscmake-error-bk1505"></a>BSCMAKE-Fehler BK1505

aus Datei ' Dateiname ' kann nicht gelesen werden [: Reason]

BSCMAKE kann nicht aus der Datei lesen.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:

1. Datei Beschädigung.

1. Das Abschneiden der Datei an den Compiler ist nicht genügend Speicherplatz, oder die Datei wird während der Erstellung der SBR-Datei unterbrochen.
