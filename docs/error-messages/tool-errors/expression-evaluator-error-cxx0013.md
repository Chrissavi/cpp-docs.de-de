# <a name="expression-evaluator-error-cxx0013"></a>Ausdrucksauswertungsfehler CXX0013
Operator fehlt  
  
 Ein Operator wurde im Ausdruck erwartet, aber es wurde nicht gefunden. Überprüfen Sie die Syntax des Ausdrucks.  
  
 Dieser Fehler ist mit CAN0013 identisch.
