BSCMAKE kann nicht auf eine Browseinformationsdatei erstellen, da die Anzahl der Definitionen "," Verweise "," Module "oder" Weitere Informationen über den Grenzwert überschreitet.

1. Schließen Sie einige Informationen, die mit/em, / es oder Ei.

1. Lassen Sie die IU-Option.
