# <a name="bscmake-error-bk1512"></a>BSCMAKE-Fehler BK1512

Dateiname: Kapazität überschritten

BSCMAKE kann eine Browseinformationsdatei nicht erstellen, da die Anzahl der Definitionen, Verweise, Module oder anderen Informationen den Grenzwert überschreitet.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)

1. Ausschließen einiger Informationen mithilfe von/EM,/es oder/Ei.

1. Lassen Sie die Option/iu aus.
