# <a name="resource-compiler-fatal-error-rc1105"></a>Ressourcencompiler: Schwerwiegender Fehler RC1105

Ungültiger Schalter, wählen Sie Option: zu viele/d-Schalter

Zu viele Symbole mit dem Definieren von Symbolen definiert (/ d) Option. Definieren Sie einige Symbole in der Quelle und die Neukompilierung an.
