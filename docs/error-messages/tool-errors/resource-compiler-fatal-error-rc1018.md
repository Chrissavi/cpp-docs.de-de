# <a name="resource-compiler-fatal-error-rc1018"></a>Ressourcencompiler: Schwerwiegender Fehler RC1018

Unerwartetes "#elif".

Die `#elif`-Direktive wurde nicht in einem `#if`-, **#ifdef**-oder **#ifndef** -Konstrukt angezeigt.

Stellen Sie sicher, dass vor dieser Anweisung eine `#if`-, **#ifdef**-oder **#ifndef** -Anweisung vorhanden ist.
