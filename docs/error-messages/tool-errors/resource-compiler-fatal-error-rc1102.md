# <a name="resource-compiler-fatal-error-rc1102"></a>Ressourcencompiler: Schwerwiegender Fehler RC1102

Interner Fehler: zu viele Argumente für rcpp.

Es wurden zu viele Argumente an den Ressourcen Compiler-Präprozessor übermittelt. Verringern Sie die Anzahl der Symbole, die mit der Option Symbole definieren (/d) definiert werden, indem Sie Sie in der Quelle definieren. Dieser Fehler kann auch verursacht werden, wenn zu viele include-Datei Suchpfade mithilfe der Option Suchpfad einschließen (/i) angegeben werden.
