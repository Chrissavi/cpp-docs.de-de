# <a name="expression-evaluator-error-cxx0005"></a>Ausdrucksauswertungsfehler CXX0005
Operator nicht unterstützt.  
  
 Ein nicht unterstützter C-Operator wurde in einem Ausdruck angegeben. Schreiben Sie einen entsprechenden Ausdruck mit den unterstützten C-Operatoren.  
  
 Dieser Fehler ist mit CAN0005 identisch.
