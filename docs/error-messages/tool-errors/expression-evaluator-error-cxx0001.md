# <a name="expression-evaluator-error-cxx0001"></a>Ausdrucksauswertungsfehler CXX0001

Ausnahme ausgeführten User-Funktion

Der ausgeführte Code verursacht eine allgemeine schutzverletzung. Dieser Fehler ist mit CAN0001 identisch.
