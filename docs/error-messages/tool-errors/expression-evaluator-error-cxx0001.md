# <a name="expression-evaluator-error-cxx0001"></a>Ausdrucksauswertungsfehler CXX0001

Ausnahme beim Ausführen der Benutzerfunktion

Der Code, der ausgeführt wird, hat einen allgemeinen Schutz Fehler verursacht. Dieser Fehler ist mit CAN0001 identisch.
