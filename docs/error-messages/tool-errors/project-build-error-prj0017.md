# <a name="project-build-error-prj0017"></a>Projektbuildfehler PRJ0017

Das aktuelle Arbeitsverzeichnis ist ungültig.

Der Pfad zum aktuellen Arbeitsverzeichnis ist länger als _MAXPATH. Um diesen Fehler zu beheben, erstellen Sie das Projekt nicht auf einer solchen tiefgreifenden Ebene.
