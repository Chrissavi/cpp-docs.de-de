# <a name="linker-tools-warning-lnk4039"></a>Linkertoolwarnung LNK4039

der mit der Option/section angegebene Abschnitt ' Name ' ist nicht vorhanden.

[DUMPBIN](../../build/reference/dumpbin-reference.md) oder [EDITBIN](../../build/reference/editbin-reference.md) konnte einen Abschnitt mit dem Namen `name` in der Eingabedatei nicht finden.

Wenn Sie die Abschnitte in einer Datei anzeigen möchten, führen Sie DUMPBIN mit der [/Headers](../../build/reference/headers.md) -Option aus.
