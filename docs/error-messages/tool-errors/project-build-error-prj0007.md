# <a name="project-build-error-prj0007"></a>Projektbuildfehler PRJ0007

Das Ausgabeverzeichnis "Verzeichnis" konnte nicht erstellt werden.

Visual C++ konnte kein Ausgabeverzeichnis erstellen. Zu den möglichen Gründen gehören:

- Das Verzeichnis, in dem das Ausgabeverzeichnis erstellt werden soll, ist schreibgeschützt.

- Nicht genügend Speicherplatz auf dem Computer.

- Sie haben einen ungültigen Verzeichnispfad angegeben.

- Benutzerberechtigungen reichen nicht aus, um das Verzeichnis zu erstellen.
