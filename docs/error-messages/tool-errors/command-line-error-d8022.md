# <a name="command-line-error-d8022"></a>Befehlszeilenfehler D8022

'Nachrichtendatei' kann nicht geöffnet werden.

Die angegebene Datei wurde nicht im aktuellen Verzeichnis oder in einem in der PATH-Umgebungsvariablen angegebenen Verzeichnis. Die eingabeinstanznachrichten-Datei enthält eine kurze Zusammenfassung der Optionen und Syntax für die Compilerbefehlszeile.

Verschieben Sie diese Datei, um das aktuelle Verzeichnis oder ein Verzeichnis im aktuellen Pfad. Wenn diese Datei kann nicht gefunden werden, führen Sie das Setupprogramm von den Datenträgern Verteilung kopieren.
