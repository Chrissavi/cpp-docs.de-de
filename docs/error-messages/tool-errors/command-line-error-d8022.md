# <a name="command-line-error-d8022"></a>Befehlszeilenfehler D8022

"messageFile" kann nicht geöffnet werden.

Die angegebene Datei befand sich nicht im aktuellen Verzeichnis oder in einem Verzeichnis, das in der PATH-Umgebungsvariablen angegeben ist. Die Nachrichtendatei enthält eine kurze Zusammenfassung der compilerbefehlszeilensyntax und-Optionen.

Verschieben Sie diese Datei in das aktuelle Verzeichnis oder in ein Verzeichnis im aktuellen Pfad. Wenn diese Datei nicht gefunden werden kann, führen Sie das Setup Programm aus, um Sie von den Verteilungs Datenträgern zu kopieren.
