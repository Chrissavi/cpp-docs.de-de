# <a name="linker-tools-error-lnk1248"></a>Linkertoolfehler LNK1248

> die Bildgröße ("*size*") überschreitet die maximal zulässige Größe (80 Millionen).

Der Linker hat festgestellt, dass die Größe der Ausgabedatei die größtmögliche Größe für ein 32-Bit-Programm Abbild überschreitet. Möglicherweise möchten Sie das Programm in mehrere DLLs Unternehmen.
