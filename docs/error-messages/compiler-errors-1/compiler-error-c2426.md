# <a name="compiler-error-c2426"></a>Compilerfehler C2426

"Token": Unzulässiger Operator in "Context"

Das Token kann in diesem Kontext nicht als Operator verwendet werden. Index Operatoren können z. b. nicht eingebettet werden.
