# <a name="compiler-error-c2112"></a>Compilerfehler C2112

"-": die Zeiger Subtraktion erfordert einen ganzzahligen oder Zeiger Operanden.

Es wurde versucht, Zeiger, die auf verschiedene Typen verweisen, zu subtrahieren.
