# <a name="compiler-error-c2412"></a>Compilerfehler C2412

'Bezeichnung': Groß-/Kleinschreibung Neudefinition einer Bezeichnung

Die Bezeichnung ist mehr als einmal in der aktuellen Funktion definiert. Ändern Sie die Schreibweise der Bezeichnung und seinen verweisen.
