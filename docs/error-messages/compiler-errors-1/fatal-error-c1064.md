# <a name="fatal-error-c1064"></a>Schwerwiegender Fehler C1064

Compilerlimit: Überlauf des internen Puffers durch Token

Ein Bezeichner überschreitet die Länge des internen Puffers für Bezeichner verwendet. Kürzen Sie den Namen ein.
