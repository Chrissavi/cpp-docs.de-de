# <a name="fatal-error-c1309"></a>Schwerwiegender Fehler C1309
Nicht übereinstimmende Versionen von C2. DLL und PGODB\<Version >. DLL  
  
 Das Toolset, das Sie verwenden, um das Erstellen und Verwenden von [Profilgesteuerte Optimierung](../../build/reference/profile-guided-optimizations.md) enthält nicht übereinstimmende Komponenten.  Wenn Sie diesen Fehler nicht manuell beheben können, installieren Sie Visual C++ neu.
