# <a name="fatal-error-c1009"></a>Schwerwiegender Fehler C1009

Compilerlimit: zu tiefe Schachtelung von Makros

Der Compiler hat versucht, zu viele Makros gleichzeitig zu erweitern. Der Compiler hat ein Limit von 256 Ebenen von unterstrichen. Aufteilen Sie die gemusteten Makros in einfachere Makros.
