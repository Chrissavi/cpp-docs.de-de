# <a name="compiler-error-c2471"></a>Compilerfehler C2471

Programmdatenbank 'Datei' kann nicht aktualisiert werden

Der Compiler hat keinen Schreibzugriff auf die Datenbankdatei.

### <a name="to-fix-the-problem"></a>So beheben Sie dieses Problem

- Stellen Sie sicher, dass auf dem Datenträger genug Speicherplatz verfügbar ist.

- Stellen Sie sicher, dass die Datei nicht schreibgeschützt ist.

- Stellen Sie sicher, dass es keine Freigabeverletzungen gibt.
