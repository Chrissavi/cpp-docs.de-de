# <a name="fatal-error-c1055"></a>Schwerwiegender Fehler C1055

Compilerlimit: keine weiteren Schlüssel

Die Quelldatei enthält zu viele Symbole. Der Compiler hat aus den Hash Schlüsseln für die Symboltabelle ausgegangen.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>So beheben Sie den Fehler (unterschiedliche Lösungsmöglichkeiten)

1. Teilen Sie die Quelldatei in kleinere Dateien auf.

1. Vermeiden Sie unnötige Header Dateien.

1. Verwenden Sie temporäre und globale Variablen, anstatt neue zu erstellen.
