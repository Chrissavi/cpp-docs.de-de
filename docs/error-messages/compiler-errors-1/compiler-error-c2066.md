# <a name="compiler-error-c2066"></a>Compilerfehler C2066

Typumwandlung in Funktionstyp ist nicht zulässig

In ANSI C ist es nicht zulässig, die zwischen einem Funktionszeiger und einen Datenzeiger umgewandelt.
