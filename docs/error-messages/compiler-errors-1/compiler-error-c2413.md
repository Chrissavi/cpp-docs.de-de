# <a name="compiler-error-c2413"></a>Compilerfehler C2413

"token": Unzulässige Ausrichtungsgröße

Die Größe der Verwendung der `ALIGN` -Direktive nicht vorhanden ist oder außerhalb des gültigen Bereichs.
