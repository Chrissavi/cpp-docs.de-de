# <a name="compiler-error-c2413"></a>Compilerfehler C2413

"Token": unzulässige Ausrichtungs Größe

Die mit der `ALIGN`-Direktive verwendete Größe fehlt oder liegt außerhalb des gültigen Bereichs.
