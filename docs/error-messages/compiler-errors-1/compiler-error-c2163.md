# <a name="compiler-error-c2163"></a>Compilerfehler C2163

"Function": nicht als intrinsische Funktion verfügbar

Ein `intrinsic`-oder `function`-Pragma listet eine Funktion auf, die nicht in System interner Form verfügbar ist. Beispielsweise sind bestimmte intrinsie nicht verfügbar, wenn ein Programm kompiliert wird, das/CLR-Programmierung verwendet.
