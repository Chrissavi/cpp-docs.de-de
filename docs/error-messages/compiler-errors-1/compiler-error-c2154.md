# <a name="compiler-error-c2154"></a>Compilerfehler C2154
'Typ' : Als Argument für das systeminterne Typmerkmal '__underlying_type' des Compilers ist nur ein Enumerationstyp zulässig.  
  
 Sie können nur den zugrunde liegenden Typ eines Enumerationstyps abrufen.  
  
 Weitere Informationen finden Sie unter [Compilerunterstützung für Typmerkmale](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md).
