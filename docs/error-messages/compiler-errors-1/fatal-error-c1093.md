# <a name="fatal-error-c1093"></a>Schwerwiegender Fehler C1093

> Der API-Aufruf "*Funktionsname*" konnte nicht durch den*Aufruf*":"*aus der Laufzeit*"" aufgerufen werden.

Fehler beim Abrufen einer .net-Funktion. Der *Text aus der Lauf Zeit* Zeichenfolge kann von der com-Laufzeit bereitgestellt werden.

Weitere Informationen zu System Fehlermeldungen finden Sie in der Systemdatei "Winerror. h" und in " [FormatMessage](/windows/win32/api/winbase/nf-winbase-formatmessage)".
