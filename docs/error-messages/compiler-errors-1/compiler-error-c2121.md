# <a name="compiler-error-c2121"></a>Compilerfehler C2121

"#": Ungültiges Zeichen: möglicherweise das Ergebnis einer Makro Erweiterung

Möglicherweise wurde ein ungültiges #-Zeichen von einem falschen Makro eingefügt, das den tokeneinfügenden Operator (# #) anstelle des Zeichen folgen Operators (#) verwendet.
