# <a name="compiler-error-c2121"></a>Compilerfehler C2121

'#': Ungültiges Zeichen: möglicherweise das Ergebnis einer makroerweiterung

Möglicherweise wurde ein ungültiges #-Zeichen durch ein falsches Makro, das die tokeneinfügenden Operator (##) anstelle des Zeichenfolgenoperators verwendet eingefügt (#).
