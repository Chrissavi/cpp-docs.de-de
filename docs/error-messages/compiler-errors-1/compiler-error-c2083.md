# <a name="compiler-error-c2083"></a>Compilerfehler C2083

struct/union-Vergleich ungültig

Eine Struktur oder Union wird direkt mit einem anderen benutzerdefinierten Typ verglichen. Dies ist nur zulässig, wenn ein Vergleichsoperator definiert wurde oder eine Konvertierung in einen skalaren Typ vorhanden ist.
