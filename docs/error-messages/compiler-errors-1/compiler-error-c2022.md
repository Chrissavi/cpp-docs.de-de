# <a name="compiler-error-c2022"></a>Compilerfehler C2022

"Zahl": zu groß für ein Zeichen

Die oktale Zahl, die einem umgekehrten Schrägstrich (\\) in einer Zeichen-oder Zeichen folgen Konstante folgt, ist zu groß, um ein Zeichen darzustellen.
