# <a name="fatal-error-c1201"></a>Schwerwiegender Fehler C1201

Fortfahren nach einem Syntaxfehler in der Klassenvorlagendefinition nicht möglich.

Beim Analysieren der Definition einer Klassenvorlage ist ein unerwarteter Fehler aufgetreten.

Beheben Sie alle anderen Fehler, und kompilieren Sie den Code neu. Wenn dies nicht erfolgreich war, notieren Sie die Umstände des Fehlers, versuchen Sie, das Problem zu isolieren, und erstellen Sie einen reproduzierbaren Testfall. Wenden Sie sich erst dann an den [Produktsupport von Microsoft](/visualstudio/ide/talk-to-us).
