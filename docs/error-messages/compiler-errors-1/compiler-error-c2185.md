# <a name="compiler-error-c2185"></a>Compilerfehler C2185

"Bezeichner": Ungültige basierte Zuordnung

Eine Register Variable oder eine automatische (lokale) Variable wird deklariert **`__based`** . Es können nur globale Variablen deklariert werden **`__based`** .
