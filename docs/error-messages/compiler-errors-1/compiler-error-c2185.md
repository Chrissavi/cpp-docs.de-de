# <a name="compiler-error-c2185"></a>Compilerfehler C2185

"Bezeichner": Ungültige basierte Zuordnung

Eine Register Variable oder eine automatische (lokale) Variable wird als `__based`deklariert. Nur globale Variablen können als `__based`deklariert werden.
