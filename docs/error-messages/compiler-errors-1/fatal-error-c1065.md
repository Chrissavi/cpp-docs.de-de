# <a name="fatal-error-c1065"></a>Schwerwiegender Fehler C1065

Compilerlimit: nicht genügend Tags

Die Quelldatei enthält mehr als 65.523 Klassen, Strukturen, Unions, Namespaces oder enumerationszeichen.

Teilen Sie die Datei in kleinere Dateien auf.
