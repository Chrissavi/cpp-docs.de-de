# <a name="compiler-error-c2301"></a>Compilerfehler C2301

Der linke Teil von "->~Identifizierer" muss auf eine Klasse/Struktur/Union zeigen.

Der Ausdruck links des `->` -Operators kann keinen Zeiger auf eine Klasse, Struktur oder Union auswerten.
