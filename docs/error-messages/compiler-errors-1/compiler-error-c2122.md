# <a name="compiler-error-c2122"></a>Compilerfehler C2122
'Bezeichner': Prototypparameter in der Liste ist ungültig  
  
 Der Parameter ist keinem gültigen Typ. ANSI C unterstützt keine benutzerdefinierten Typen.
