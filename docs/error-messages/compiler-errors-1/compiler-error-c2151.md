# <a name="compiler-error-c2151"></a>Compilerfehler C2151

mehr als ein Sprachattribut

Eine Funktion verfügt über mehr als ein Schlüsselwort ( `__cdecl`, `__stdcall`, oder `__fastcall`) eine Aufrufkonvention angibt.
