# <a name="compiler-error-c2174"></a>Compilerfehler C2174

"Function": der tatsächliche Parameter hat den Typ "void": Parameter Zahl1, Parameterliste Zahl2

Parameter `number1` an die Parameterliste übergeben `number2` ist ein `void` Parameter. Parameter dürfen nicht den Typ "`void`" aufweisen. Verwenden Sie stattdessen `void*`.
