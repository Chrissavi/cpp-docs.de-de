# <a name="compiler-error-c2113"></a>Compilerfehler C2113

'-': Zeiger kann nur von einem anderen Zeiger subtrahiert werden.

Der rechte Operand einer Subtraktion war ein Zeiger, der linke Operand jedoch nicht.
