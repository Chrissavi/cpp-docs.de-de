# <a name="compiler-error-c2446"></a>Compilerfehler C2446

"Operator": keine Konvertierung von "Typ1" in "Typ2"

Der Compiler kann `type1` nicht in `type2`konvertieren. Die Konvertierung ist möglicherweise nicht sinnvoll, da Sie gegenC++ C/Semantik verstößt.
