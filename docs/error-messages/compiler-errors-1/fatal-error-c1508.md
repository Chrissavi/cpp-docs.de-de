# <a name="fatal-error-c1508"></a>Schwerwiegender Fehler C1508

Compilerlimit: "Funktion": Mehr als 65535 Bytes für Argumente.

Die formalen Parameter der Funktion überschreiten den Grenzwert von 65535 Bytes.
