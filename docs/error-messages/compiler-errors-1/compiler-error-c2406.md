# <a name="compiler-error-c2406"></a>Compilerfehler C2406

'identifier': Name 'context' undefiniert

Ein undefinierter Bezeichner wird verwendet, mit der `SIZE`, `LENGTH`, oder dem Memberauswahloperator (.).
