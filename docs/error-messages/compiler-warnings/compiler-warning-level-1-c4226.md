# <a name="compiler-warning-level-1-c4226"></a>Compilerwarnung (Stufe 1) C4226

> nicht dem Standard entsprechende Erweiterung: "*Schlüsselwort*" ist ein veraltetes Schlüsselwort

Die aktuelle Version von Visual C++ wird dieses Schlüsselwort nicht verwendet.

Diese Warnung wird automatisch zu einem Fehler heraufgestuft.
