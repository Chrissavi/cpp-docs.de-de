# <a name="compiler-warning-level-1-c4226"></a>Compilerwarnung (Stufe 1) C4226

> nicht dem Standard entsprechende Erweiterung verwendet: '*Schlüsselwort*' ist ein veraltetes Schlüsselwort

In der aktuellen Version von C++ Visual wird dieses Schlüsselwort nicht verwendet.

Diese Warnung wird automatisch zu einem Fehler heraufgestuft.
