Notieren Sie die Umstände des Fehlers, versuchen Sie, das Problem zu isolieren, und erstellen Sie einen reproduzierbaren Testfall. Wenden Sie sich erst dann an den [Produktsupport von Microsoft](/visualstudio/ide/talk-to-us).
