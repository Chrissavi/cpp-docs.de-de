# <a name="compiler-warning-level-3-c4511"></a>Compilerwarnung (Stufe 3) C4511

'Klasse': Kopierkonstruktor konnte nicht generiert werden

Der Compiler konnte einen standardmäßigen Kopierkonstruktor für eine Klasse nicht generiert werden; eine Basisklasse möglicherweise einen privaten Kopierkonstruktor.
