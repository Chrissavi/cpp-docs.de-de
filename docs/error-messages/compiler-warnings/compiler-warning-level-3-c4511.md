# <a name="compiler-warning-level-3-c4511"></a>Compilerwarnung (Stufe 3) C4511

"Klasse": der Kopierkonstruktor konnte nicht generiert werden.

Der Compiler konnte keinen Standardkopierkonstruktor für eine Klasse generieren. eine Basisklasse kann über einen privaten Kopierkonstruktor verfügen.
