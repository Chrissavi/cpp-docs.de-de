# <a name="compiler-warning-level-1-c4711"></a>Compilerwarnung (Stufe 1) C4711

Funktion "Function" für Inline Erweiterung ausgewählt

Der Compiler hat Inlining für die angegebene Funktion ausgeführt, obwohl er nicht für das Inlining markiert war.

C4711 ist aktiviert, wenn [/Ob2](../../build/reference/ob-inline-function-expansion.md) angegeben wird.

Inlining wird nach dem Ermessen des Compilers ausgeführt. Diese Warnung dient nur zu Informationszwecken.

Diese Warnung ist standardmäßig deaktiviert. Verwenden Sie [#pragma Warnung](../../preprocessor/warning.md), um eine Warnung zu aktivieren. Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
