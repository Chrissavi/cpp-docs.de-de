# <a name="compiler-warning-level-1-c4264"></a>Compilerwarnung (Stufe 1) C4264

' Virtual_function ': für die Funktion eines virtuellen Members von der Basisklasse ' ' ist keine Überschreibung verfügbar. die Funktion ist ausgeblendet.

C4264 wird immer nach [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md)generiert.

Diese Warnung ist standardmäßig deaktiviert. Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
