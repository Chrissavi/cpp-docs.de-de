# <a name="compiler-warning-level-3-c4013"></a>Compilerwarnung (Stufe 3) C4013

"Function" ist nicht definiert. angenommen, extern gibt int zurück

Der Compiler hat einen-Rückruf für eine nicht definierte Funktion gefunden.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Dieser Fehler kann eine der folgenden Ursachen haben:

1. Falsche Schreibweise des Funktionsnamens.

1. Externe Funktionen, nicht als prototyptypisiert**`extern`**
