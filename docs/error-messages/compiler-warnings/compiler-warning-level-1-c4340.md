# <a name="compiler-warning-level-1-c4340"></a>Compilerwarnung (Stufe 1) C4340

"Value": Umschließen von positivem in negativen Wert des Werts

Die `enum` Wert ist größer als die größte `enum` positiver Wert, der auf einen negativen Wert umschließt.
