# <a name="compiler-warning-level-1-c4402"></a>Compilerwarnung (Stufe 1) C4402
PTR-Operator muss verwendet werden.  
  
 Ein Typ ist bei einem Operanden ohne Operator PTR beim Verweisen auf oder auf einen Typ in Inline-Assemblycode Umwandlung verwendet.  
  
 Der Compiler davon aus, der PTR-Operator.
