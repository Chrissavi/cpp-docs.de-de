# <a name="compiler-warning-level-1-c4168"></a>Compilerwarnung (Stufe 1) C4168

Compilerlimit: nicht genügend Debuggertypen; Programmdatenbank 'datenbank' muss gelöscht und neu erstellt werden

Die Programmdatenbankdatei muss neu erstellt werden, um alle Typen im Programm aufzunehmen.
