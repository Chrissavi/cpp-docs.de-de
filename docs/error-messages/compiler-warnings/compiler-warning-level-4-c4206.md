# <a name="compiler-warning-level-4-c4206"></a>Compilerwarnung (Stufe 4) C4206

**nicht dem Standard entsprechende Erweiterung: Übersetzungseinheit ist leer**

Die Datei war nach der Vorverarbeitung leer.

Mit dieser Erweiterung kann verhindert werden, dass Ihr Code auf andere Compiler übertragbar ist. Er generiert einen Fehler unter ANSI-Kompatibilität ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) und gilt nur für C-Quellcode.
