# <a name="compiler-warning-level-3-c4543"></a>Compilerwarnung (Stufe 3) C4543

Eingefügter Text, der vom Attribut "no_injected_text

Das [no_injected_text](../../windows/no-injected-text.md) -Attribut ist im Quellcode aufgetreten, was bedeutet, dass der Compiler die Eingabe von Code durch Attribute verhindert.

Diese Warnung soll Sie daran erinnern, dass Attribute keinen Code einfügen können.
