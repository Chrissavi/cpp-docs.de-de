# <a name="compiler-warning-level-4-c4053"></a>Compilerwarnung (Stufe 4) C4053

Ein void-Operand für "?:"

Der `?:` -Operator erhält einen Ausdruck vom Typ `void`. Der Wert des `void` -Operanden ist nicht definiert.
