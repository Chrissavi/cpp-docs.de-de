# <a name="compiler-warning-level-3-c4608"></a>Compilerwarnung (Stufe 3) C4608

"union_member" wurde bereits von einem anderen Union-Member in der Initialisierungsliste initialisiert, "union_member"

In einer Initialisierungsliste wurden zwei Member desselben union-Elements initialisiert. Sie k√∂nnen nur auf ein Member des union-Elements zugreifen.

Im folgenden Beispiel wird C4608 generiert:

```
// C4608.cpp
// compile with: /W3 /c
class X {
public:
   X(char c) : m_i( c + 1), m_c(c) {}   // C4608
   // try the following line instead
   // X(char c) : m_c(c) {}

private:
   union {
      int m_i;
      char m_c;
   };
};

union Y {
public:
   Y(char * name) : m_number(0.3), m_string( name ) {} // C4608

private:
   double m_number;
   char * m_string;
};
```
