# <a name="compiler-warning-level-1-c4410"></a>Compilerwarnung (Stufe 1) C4410

Ungültige Größe für Operand

Einer der Operanden in der Anweisung hatte eine falsche Größe. Die kleinste zulässige Größe für den Operanden wurde verwendet.
