# <a name="compiler-warning-level-4-c4296"></a>Compilerwarnung (Stufe 4) C4296

"Operator": der Ausdruck ist immer "false".

Eine nicht signierte Variable wurde in einem Vergleichs Vorgang mit 0 (null) verwendet.

Diese Warnung ist standardmäßig deaktiviert. Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .

Im folgenden Beispiel wird C4296 generiert:

```cpp
// C4296.cpp
// compile with: /W4
#pragma warning(default : 4296)
int main()
{
   unsigned int u = 9;
   if (u < 0)    // C4296
      u++;
   if (u >= 0)   // C4296
      u++;
}
```
