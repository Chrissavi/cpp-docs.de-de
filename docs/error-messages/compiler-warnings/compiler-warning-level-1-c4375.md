# <a name="compiler-warning-level-1-c4375"></a>Compilerwarnung (Stufe 1) C4375

die nicht öffentliche Methode ' Methode2 ' überschreibt ' Methode1 ' nicht.

Ein Typ, der einen anderen Typ implementiert, hat eine Überschreibungs Methode definiert, aber die Überschreibung war nicht öffentlich. Daher überschreibt die-Methode nicht die-Basistyp Methode.
