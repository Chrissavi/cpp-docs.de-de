# <a name="compiler-warning-level-1-c4375"></a>Compilerwarnung (Stufe 1) C4375

nicht öffentliche Methode "Methode2" wird "Methode1" nicht überschrieben.

Ein Typ, der einen anderen Typ implementiert definiert eine Methode außer Kraft setzen, aber die Außerkraftsetzung war nicht öffentlich. Aus diesem Grund überschreibt die Methode nicht die Basistyp-Methode.
