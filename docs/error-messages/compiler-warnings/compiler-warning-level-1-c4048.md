# <a name="compiler-warning-level-1-c4048"></a>Compilerwarnung (Stufe 1) C4048

andere deklarierte Array Indizes: "Bezeichner1" und "Bezeichner2"

Ein Ausdruck beinhaltet Zeiger auf Arrays mit unterschiedlicher Größe. Die Zeiger werden ohne Konvertierung verwendet.

Diese Warnung wird möglicherweise korrigiert, wenn Sie die Arrays explizit in denselben oder einen entsprechenden Typ umwandeln.
