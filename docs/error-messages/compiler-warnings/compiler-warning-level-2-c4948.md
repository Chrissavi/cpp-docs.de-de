# <a name="compiler-warning-level-2-c4948"></a>Compilerwarnung (Stufe 2) C4948

Rückgabetyp von 'Accessor' stimmt nicht mit dem letzten Parametertyp des entsprechenden Setters überein.

Der Compiler hat einen Konflikt zwischen bezüglich des Datentyps ist, wird zu erhalten, und legen Sie für eine indizierte Eigenschaft gefunden.

C4948 ist nur über die veraltete Compileroption erreichbar **oldSyntax**.
