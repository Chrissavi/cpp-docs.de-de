# <a name="compiler-warning-level-2-c4007"></a>Compilerwarnung (Stufe 2) C4007

"Function": muss "Attribute" lauten.

Ein erforderliches Attribut für eine Funktion wird nicht explizit angegeben. Die **Main** -Funktion muss beispielsweise über das `__cdecl`-Attribut verfügen. Der Compiler erzwingt das-Attribut.
