# <a name="compiler-warning-level-4-c4536"></a>Compilerwarnung (Stufe 4) C4536

' Typname ': der Typname überschreitet das Metadatenlimit von ' Limit '-Zeichen.

Ein Typname wird in den Metadaten abgeschnitten, wenn es sich um einen verwalteten Typ handelt. Weitere Informationen finden Sie unter [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md) .

Diese Warnung ist standardmäßig deaktiviert. Weitere Informationen finden Sie unter [Standardmäßig deaktivierte Compilerwarnungen](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
