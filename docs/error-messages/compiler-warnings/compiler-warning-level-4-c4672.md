# <a name="compiler-warning-level-4-c4672"></a>Compilerwarnung (Stufe 4) C4672

'identifier1': Mehrdeutig. Zuerst als 'identifier2' aufgetreten

Das angegebene Objekt, das in einem-Block ausgelöst werden soll, **`try`** ist mehrdeutig. Das Objekt kann nicht eindeutig gemacht werden, wenn es ausgelöst wird.
