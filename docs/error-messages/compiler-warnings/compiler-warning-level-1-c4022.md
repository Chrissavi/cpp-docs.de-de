# <a name="compiler-warning-level-1-c4022"></a>Compilerwarnung (Stufe 1) C4022

' function ': Zeiger Stimmen für den tatsächlichen Parameter ' Number ' nicht überein.

Der Zeigertyp des tatsächlichen Parameters unterscheidet sich vom Zeigertyp des entsprechenden formalen Parameters. Der tatsächliche Parameter wird unverändert übergeben.
