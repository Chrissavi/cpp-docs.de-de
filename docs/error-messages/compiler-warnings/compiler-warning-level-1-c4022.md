# <a name="compiler-warning-level-1-c4022"></a>Compilerwarnung (Stufe 1) C4022

'Funktion': Zeiger nicht übereinstimmende Versionen für tatsächliche Parameter 'Nummer'

Der Zeigertyp des tatsächlichen Parameters unterscheidet sich von der Zeigertyp des entsprechenden formalen Parameters. Der tatsächliche Parameter wird unverändert übergeben.
