# <a name="compiler-warning-level-1-c4502"></a>Compilerwarnung (Stufe 1) C4502

"Bindungsangabe" erfordert die Verwendung des Schlüsselworts "Extern" und muss vor allen anderen Spezifizierern stehen.

Eine Verknüpfung angegeben wurde, ohne die `extern` Schlüsselwort. Verknüpfung ist nicht für nicht externe Typen relevant.

Der Compiler davon ausgegangen, dass die `extern` Schlüsselwort.
