# <a name="compiler-warning-level-1-c4379"></a>Compilerwarnung (Stufe 1) C4379
Version der common Language Runtime Version wird von diesem Compiler nicht unterstützt. Mit dieser Version kann zu unerwarteten Ergebnissen führen.  
  
 Sie haben eine frühere Version der common Language Runtime auf dem Computer, jedoch nicht die aktuelle Version. Um C4379 zu beheben, installieren Sie die Version der common Language Runtime, die geliefert wurden, mit dem Compiler.
