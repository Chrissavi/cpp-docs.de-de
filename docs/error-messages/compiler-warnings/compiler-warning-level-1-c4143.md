# <a name="compiler-warning-level-1-c4143"></a>Compilerwarnung (Stufe 1) C4143

Pragma "same_seg" wird nicht unterstützt; reservieren Sie Segmente mit __based

Das Pragma **#pragma same_seg** wird nicht mehr unterstützt. Verwenden Sie stattdessen das [__based](../../cpp/based-pointers-cpp.md) -Schlüsselwort.
