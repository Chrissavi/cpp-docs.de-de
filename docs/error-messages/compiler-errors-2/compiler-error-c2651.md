# <a name="compiler-error-c2651"></a>Compilerfehler C2651

'Data type': links von 'Operator' muss eine Klasse, Struktur oder Union sein

Um eine Template-Parameter verwenden, als ob es sich um eine Klasse ist, sind Sie die Klassenvorlage mit einer Klasse anstelle eines ganzzahligen Typs.
