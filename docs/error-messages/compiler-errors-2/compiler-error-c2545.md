# <a name="compiler-error-c2545"></a>Compilerfehler C2545

"Operator": überladener Operator konnte nicht gefunden werden.

Der Operator kann nicht mit den angegebenen Operanden verwendet werden. Sie müssen einen überladenen Operator mit den erforderlichen Operanden angeben.

Dieser Fehler kann verursacht werden, wenn Operanden einen falschen Typ aufweisen.

Dieser Fehler kann korrigiert werden, wenn Sie einen Konvertierungs Operator oder Konstruktor definieren, der einen einzelnen Parameter annimmt.
