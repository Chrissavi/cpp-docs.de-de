# <a name="compiler-error-c2994"></a>Compilerfehler C2994

Unbenannte Klasse in der Vorlagen-Parameterliste

Sie können das `class` -Schlüsselwort nicht als Vorlagenargument verwenden, ohne einen Klassennamen anzugeben.
