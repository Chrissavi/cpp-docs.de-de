# <a name="compiler-error-c3501"></a>Compilerfehler C3501

Es ist keine Typbibliothek registriert, die für die ProgID "progid"

Die Klassen-ID für einen angegebenen progid verfügt nicht über eine zugeordnete Typbibliothek. Aus diesem Grund kann nicht übergeben dieser progid, die `#import` Anweisung.
