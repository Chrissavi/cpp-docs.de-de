# <a name="compiler-error-c3603"></a>Compilerfehler C3603

'Symbol': Typ "Type" noch nicht unterstützt

Sie versucht, einen Datentyp zu verwenden, der von der .NET-Laufzeit in ein verwaltetes Objekt noch nicht unterstützt wird.
