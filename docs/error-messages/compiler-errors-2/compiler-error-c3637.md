# <a name="compiler-error-c3637"></a>Compilerfehler C3637
'Funktion': die Definition einer Friend-Funktion darf keine Spezialisierung der kein Funktionstyp sein  
  
 Eine Friend-Funktion wurde nicht ordnungsgemäß für eine Vorlage oder die generische definiert.  
  
 Im folgenden Beispiel wird C3637 generiert:  
  
```  
// C3637.cpp  
template <class T>  
void f();  
  
struct S {  
   friend void f<int>() {}   // C3637  
};  
```  
  
 Mögliche Lösung:  
  
```  
// C3637b.cpp  
// compile with: /c  
template <class T>  
void f();  
  
struct S {  
   friend void f() {}  
};  
```  
  
 C3637 kann auch auftreten, wenn Generika verwendet werden:  
  
```  
// C3637c.cpp  
// compile with: /clr  
  
generic <class T>  
void gf();  
  
struct S {  
   friend void gf<int>() {}   // C3637  
};  
```  
  
 Mögliche Lösung:  
  
```  
// C3637d.cpp  
// compile with: /clr /c  
generic <class T>  
void gf();  
  
struct S {  
   friend void gf() {}  
};  
```
