# <a name="compiler-error-c2526"></a>Compilerfehler C2526

"Bezeichner1": die C-Bindungs Funktion C++ kann die Klasse "Bezeichner2" nicht zurückgeben.

Eine mit C-Verknüpfung definierte Funktion kann keinen benutzerdefinierten Typ zurückgeben.
