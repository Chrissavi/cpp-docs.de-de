# <a name="compiler-error-c2551"></a>Compilerfehler C2551

"void *"-Zeiger muss explizit in einen Typ umgewandelt werden

Ein **`void`** Zeiger wird durch implizite Konvertierung einem nicht-void-Zeiger zugewiesen. Sie müssen eine explizite Umwandlung durchführen.
