# <a name="compiler-error-c2654"></a>Compilerfehler C2654

' Identifier ': Versuch, auf Member außerhalb einer Member-Funktion zuzugreifen.

Auf einen Member wird in einer Deklaration zugegriffen. Auf Elementdaten kann nur in Element Funktionen zugegriffen werden.

Dieser Fehler kann verursacht werden, wenn Sie versuchen, Variablen in einer Deklaration zu initialisieren. Verwenden Sie für diesen Zweck einen Konstruktor.
