# <a name="compiler-error-c3334"></a>Compilerfehler C3334

#import für beschädigte Typbibliothek nicht möglich.

Die Bibliotheksdatei in einer `#import` -Anweisung ist beschädigt und kann vom Compiler nicht verwendet werden.  Die Beschädigung kann durch die fehlerhafte Erstellung des Moduls, einen fehlerhaften Datenträger oder durch Bearbeiten oder Ändern der Binärdatei erfolgen.

Weitere Informationen finden Sie unter [#import-Anweisung](../../preprocessor/hash-import-directive-cpp.md).
