# <a name="compiler-error-c3832"></a>Compilerfehler C3832

"Typbibliothek": Typbibliothek aussieht, als ob sie für 32-Bit-Zeiger; erstellt wurde Ändern Sie den Qualifizierer 'Ptrsize'

Explizite Informationen bereitgestellt, mit der `ptrsize` Attribut der [#import](../../preprocessor/hash-import-directive-cpp.md) Richtlinie stimmen nicht mit dem überein, was der Compiler in der Typbibliothek gefunden hat.
