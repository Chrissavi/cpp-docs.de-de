# <a name="compiler-error-c3832"></a>Compilerfehler C3832

"Typbibliothek": die Typbibliothek sieht so aus, als ob Sie für 32-Bit-Zeiger erstellt wurde. Ändern Sie den "ptrsize"-Qualifizierer.

Explizite Informationen, die mit dem `ptrsize`-Attribut der [#Import](../../preprocessor/hash-import-directive-cpp.md) Direktive bereitgestellt wurden, stimmten nicht mit dem, was der Compiler in der Typbibliothek gefunden hat.
