# <a name="compiler-error-c3135"></a>Compilerfehler C3135

"Property": eine Eigenschaft kann keinen Typ "Konst" oder "volatile" aufweisen.

Die Schlüsselwörter " [Konstanten](../../cpp/const-cpp.md) " und " [volatile](../../cpp/volatile-cpp.md) " sind für Eigenschaften nicht zulässig.
