# <a name="compiler-error-c2839"></a>Compilerfehler C2839

Ungültiger Rückgabetyp "Type" für überladenen Operator->.

Der `->`-Operator muss eine Klasse, eine Struktur oder eine Union oder einen Verweis auf einen zurückgeben.
