 Ein paralleler `omp` -Bereich darf keine Inlineassemblyanweisungen enthalten.  
  
 Im folgenden Beispiel wird C3011 generiert:  
  
```  
// C3011.cpp

// processor: /x86

   int   n = 0;  
  
   #pragma omp parallel
```
