# <a name="compiler-error-c3800"></a>Compilerfehler C3800

'Declaration': nicht gemischt werden, Eigenschaften und Ereignisse

Sie können ein Konstrukt einer Eigenschaft und ein Ereignis nicht deklarieren.

C3800 ist nur über die veraltete Compileroption erreichbar **oldSyntax**.
