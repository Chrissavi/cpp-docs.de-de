# <a name="compiler-error-c2773"></a>Compilerfehler C2773

\#Importieren und #using nur in C++-Compiler verfügbar

Der C-Compiler erkennt nicht die `#import` Präprozessordirektive angegeben. Kompilieren Sie die Quelle als C++. Verwendung [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) bei Bedarf.
