# <a name="compiler-error-c2773"></a>Compilerfehler C2773

\#Import und #using nur im C++ Compiler verfügbar.

Der C-Compiler erkennt die `#import` Präprozessordirektive nicht. Kompilieren Sie die Quelle C++als. Verwenden Sie [/tp](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) bei Bedarf.
