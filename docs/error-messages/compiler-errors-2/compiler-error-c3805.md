# <a name="compiler-error-c3805"></a>Compilerfehler C3805

"Token": Unerwartetes Token, erwartet wurde entweder "}" oder ein Bezeichner.

Beim Definieren einer Eigenschaft wurde ein ungültiges Token gefunden. Entfernen Sie das ungültige Token.
