# <a name="compiler-error-c3167"></a>Compilerfehler C3167

.NET Framework kann nicht initialisiert werden: Stellen Sie sicher, dass Sie installiert ist.

Der .NET Framework ist auf diesem Computer nicht installiert. Installieren Sie die .NET Framework.
