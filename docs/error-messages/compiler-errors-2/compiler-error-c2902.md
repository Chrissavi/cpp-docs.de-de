# <a name="compiler-error-c2902"></a>Compilerfehler C2902

"Token": Unerwartetes Token nach "Vorlage", Bezeichner erwartet.

Das auf das Schl√ºsselwort `template` folgende Token war kein Bezeichner.

Im folgenden Beispiel wird C2902 generiert:

```
// C2902.cpp
// compile with: /c
namespace N {
   template<class T> class X {};
   class Y {};
}
void g() {
   N::template + 1;   // C2902
}

void f() {
   N::template X<int> x1;   // OK
}
```

C2902 kann auch auftreten, wenn Generics verwendet werden:

```
// C2902b.cpp
// compile with: /clr /c
namespace N {
   generic<class T> ref class GC {};
}

void f() {
   N::generic + 1;   // C2902
   N::generic GC<int>^ x;
}
```
