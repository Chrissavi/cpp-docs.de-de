# <a name="compiler-error-c2532"></a>Compilerfehler C2532

"Bezeichner": Unzulässiger Modifizierer für Verweis

Der Verweis wurde geändert. Verweise können nicht geändert werden, um auf ein anderes Objekt zu verweisen. Verwenden Sie stattdessen einen Zeiger.
