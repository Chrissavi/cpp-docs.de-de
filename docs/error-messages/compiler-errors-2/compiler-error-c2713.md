# <a name="compiler-error-c2713"></a>Compilerfehler C2713

Es ist nur eine Art der Ausnahmebehandlung pro Funktion zulässig.

Die strukturierte Ausnahmebehandlung ( `__try` / **`__except`** ) und die C++-Ausnahmebehandlung () können nicht **`try`** / **`catch`** in derselben Funktion verwendet werden.
