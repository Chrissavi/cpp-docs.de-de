## <a name="example"></a>Beispiel

Im folgenden Beispiel wird C3282 generiert und gezeigt, wie Sie diesen Fehler beheben.

```

// compile with: /clr /c
generic <typename T> int x;   // C3282

ref struct GC0 {
   generic <typename T> int x;   // C3282
};

// OK
generic <typename T>
```
