# <a name="compiler-error-c2795"></a>Compilerfehler C2795

' Super:: function ' ist keine Member-Funktion.

Diese Fehlermeldung wird immer dann angezeigt, wenn Sie versuchen, [Super](../../cpp/super.md) zu verwenden, um auf einen anderen Member als eine Member-Funktion zuzugreifen.
