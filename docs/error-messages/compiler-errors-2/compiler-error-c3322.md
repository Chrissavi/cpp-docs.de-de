# <a name="compiler-error-c3322"></a>Compilerfehler C3322

„property“: Ist keine gültige Eigenschaft für das Attribut „attribute“.

Eine ungültige oder unbekannte Eigenschaft (bzw. ein ungültiger oder unbekannter Parameter) wurde an ein Attribut übergeben. Konsultieren Sie die Dokumentation für das Attribut.
