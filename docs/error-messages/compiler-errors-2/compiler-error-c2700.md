# <a name="compiler-error-c2700"></a>Compilerfehler C2700

"Identifizierer": throw nicht möglich (/W4 für weitere Informationen)

Throw für Objekt nicht möglich. Kompilieren Sie mit [/W4](../../build/reference/compiler-option-warning-level.md) , um weitere Diagnoseinformationen zu erhalten.
