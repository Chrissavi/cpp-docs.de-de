 Statische Memberfunktionen sollten mit externer Bindung deklariert werden.  
  
 Im folgende Beispiel wird C2724 generiert:  
  
```  
// C2724.cpp  
class C {  
   static void func();  
};  
  
static void C::func(){};   // C2724  
// try the following line instead  
// void C::func(){};  
```
