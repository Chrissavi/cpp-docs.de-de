# <a name="compiler-error-c3661"></a>Compilerfehler C3661

die explizite Überschreibungs Liste hat keine zu über schreibenden Methoden gefunden.

Ein explizites Überschreiben hat mindestens einen Typnamen angegeben.  Es gab jedoch keine Funktion mit der erforderlichen Signatur in den Typen, die mit der Signatur der über schreibenden Funktion übereinstimmen.  Wenn Sie versuchen, basierend auf dem Typnamen zu überschreiben, muss mindestens eine virtuelle Funktion in den angegebenen Typen vorhanden sein, die der Signatur der über schreibenden Funktion entsprechen.

Weitere Informationen finden Sie unter [explizite über](../../extensions/explicit-overrides-cpp-component-extensions.md)schreibungen.
