# <a name="compiler-error-c2958"></a>Compilerfehler C2958
für das linke Trennzeichen finden Sie unter "Location" wurde nicht richtig zugeordnet.  
  
 Ein Trennzeichen ist nicht ordnungsgemäß zugeordnet.  
  
 Dieser Fehler kann durch nicht übereinstimmende Klammer in einer Funktionsvorlagendeklaration verursacht werden.
