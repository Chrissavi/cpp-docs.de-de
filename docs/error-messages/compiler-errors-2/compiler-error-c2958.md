# <a name="compiler-error-c2958"></a>Compilerfehler C2958

das linke Trennzeichen, das an ' Location ' gefunden wurde, wurde nicht richtig abgeglichen.

Ein Trennzeichen ist nicht ordnungsgemäß zugeordnet.

Dieser Fehler kann durch eine nicht übereinstimmende Klammer in einer Funktions Vorlagen Deklaration verursacht werden.
