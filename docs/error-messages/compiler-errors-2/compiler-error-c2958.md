# <a name="compiler-error-c2958"></a>Compilerfehler C2958

das linke Trennzeichen finden Sie unter "Location" wurde nicht ordnungsgemäß zugeordnet.

Ein Trennzeichen ist nicht ordnungsgemäß zugeordnet.

Dieser Fehler kann durch nicht übereinstimmende Klammern in einer Funktionsvorlagendeklaration verursacht werden.
