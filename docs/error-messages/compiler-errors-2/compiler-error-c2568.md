# <a name="compiler-error-c2568"></a>Compilerfehler C2568

"Bezeichner1": die Funktions Überladung kann nicht aufgelöst werden.

Der Compiler kann nicht bestimmen, welche überladene Funktion aufgerufen werden soll. Die tatsächlich an die Funktion übergebenen Parameter müssen entsprechend den formalen Parametern für eine der überladenen Funktionen umgewandelt werden, aber es ist keine Entsprechung eindeutig besser als alle anderen.
