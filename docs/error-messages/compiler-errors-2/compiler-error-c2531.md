# <a name="compiler-error-c2531"></a>Compilerfehler C2531

'Bezeichner': Verweis auf Bitfeld ungültig

Verweise auf Bitfelder sind nicht zulässig.

Im folgende Beispiel wird die C2531 generiert:

```
// C2531.cpp
// compile with: /c
class P {
   int &b1 : 10;   // C2531
   int b2 : 10;   // OK
};
```
