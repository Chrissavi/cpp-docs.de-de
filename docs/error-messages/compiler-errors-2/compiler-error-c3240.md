# <a name="compiler-error-c3240"></a>Compilerfehler C3240

"Function": muss eine nicht Ã¼berladene abstrakte Member-Funktion von "Type" sein.

Ein Basistyp enthielt eine Funktion, die definiert wurde. Die Funktion muss virtuell sein.

## <a name="example"></a>Beispiel

Im folgenden Beispiel wird C3240 generiert.

```cpp
// C3240.cpp
// compile with: /c
__interface I {
   void f();
};

struct A1 : I {
   void f() {}
};

struct A2 : I {
   void f() = 0;
};

template <class T>
struct A3 : T {
   void T::f() {}
};

template <class T>
struct A4 : T {
   void T::f() {}
};

A3<A1> x;   // C3240
A3<I> x2;
A4<A2> x3;
```
