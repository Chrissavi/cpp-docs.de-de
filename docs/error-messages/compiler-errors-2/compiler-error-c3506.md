# <a name="compiler-error-c3506"></a>Compilerfehler C3506

Es ist kein Export der Typbibliothek für LIBID "ID" registriert.

Ein Export der Typbibliothek wurde nicht ordnungsgemäß registriert. Verwenden Sie regsvr32. exe, um eine Typbibliothek zu registrieren.
