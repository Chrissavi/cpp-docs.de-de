# <a name="active-template-library-atl-tutorial"></a>ATL-Lernprogramm (Active Template Library)

ATL ist so konzipiert, dass das Erstellen von effizienten, flexiblen und einfachen Steuerelementen vereinfacht wird. Dieses Tutorial führt Sie durch die Erstellung eines ActiveX-Steuer Elements, das viele Grundlagen zu ATL und com veranschaulicht.

In diesem Tutorial erfahren Sie, wie Sie ein-Steuerelement zu einem ATL-Projekt hinzufügen, das einen Kreis und ein ausgefülltes Polygon zeichnet. Anschließend fügen Sie eine Eigenschaft hinzu, um anzugeben, wie viele Seiten das Polygon enthalten wird, und erstellen Zeichnungs Code für die Aktualisierung des Steuer Elements, wenn sich die Eigenschaft ändert. Das-Steuerelement wird dann auf einer Webseite angezeigt, indem ein VBScript-Skript verwendet wird, damit es auf Ereignisse reagiert.

Das Tutorial ist in sieben Schritte unterteilt. Sie sollten jeden Schritt nacheinander ausführen, da spätere Schritte von bereits abgeschlossenen Aufgaben abhängen. Bevor Sie beginnen, sollten Sie sicherstellen, dass Sie über die erforderlichen Berechtigungen zum Registrieren einer ActiveX-Komponente auf dem jeweiligen Computer verfügen. Dies ist in der Regel nur dann von Bedeutung, wenn Sie Visual Studio .net über eine Terminal Dienste-Verbindung ausführen.
