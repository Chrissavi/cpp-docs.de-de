# <a name="error-information-classes"></a>Klassen für Fehlerinformationen

Die folgende Klasse gibt an, wie Fehlerinformationen behandelt werden:

- [Isupporterrorinfoimpl](../atl/reference/isupporterrorinfoimpl-class.md) Bestimmt, ob das Objekt die [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) -Schnittstelle unterstützt. `IErrorInfo`ermöglicht das Zurückgeben von Fehlerinformationen an den Client.
