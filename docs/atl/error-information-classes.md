# <a name="error-information-classes"></a>Fehlerinformationsklassen

Die folgende Klasse gibt an, wie Fehlerinformationen verarbeitet wird:

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) bestimmt, ob das Objekt unterstützt die [IErrorInfo](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-ierrorinfo) Schnittstelle. `IErrorInfo` ermöglicht die Fehlerinformationen zurück an den Client weitergegeben werden.
