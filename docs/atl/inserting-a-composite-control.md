# <a name="inserting-a-composite-control"></a>Einfügen eines zusammengesetzten Steuerelements

Die **Klasse hinzufügen** Dialogfeld können Sie eine ATL-Objekt in ein Projekt eingefügt werden soll. Zugriff auf dieses Dialogfeld können Sie per Rechtsklick auf den Projektnamen im Projektmappen-Explorer auf **hinzufügen**, und klicken Sie dann auf **Klasse hinzufügen**.

In der **Klasse hinzufügen** Dialogfeld wählen **ATL-Steuerelement**. Dies startet den [ATL-Steuerelement-Assistent](../atl/reference/atl-control-wizard.md). Wählen Sie zum Erstellen eines zusammengesetzten Steuerelements die **Optionen** Registerkarte, und klicken Sie auf die **zusammengesetztes Steuerelement** Kontrollkästchen.

Eine Standard-HTML-Seite wird zur Anzeige des Steuerelements erstellt.
