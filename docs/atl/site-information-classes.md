# <a name="site-information-classes"></a>Website-Informationsklassen
Die folgenden Klassen ermöglichen, ein Objekt mit dem Standort kommunizieren:  
  
-   [IObjectWithSiteImpl](../atl/reference/iobjectwithsiteimpl-class.md) Ruft ab und legt einen Zeiger auf ein Objekt Website fest. Verwendet für Objekte, die keine Steuerelemente sind.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) Ruft ab und legt einen Zeiger auf ein Objekt Website fest. Wird für Steuerelemente verwendet.
