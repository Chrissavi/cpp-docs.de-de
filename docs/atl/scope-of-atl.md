# <a name="scope-of-atl"></a>ATL-Bereich

ATL ermöglicht Ihnen das einfache Erstellen von COM-Objekte, Automatisierungsserver und ActiveX-Steuerelemente. ATL bietet integrierten Unterstützung für viele der grundlegenden COM-Schnittstellen.

ATL wird als Quellcode ausgeliefert, die Sie in Ihrer Anwendung enthalten. ATL stellt außerdem eine DLL zur Verfügung (atl90.dll), das Code enthält, die in Komponenten gemeinsam genutzt werden können. Diese DLL-Datei ist jedoch nicht erforderlich.
