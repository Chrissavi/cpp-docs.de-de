# <a name="modifying-the-atl-project"></a>Ändern des ATL-Projekts

An diesem Punkt wird Ihr zusammengesetztes Steuerelement-Projekt die erforderlichen Objekte für das zusammengesetzte Steuerelement implementiert. Im nächste Schritt werden alle Steuerelemente hinzufügen, dass das zusammengesetzte Steuerelement enthält, und alle erforderlichen Ereignisse verarbeiten.

Um zusätzliche ActiveX oder einem Windows-Steuerelemente hinzuzufügen, fügen Sie ein neues Ressourcenskript hinzu, und verwenden Sie dann den Dialog-Editor. Weitere Informationen zum Hinzufügen von Steuerelementen (und Verwandte Aufgaben), finden Sie unter [Dialog-Editor](../windows/dialog-editor.md).

Behandeln Sie alle erforderlichen Ereignisse vom ActiveX-Steuerelemente finden Sie unter [Hinzufügen neuer Funktionen zum zusammengesetzten Steuerelement](../atl/adding-functionality-to-the-composite-control.md).
