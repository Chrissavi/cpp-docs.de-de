# <a name="data-transfer-classes"></a>Datenübertragungsklassen
Die folgenden Klassen unterstützen verschiedene Arten von Datenübertragung:  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) unterstützt, Uniform Data Transfer zum Abrufen und Festlegen von Daten mithilfe von standard-Formate. Verarbeitet Daten änderungsbenachrichtigungen durch die Verwaltung von Verbindungen senken mitteilen.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) ein asynchrones Monikers zum Senden und Empfangen von Informationen über die asynchrone Datenübertragung in und aus Ihr Objekt ermöglicht.
