# <a name="class-factories-classes"></a>Klasse Factorys Klassen
Die folgenden Klassen implementieren oder eine Klassenfactory unterstützen:  
  
-   [CComClassFactory](../atl/reference/ccomclassfactory-class.md) bietet eine Standard-Klassenfactory für die objekterstellung.  
  
-   [CComClassFactory2](../atl/reference/ccomclassfactory2-class.md) steuert die objekterstellung über eine Lizenz.  
  
-   [CComClassFactoryAutoThread](../atl/reference/ccomclassfactoryautothread-class.md) ermöglicht es Objekten, die in mehreren Thread Pool Apartments erstellt werden.  
  
-   [CComClassFactorySingleton](../atl/reference/ccomclassfactorysingleton-class.md) erstellt ein einzelnes Objekt.  
  
-   [CComCoClass](../atl/reference/ccomcoclass-class.md) das Klassenfactory für das Objekt definiert.
