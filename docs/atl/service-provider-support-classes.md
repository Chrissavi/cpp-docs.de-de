# <a name="service-provider-support-classes"></a>Service Provider-Unterstützungsklassen

Die folgende Klasse bietet Unterstützung für Dienstanbieter:

- [IServiceProviderImpl](../atl/reference/iserviceproviderimpl-class.md) sucht einen angegebenen Dienst mithilfe der GUID und gibt Sie den Schnittstellenzeiger für die angeforderte Schnittstelle zurück, für den Dienst.
