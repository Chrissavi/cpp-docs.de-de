# <a name="creating-an-atl-project"></a>Erstellen eines ATL-Projekts

Die einfachste Möglichkeit zum Erstellen eines ATL-Projekts ist die Verwendung von ATL-Projektassistenten, im Ordner "Win32-Projekte" von der **im Dialogfeld Neues Projekt**.

### <a name="to-create-an-atl-project-using-the-atl-project-wizard"></a>Zum Erstellen eines ATL-Projekts mithilfe des ATL-Projekt-Assistenten

1. Befolgen Sie die Anweisungen im Thema [Erstellen eines Projekts mit einem Visual C++-Anwendungs-Assistenten](../../ide/creating-desktop-projects-by-using-application-wizards.md).

2. Wählen Sie die **ATL-Projekt** Symbol im Bereich "Vorlagen" auf den ATL-Projekt-Assistenten zu öffnen.

3. Definieren Sie Ihre Anwendungseinstellungen mit dem [Anwendungseinstellungen](../../atl/reference/application-settings-atl-project-wizard.md) auf der Seite die `ATL Project Wizard`.

   > [!NOTE]
   > Überspringen Sie diesen Schritt, um die Standardeinstellungen des Assistenten beizubehalten.

4. Klicken Sie auf **Fertig stellen** den Assistenten zu schließen und öffnen Sie das neue Projekt in der Entwicklungsumgebung.

Nachdem das Projekt erstellt wurde, sehen Sie die Dateien im **Projektmappen-Explorer**. Weitere Informationen zu den vom Assistenten erstellten Dateien im Projekt finden Sie in der projekteigenen Datei "Readme.txt". Weitere Informationen zu den Dateitypen finden Sie unter [für Visual C++-Projekte erstellte Dateitypen](../../ide/file-types-created-for-visual-cpp-projects.md). Weitere Informationen zu den Konfigurationen für das neue ATL-Projekt, und deren Änderung finden Sie unter [standardmäßige ATL-Projektkonfigurationen](../../atl/reference/default-atl-project-configurations.md).
