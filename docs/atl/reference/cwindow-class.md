# <a name="cwindow-class"></a>CWindow-Klasse

Diese Klasse stellt Methoden zum Bearbeiten eines Fensters bereit.

> [!IMPORTANT]
> Diese Klasse und ihre Member können in Anwendungen, die im Windows-Runtime ausgeführt werden, nicht verwendet werden.

## <a name="syntax"></a>Syntax

```
class CWindow
```

## <a name="remarks"></a>Bemerkungen

`CWindow`stellt die Basisfunktionen für die Bearbeitung eines Fensters in ATL bereit. Viele der Methoden wrappen `CWindow` einfach eine der Win32-API-Funktionen. Vergleichen Sie beispielsweise die Prototypen für `CWindow::ShowWindow` und `ShowWindow` :

|CWindow-Methode|Win32-Funktion|
|--------------------|--------------------|
|**Bool ShowWindow (int** `nCmdShow` **);**|**Bool ShowWindow (HWND** `hWnd` **, int** `nCmdShow` **);**|

`CWindow::ShowWindow`Ruft die Win32-Funktion `ShowWindow` auf, indem `CWindow::m_hWnd` als erster Parameter übergeben wird. Jede `CWindow` Methode, die eine Win32-Funktion direkt umschließt, übergibt den `m_hWnd` Member. daher `CWindow` werden Sie in einem Großteil der Dokumentation auf den Windows SDK verwiesen.

> [!NOTE]
> Nicht jede Fenster verwandte Win32-Funktion wird von umschlossen `CWindow` , und nicht jede Methode umschließt `CWindow` eine Win32-Funktion.

`CWindow::m_hWnd`speichert das HWND, das ein Fenster identifiziert. Ein HWND wird an Ihr Objekt angefügt, wenn Sie Folgendes ausführen:

- Geben Sie ein HWND im `CWindow` Konstruktor an.

- Rufen Sie `CWindow::Attach` auf.

- Verwenden Sie `CWindow` den **Operator =**.

- Erstellen oder unterteilen Sie ein Fenster mit einer der folgenden Klassen, die von abgeleitet sind `CWindow` :

[CWindowImpl](../../atl/reference/cwindowimpl-class.md) Ermöglicht das Erstellen eines neuen Fensters oder einer Unterklasse eines vorhandenen Fensters.

[CContainedWindow](../../atl/reference/ccontainedwindowt-class.md) Implementiert ein Fenster, das in einem anderen Objekt enthalten ist. Sie können ein neues Fenster oder eine Unterklasse für ein vorhandenes Fenster erstellen.

[CDialogImpl](../../atl/reference/cdialogimpl-class.md) Ermöglicht das Erstellen eines modalen oder nicht modalen Dialog Felds.

Weitere Informationen zu Windows finden Sie unter [Windows](/windows/win32/winmsg/windows) und nachfolgende Themen in der Windows SDK. Weitere Informationen zur Verwendung von Windows in ATL finden Sie im Artikel [ATL-Fenster Klassen](../../atl/atl-window-classes.md).
