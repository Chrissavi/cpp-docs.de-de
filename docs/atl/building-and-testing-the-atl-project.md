# <a name="building-and-testing-the-atl-project"></a>Erstellen und Testen des ATL-Projekts

Siehe [Einfügen eines zusammengesetzten Steuerelements](../atl/inserting-a-composite-control.md), eine der ersten Komponenten des Projekts wird eine Standard-HTML-Seite, die das neue zusammengesetzte Steuerelement hostet. Wenn Sie fertig sind, des zusammengesetzten Steuerelements zu ändern, klicken Sie auf **Projektmappe** oder **Projektmappe neu erstellen** aus der **erstellen** im Menü. Nachdem das Projekt erfolgreich erstellt wurde, laden Sie die HTML-Seite, befindet Sie sich im Stammverzeichnis des Projekts zusammengesetztes Steuerelement in Internet Explorer oder einen anderen Browser, und Testen Sie die Funktionalität des Steuerelements.

Sie können auch testen, Ihrem zusammengesetzte Steuerelement mit dem Testcontainer-Tool oder eine andere Anwendung, die ein ActiveX-Steuerelement hosten können. Informationen zum Zugriff auf den Testcontainer finden Sie unter [Testen von Eigenschaften und Ereignissen mit dem Testcontainer](../mfc/testing-properties-and-events-with-test-container.md) .
