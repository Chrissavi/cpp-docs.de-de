# <a name="schema-rowsets"></a>Schemarowsets

Schema (oder Metadaten) sind Informationen über die Struktur oder Organisation einer Datenbank, z. b. Informationen über den Anbieter, das Rowset, die Tabelle, die Spalten oder andere Informationen, die von den eigentlichen Datenbankinhalten abhängen. Mit OLE DB können Sie solche Informationen abrufen, ohne das Rowset mithilfe von Schemarowsets zu öffnen.

Weitere Informationen zum Verwenden von Schemarowsets finden Sie unter Abrufen von [Metadaten mit Schemarowsets](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
