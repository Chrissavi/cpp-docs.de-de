# <a name="cenumeratoraccessormszparsename"></a>CEnumeratorAccessor::m_szParseName
Zeichenfolge zu übergeben [IParseDisplayName](http://msdn.microsoft.com/library/windows/desktop/ms680604) zum Abrufen eines Monikers für die Datenquelle oder eines Enumerators.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
WCHAR m_szParseName[129];  
  
```  
  
## <a name="remarks"></a>Hinweise  
 Finden Sie unter [ISourcesRowset:: GetSourcesRowset](https://msdn.microsoft.com/en-us/library/ms711200.aspx) in der *OLE DB Programmer's Reference* für Weitere Informationen.
