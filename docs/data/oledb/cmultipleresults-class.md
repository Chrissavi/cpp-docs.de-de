# <a name="cmultipleresults-class"></a>CMultipleResults-Klasse

Wenn Sie möchten, dass ein Befehl mehrere Resultsets behandelt, verwenden Sie `CMultipleResults` für das `CCommand` Template-Argument *TMultiple*.

## <a name="syntax"></a>Syntax

```cpp
class CMultipleResults
```

## <a name="remarks"></a>Bemerkungen

Um mehrere Resultsets zu verarbeiten, muss [CCommand](../../data/oledb/ccommand-class.md) von dieser Klasse erben.

## <a name="requirements"></a>Requirements (Anforderungen)

**Header:** atldbcli
