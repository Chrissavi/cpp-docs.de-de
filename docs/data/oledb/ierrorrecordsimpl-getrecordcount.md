# <a name="ierrorrecordsimplgetrecordcount"></a>IErrorRecordsImpl::GetRecordCount
Gibt die Anzahl der Datensätze in der OLE DB-Datensatz-Objekt zurück.  
  
## <a name="syntax"></a>Syntax  
  
```cpp
      STDMETHOD(GetRecordCount )(ULONG *pcRecords);  
```  
  
#### <a name="parameters"></a>Parameter  
 Finden Sie unter [IErrorRecords::GetRecordCount](https://msdn.microsoft.com/en-us/library/ms722724.aspx) in der *OLE DB Programmer's Reference*.
