# <a name="work-with-odbc-and-drivers"></a>Arbeiten mit ODBC und Treibern

ODBC ermöglicht das Schreiben von Anwendungen, die nicht von nur einem Datenbank-Management System (DBMS) abhängen. Sie können eine Anwendung schreiben, die auf mehreren DBMSs ausgeführt werden kann, für die Ihre Benutzer über den richtigen DBMS-spezifischen ODBC-Treiber verfügen.
