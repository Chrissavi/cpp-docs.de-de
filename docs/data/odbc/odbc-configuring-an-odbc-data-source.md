# <a name="odbc-configuring-an-odbc-data-source"></a>ODBC: Konfigurieren einer ODBC-Datenquelle

Um eine [Datenquelle](../../data/odbc/data-source-odbc.md) mit einer von Ihnen entwickelten Anwendung zu verwenden, müssen Sie sie mit ODBC-Administrator konfigurieren. ODBC Administrator verfolgt die verfügbaren Datenquellen und deren Verbindungsinformationen in der Windows-Registrierung. Verwenden Sie ODBC-Administrator, um Datenquellen im Dialogfeld **Datenquellen** hinzuzufügen, zu ändern und zu löschen sowie ODBC-Treiber hinzuzufügen und zu löschen.

> [!NOTE]
> Diese Informationen gelten, wenn Sie DaO-Klassen (MFC Data Access Object) für den ODBC-Zugriff und MFC-ODBC-Klassen verwenden.

ODBC Administrator wird automatisch mit der Unterstützung der Microsoft Foundation Classes (MFC)-Bibliotheksdatenbank installiert. Weitere Informationen zum ODBC-Administratorprogramm finden Sie unter [ODBC-Administrator](../../data/odbc/odbc-administrator.md) und das Online-Referenzhilfesystem der ODBC-API.
