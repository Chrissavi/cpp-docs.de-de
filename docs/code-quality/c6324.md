# <a name="c6324"></a>C6324

> Warnung C6324: potenziell falsche Verwendung von \<function1> : wollten Sie verwenden \<function2> ?

Diese Warnung weist darauf hin, dass eine Funktion zum Kopieren von Zeichen folgen verwendet wurde, in der eine Zeichen folgen Vergleichsfunktion verwendet werden sollte. Eine falsche Verwendung der Funktion kann zu einem unerwarteten Logikfehler f√ºhren.

## <a name="example"></a>Beispiel

Der folgende Code generiert diese Warnung:

```cpp
#include <string.h>

void f(char *title )
{
  if (strcpy (title, "Manager") == 0) // warning C6324
  {
    // code
  }
}
```

Um die Warnung zu korrigieren, verwenden Sie `strcmp`, wie im folgenden Code dargestellt:

```cpp
#include <string.h>

void f(char *title )
{
  if (strcmp (title, "Manager") == 0)
  {
    // code
  }
}
```
