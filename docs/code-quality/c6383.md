## <a name="example"></a>Beispiel

Der folgende Code generiert diese Warnung:

```cpp

{
  // code ...

```

```cpp

  wcsncpy (t, s, n);
  // code ...
}
```

Der folgende Code korrigiert diese Warnung mithilfe der sicheren Zeichen folgen-Manipulations Funktion:

```cpp

{
  // code ...

  // code ...
}
```
