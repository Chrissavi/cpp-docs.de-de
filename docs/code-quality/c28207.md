# <a name="c28207"></a>C28207

> Warning C28207: \<Expression >: der linke Operand ist eine Struktur, verwenden Sie.

Diese Warnung wird gemeldet, wenn ein Struktur dereferenzierungspunkt (.) erwartet wurde.
