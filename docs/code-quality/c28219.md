# <a name="c28219"></a>C28219

> Warning C28219: Member von Enumeration für den Parameter in der Anmerkung erwartet

Es wird erwartet, dass ein Parameter für eine Anmerkung ein Member des benannten **`enum`** Typs ist, und ein anderes Symbol wurde gefunden. verwenden Sie einen Member dieses **`enum`** Typs. Dies weist normalerweise auf ein falsch codiertes Anmerkung-Makro hin.
