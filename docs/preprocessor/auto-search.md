# <a name="auto_search-import-attribute"></a>Auto_search-Attribut importieren

**C++Zugeschnitten**

Gibt an, dass der Compiler für die andere Typbibliothek einen impliziten `#import` `#import` ausführen kann, wenn auf eine Typbibliothek mit verwiesen wird und selbst auf eine andere Typbibliothek verweist.

## <a name="syntax"></a>Syntax
