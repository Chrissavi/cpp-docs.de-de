# <a name="omp"></a>omp
Akzeptiert mindestens eine OpenMP-Anweisung sowie beliebige optionale Anweisungsklauseln.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
#pragma omp directive  
```
