# <a name="push_macro-pragma"></a>push_macro-Pragma

Speichert den Wert des *Makronamen* Makros am Anfang des Stapels für dieses Makro.

## <a name="syntax"></a>Syntax

> **#pragma push_macro ("** _Makroname_ **")**

## <a name="remarks"></a>Hinweise

Sie können den Wert für " *Makroname* " mit `pop_macro`abrufen.

Ein Beispiel finden Sie unter [pop_macro](../preprocessor/pop-macro.md) .
