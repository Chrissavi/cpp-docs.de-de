# <a name="null-directive"></a>NULL-Anweisung
Die präprozessoranweisung null ist ein einzelnen Nummernzeichen (**#**) nur in einer Zeile. Sie hat keine Auswirkungen.

## <a name="syntax"></a>Syntax

```
#
```
