# <a name="null-directive"></a>NULL-Anweisung

Die NULL-Präprozessordirektive ist nur ein **#** einzelnes Nummern Zeichen () in einer Zeile. Sie hat keine Auswirkungen.

## <a name="syntax"></a>Syntax

> **#**
