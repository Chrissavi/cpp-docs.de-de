# <a name="resource-management-classes"></a>Ressourcenverwaltungsklassen

Diese Klassen ermöglichen die automatische Verwaltung verwalteter Klassen.

## <a name="in-this-section"></a>In diesem Abschnitt

[auto_gcroot](../dotnet/auto-gcroot.md)\
Bettet ein virtuelles Handle in einen systemeigenen Typ ein.

[auto_handle](../dotnet/auto-handle.md)\
Bettet ein virtuelles Handle in einen verwalteten Typ ein.
