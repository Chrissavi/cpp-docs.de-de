# <a name="how-to-hold-reference-to-value-type-in-native-type"></a>Vorgehensweise: Verweis auf Werttyp in systemeigenem Typ

Verwendung `gcroot` auf den geschachtelten Typ um einen Verweis auf einen Werttyp in einen systemeigenen Typ enthalten.

## <a name="example"></a>Beispiel

```
// reference_to_value_in_native.cpp
// compile with: /clr
#using <mscorlib.dll>
#include <vcclr.h>

using namespace System;

public value struct V {
   String ^str;
};

class Native {
public:
   gcroot< V^ > v_handle;
};

int main() {
   Native native;
   V v;
   native.v_handle = v;
   native.v_handle->str = "Hello";
   Console::WriteLine("String in V: {0}", native.v_handle->str);
}
```

```Output
String in V: Hello
```
