# <a name="utility-stlclr"></a>utility (STL/CLR)
Fügen Sie den STL/CLR-Header `<cliext/utility>` definiert die Vorlagenklasse `pair` und mehrere unterstützende Vorlagenfunktionen.  
  
## <a name="syntax"></a>Syntax  
  
```  
#include <utility>  
```
